Pulse Secure 클라이언트 창 크기 조정

Pulse Secure 클라이언트 창 크기를 조정하고 창을 화면의 다른 위치로 이동할 수 있습니다. 창을 이동한 다음 최소화하면 창이 새로운 크기와 위치를 유지합니다. 그러나 프로그램을 닫으면 다음 번에 Pulse Secure 클라이언트를 시작할 때 창이 기본 크기와 위치로 되돌아갑니다.

Pulse Secure 클라이언트 창 크기 조정 방법:

  1. Pulse Secure 클라이언트 창 오른쪽 아래 부분에 있는 크기 조정 핸들을 클릭합니다.
  2. 창의 가장자리를 마우스 버튼으로 누른 상태에서 원하는 창 크기가 될 때까지 끈 다음 마우스를 놓습니다.
