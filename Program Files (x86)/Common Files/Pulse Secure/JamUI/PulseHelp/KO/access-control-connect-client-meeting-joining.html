Pulse Secure 클라이언트 Collaboration 미팅 기능에 액세스

Pulse Secure 클라이언트가 Pulse Collaboration Suite 서버로 설정된 서버에 연결되어 있는 경우, 사용자는 Pulse Secure 클라이언트 트레이 아이콘에서 Pulse Collaboration Suite 기능에 액세스하여 미팅에 참여하거나 미팅을 예약하거나 미팅을 종료할 수 있습니다. 또는 웹 포털에서 Pulse Secure 클라이언트 서버에 로그인한 다음 미팅 버튼을 클릭하여 Pulse Collaboration 기능에 액세스할 수도 있습니다.

Pulse Collaboration 옵션에 액세스하는 방법:

  1. Windows 시스템 트레이에서 Pulse Secure 클라이언트 아이콘을 마우스 오른쪽 버튼으로 클릭하여 팝업 메뉴를 표시합니다.
  2. Pulse Collaboration, 미팅 서버를 차례로 클릭한 다음 미팅 시작을 클릭합니다.
  3. 메시지가 나타나면 로그인 자격 증명을 제공합니다. 현재 예약된 미팅이 있을 경우, Pulse Secure 클라이언트는 미팅 시작 버튼이 표시된 브라우저 창을 엽니다. 현재 예약된 미팅이 없는 경우 미팅 일정을 볼 수 있습니다. 미팅 일정에서 미팅을 선택한 후 미팅 시작을 클릭하면 Pulse Collaboration Suite 클라이언트가 열리고 미팅에 참여할 수 있습니다. Pulse Collaboration 미팅에 참여하면 Pulse Collaboration 클라이언트가 다운로드되어 열립니다. Pulse Collaboration Suite 사용 방법에 대해 자세히 알아보려면 Pulse Collaboration 인터페이스에서 도움말 링크를 클릭하십시오.

    관리자가 Microsoft Outlook 통합을 활성화한 경우 링크를 클릭하여 Microsoft Outlook 플러그인을 다운로드하여 설치하십시오. 이 플러그인으로 Outlook에서 미팅 일정을 관리하고 Outlook 이메일을 이용해 참석자에게 초대장을 보낼 수 있습니다.

    미팅을 예약하고 참석자를 초대하려면 미팅 일정을 클릭합니다.

    필요에 따라 인스턴트 미팅 또는 지원 미팅(관리자가 이 유형의 미팅을 활성화한 경우)을 클릭합니다. 인스턴트 미팅은 표준 미팅과 비슷합니다. 인스턴트 미팅은 인증 요건, 원격 제어, 보안 채팅 등과 같은 역할 수준의 설정을 자동으로 적용하여 미팅을 빠르게 생성할 수 있습니다. 지원 미팅은 두 사람 사이의 미팅으로, 주로 온라인을 통해 문제를 해결하기 위한 용도로 사용됩니다. 지원 미팅에서는 데스크톱 공유와 원격 제어 작업을 수행할 수 있습니다.

    브라우저 창 상단의 미팅 버튼을 사용하면 웹 포털 내의 어떤 위치에서든 미팅 일정 화면으로 되돌아갑니다.

    Pulse Collaboration 기능에 대해 자세히 알아보려면 브라우저 창에서 도움말 버튼을 클릭하십시오.

미팅 연결 해제 방법:

  1. Pulse Collaboration과 미팅 서버를 클릭한 다음 로그아웃을 클릭합니다.

    트레이 아이콘의 로그아웃 메뉴 항목을 사용하면 Pulse Collaboration 클라이언트 인터페이스에서 로그아웃하는 것과 동일한 작업을 수행하게 됩니다.
