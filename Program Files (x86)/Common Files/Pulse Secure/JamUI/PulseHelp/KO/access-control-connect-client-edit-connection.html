연결 속성 편집

연결을 생성한 후, 연결 창에 나타나는 URL과 이름을 편집할 수 있습니다. 연결이 활성 상태가 아니고 사용자가 수동으로 생성한 연결인 경우에만 연결을 편집할 수 있습니다. Pulse Secure 클라이언트 관리자가 연결을 생성하고 설치한 경우 URL 필드를 편집할 수 없습니다.

서버 URL이 임의로 선택됨으로 표시된다면 이 연결은 Pulse Secure 클라이언트 관리자가 정의한 것입니다. 이 연결은 하나의 대상 서버 대신 Pulse Secure 클라이언트 서버 목록으로 구성되고 Pulse Secure 클라이언트는 대상 서버를 임의로 선택합니다. 변경 버튼은 서버 목록을 포함하는 모든 연결의 연결 세부정보에 나타납니다. Pulse Secure 클라이언트 관리자는 Pulse Secure 클라이언트가 항상 목록에 있는 첫 번째 서버나 가장 최근에 연결한 서버 또는 목록에서 임의로 선택된 서버에 연결되도록 목록을 구성할 수 있습니다. 서버 목록을 포함하는 연결에 대해서는 대상 URL을 편집할 수 없습니다.

연결 편집 방법:

  1. 연결을 마우스 오른쪽 버튼으로 클릭하여 팝업 메뉴를 표시한 다음 편집을 클릭합니다.

    연결 편집 대화 상자가 열립니다.

  2. 연결을 편집한 다음, 저장을 클릭하여 변경사항을 저장하고 대화 상자를 닫거나 연결을 클릭하여 연결을 시작하고 대화 상자를 닫습니다.

또는

  1. 연결을 클릭하여 선택합니다.
  2. 연결 편집 버튼을 클릭합니다.

    연결 편집 대화 상자가 열립니다.

  3. 연결을 편집한 다음, 저장을 클릭하여 변경사항을 저장하고 대화 상자를 닫거나 연결을 클릭하여 연결을 시작하고 대화 상자를 닫습니다.
