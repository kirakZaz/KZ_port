로컬 네트워크 리소스에 액세스(연결 일시 중단)

VPN 연결이 활성 상태인 경우, 프린터와 공유 드라이브 등의 로컬 네트워크 리소스에 액세스하지 못할 수도 있습니다. VPN 연결을 일시 중단하여 로컬 리소스에 연결한 후, 로그인 자격 증명을 제공하지 않고 원격 네트워크에 대한 VPN 연결을 재개할 수 있습니다.

Pulse Secure 클라이언트 관리자가 로컬 서브넷 액세스를 활성화한 경우 Pulse Secure 클라이언트 연결을 일시 중단하지 않고 컴퓨터와 같은 서브넷에 연결된 프린터 등의 리소스에 액세스할 수 있습니다.

VPN 연결을 일시 중단하고 로컬 네트워크 리소스에 액세스하는 방법:

  1. 활성 연결을 클릭하여 선택한 다음, Pulse Secure 클라이언트 메뉴 모음에서 파일 > 연결 > 일시 중단을 클릭합니다.

    연결이 활성 상태인 경우 저장된 설정 잊음을 선택하면 이 옵션을 사용할 수 없습니다.

    또는

    활성 연결을 마우스 오른쪽 버튼으로 클릭한 다음, 일시 중단을 클릭합니다.

    또는

    Windows 시스템 트레이에서 Pulse Secure 클라이언트 아이콘을 클릭하여 팝업 메뉴를 표시하고, 활성 연결을 클릭한 다음, 일시 중단을 클릭합니다.

    일시 중단된 연결에 대해 Pulse Secure 클라이언트 연결 창의 버튼 레이블이 다시 시작으로 바뀌고, 일시 중단 아이콘이 연결 옆에 나타납니다.

    연결이 일시 중단되어 있는 동안 트레이 아이콘이 일시 중단된 상태를 표시합니다.

일시 중단된 VPN 연결을 재개하는 방법:

  1. 일시 중단된 연결을 선택한 다음 파일 > 연결 > 다시 시작을 클릭합니다.

    또는

    다시 시작을 클릭합니다.

    또는

    Windows 시스템 트레이에서 Pulse Secure 클라이언트 아이콘을 클릭하여 팝업 메뉴를 표시하고, 일시 중단된 연결을 클릭한 다음, 다시 시작을 클릭합니다.
