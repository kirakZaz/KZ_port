로그 수준 설정

로그 수준 설정 방법:

  1. 파일 > 로그 > 로그 수준 > 자세히 또는 파일 > 로그 > 로그 수준 > 기본을 클릭합니다. 확인 표시는 현재 활성화된 옵션을 나타냅니다.

기본 로깅은 기본값입니다. 연결 문제를 해결해야 하는 상황이 아닌 한 로깅은 ‘기본’으로 설정해야 합니다.

자세히 로깅은 보다 많은 수의 로그 항목을 생성하고 로그 파일의 크기를 증가시킵니다. '자세히' 로깅은 일반적으로 해결해야 할 문제가 있는 경우에만 활성화 됩니다.
