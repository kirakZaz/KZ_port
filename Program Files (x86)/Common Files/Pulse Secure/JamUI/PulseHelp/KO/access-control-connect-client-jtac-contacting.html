기술 지원 요청

기술 관련 지원은 Pulse Secure Global Support Center(PSGSC)를 통해 제공됩니다.

  • 제품 보증 정보는 다음 사이트에서 확인하십시오.
    https://www.pulsesecure.net/support/

PSGSC 센터는 주 7일, 하루 24시간, 1년 365일 운영됩니다.

자체 도움말 온라인 도구 및 리소스

빠르고 쉽게 문제를 해결할 수 있도록 Pulse Secure는 다음과 같은 기능을 제공하는 Pulse Secure Global Support Center(PSGSC)라는 온라인 셀프 서비스 포털을 준비하였습니다.

  • PSGSC 서비스 확인하기
    https://www.pulsesecure.net/support/

  • 제품 문서 찾기
    https://www.pulsesecure.net/techpubs/

  • 최신 버전의 소프트웨어 및 릴리즈 노트 다운로드
    https://www.pulsesecure.net/support/

  • 기술 게시판에서 관련 하드웨어 및 소프트웨어 공지 찾기
    https://www.pulsesecure.net/support/

  • SNE(Serial Number Entitlement) 도구를 이용하여 제품 일련 번호별로 서비스 자격 여부 확인
    https://www.pulsesecure.net/support/

PSGSC에서 케이스 열기

웹 또는 전화 통화를 통해 PSGSC에서 케이스를 열 수 있습니다.

  • PSGSC에서 케이스 매니저 도구 사용:
    https://pulsesecure.net/support

  • 1-844 751 7629로 전화하십시오(미국내, 수신자 부담).

  • 수신자 부담 번호가 없는 국가의 국제 전화 또는 직통 전화 옵션을 보려면
    https://www.pulsesecure.net/support/support-contacts/를 방문하십시오.
