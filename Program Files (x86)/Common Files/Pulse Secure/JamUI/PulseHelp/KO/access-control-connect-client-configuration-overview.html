Pulse Secure 클라이언트 구성 개요

네트워크에 연결하면 Pulse Secure 클라이언트가 사용자의 종점으로 설정을 다운로드합니다. Pulse Secure 클라이언트 구성 작업의 대부분은 네트워크 관리자가 수행합니다. 네트워크 관리자는 Pulse Secure 클라이언트의 기능을 활성화하거나 비활성화할 수도 있습니다. 예를 들어, 관리자는 Pulse Secure 클라이언트가 자동으로 특정 네트워크로 연결되도록 구성하거나 사용자가 원하는 연결을 선택할 수 있도록 구성할 수 있습니다.

사용자가 액세스하게 될 네트워크는 일반적으로 네트워크 관리자가 정의합니다. 네트워크 관리자는 사용자가 직접 추가 연결을 정의하도록 할 수도 있습니다. 사무실, 가정 및 휴대용으로 사용하는 노트북을 가지고 있다면, 사무실에서 연결할 경우에 필요한 연결 하나와 원격 위치에서 액세스하기 위한 다른 연결이 필요할 것입니다. 그러나 한 가지 유형의 네트워크 연결을 설정한 후에는 Pulse Secure 클라이언트가 조정되어 사용자의 연결 상태가 계속 유지됩니다. 예를 들어 원격 위치에서 회사 네트워크에 연결한 후, 로그오프하지 않고 노트북을 닫은 다음 회사로 복귀하였다면 다시 로그인할 필요 없이 사용자의 컴퓨팅 세션을 재시작할 수 있습니다. 이 휴대성 기능은 세션 마이그레이션이라고 하며 관리자가 어떻게 네트워크 액세스를 구성했는지에 따라 달라집니다.
