접근성/섹션 508 준수

Pulse Secure 데스크톱 클라이언트에는 신체적, 감각적, 인지적 장애가 있는 사용자가 데스크톱 클라이언트 사용자 인터페이스를 더 쉽게 탐색할 수 있도록 도와주는 많은 기능이 포함되어 있습니다.

이 기능 중 일부를 사용하려면 레지스트리 키 설정을 활성화해야 합니다. 이 키는 다음 위치에 설정해야 합니다.

HKEY_CURRENT_USER\Software\Pulse Secure\Pulse\Accessibility

접근성 기능에는 다음이 포함됩니다.

  • 태빙(tabbing) 지원 - 데스크톱 클라이언트 UI는 Tab 키를 사용하여 UI 위젯 간 키보드 포커스 이동을 처리합니다. 이를 통해 마우스를 사용하지 않고도 사용자 인터페이스를 더 쉽게 탐색할 수 있습니다.
  • 기타 탐색 키 -
    • 대부분의 데이터 입력 필드 및 대화 상자 컨트롤은 Alt 키를 누른 상태에서 바로가기 키를 누르면 액세스할 수 있습니다. Alt 키를 눌렀다 놓으면 사용할 특정 바로가기를 표시할 수 있습니다.
    • Pulse 메뉴도 Alt 키를 누른 후 메뉴 바로가기를 눌러 액세스할 수 있습니다. 메뉴 시스템을 입력한 후 Alt 키 바로가기 또는 화살표 및 입력 키를 사용하여 메뉴 옵션을 탐색 및 선택할 수 있습니다.
    • 대화 상자 내에서는 일반적인 조합의 탭 및 화살표 키를 사용하여 탐색할 수 있습니다.
  • 스크린 리더 지원 - JAWS 스크린 리더를 사용하여 연결 상태를 읽을 수 있습니다. 스크린 리더 기능을 활성화하려면 사용자 또는 사용자의 시스템 관리자가 레지스트리 키 ScreenreaderMode DWORD를 1로 설정해야 합니다.
  • 시스템 색상 설정 사용 - Pulse 데스크톱 클라이언트 UI는 고대비 모드의 Windows OS 시스템 색상을 사용할 수 있습니다. 이 기능을 활성화하려면 사용자 또는 사용자의 시스템 관리자가 레지스트리 키 UseSystemColors REG_BINARY를 00000001로 설정해야 합니다.
