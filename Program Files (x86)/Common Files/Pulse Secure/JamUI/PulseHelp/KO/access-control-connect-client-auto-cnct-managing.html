자동 연결에 대한 수동 덮어쓰기 제거

Pulse Secure 클라이언트 관리자는 자동으로 연결되는 Pulse Secure 클라이언트 연결을 제공할 수 있습니다. 예를 들어 사무실 안에서 LAN에 연결되어 있을 때 하나의 Pulse Secure 클라이언트 연결이 활성화되고 사무실 밖에 있을 때 다른 Pulse Secure 클라이언트 연결이 활성화되도록 할 수 있습니다. 자동 연결은 수동 연결 결정을 제한하여 연결을 단순화합니다. Pulse Secure 클라이언트 관리자는 사용자가 자동 연결을 덮어쓰도록 할 수 있습니다. Pulse Secure 클라이언트는 더 이상 자동으로 응답하지 않는 자동 연결에 대한 수동 덮어쓰기를 표시합니다.

수동 덮어쓰기로 설정된 연결에 대한 자동 동작 복원 방법:

  1. 연결을 클릭하여 선택합니다.
  2. 파일 > 연결 > 덮어쓰기 제거(자동 컨트롤로 돌아가기)를 클릭합니다.

또는

  1. Pulse Secure 클라이언트 연결 창에서 연결을 마우스 오른쪽 버튼으로 클릭하여 팝업 메뉴를 표시합니다.
  2. 덮어쓰기 제거(자동 컨트롤로 돌아가기)를 클릭합니다.
