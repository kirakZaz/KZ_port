Pulse Secure 클라이언트 트레이 아이콘 사용 및 연결 상태 보기

Pulse Secure 클라이언트 트레이 아이콘 은 네트워크 연결의 현재 상태를 나타냅니다. 사용자가 현재 둘 이상의 연결을 가지고 있다고 해도 트레이에는 하나의 연결 아이콘만 표시됩니다. 활성화된 어느 연결에 문제가 생기건, 시스템 트레이 아이콘의 상태가 변경됩니다. 다음과 같은 아이콘은 연결 상태를 나타냅니다.

연결되지 않음.

연결 중. 연결이 실패하거나 성공할 때까지 이 상태를 유지합니다.

일시 중단.

연결되었으나 문제가 있습니다.

연결 실패.

연결됨.

로컬 네트워크에 연결되어 있으나 인터넷에 연결되어 있지 않습니다. 공용 WiFi 위치는 네트워크 액세스를 허용하기 전에 사용자에게 인증 정보를 입력하게 하거나 서비스 조건을 수락할 것을 요구하는 종속 포털을 사용하는 경우가 많습니다. Pulse Secure 클라이언트가 종속 포털이 있는지 검사하고 인터넷 액세스가 허용될 때까지 Pulse Secure 클라이언트 서버에 대한 연결을 시작하지 않습니다.

Pulse Secure 클라이언트 트레이 아이콘을 사용하여 다음 작업을 수행할 수 있습니다.

  • Pulse Secure 클라이언트 열기 - Pulse Secure 클라이언트 인터페이스를 표시합니다.
  • Pulse Secure 클라이언트 Collaboration- Pulse Secure 클라이언트 Collaboration Suite 미팅 기능에 액세스합니다. 이 항목은 Pulse Secure 클라이언트 관리자가 Pulse Secure 클라이언트 Collaboration을 활성화한 경우에만 표시됩니다.
  • 연결 목록 - 각 Pulse Secure 클라이언트 연결이 나열됩니다. 연결 이름을 클릭하면 해당 연결에 대해 수행할 수 있는 작업이 표시됩니다.
    • 연결 - 연결을 시작합니다.
    • 연결 해제 - 선택한 네트워크의 연결을 해제합니다.
    • 취소 - Pulse Secure 클라이언트 서버에서 시작한 작업을 취소합니다. 취소 옵션은 서버에서 호스트 체커 작업을 시작한 경우에 사용할 수 있습니다. 작업을 취소하면 Pulse Secure 클라이언트가 네트워크에 연결되지 않습니다.
    • 일시 중단 - VPN 연결을 일시 중단하여 로컬 네트워크 리소스에 액세스합니다. 연결이 활성 상태인 경우 저장된 설정 잊음을 선택하면 이 옵션을 사용할 수 없습니다.
    • 다시 시작 - 일시 중단된 VPN 연결을 다시 시작합니다.
    • 세션 연장 - Pulse Secure 클라이언트에서 세션 만료를 통지할 때까지 기다리지 않고 활성 연결 세션을 연장합니다. 세션 및 유휴 타이머를 최대값으로 재설정하려면 언제든지 이 옵션을 선택할 수 있습니다.
  • 종료 - Pulse Secure 클라이언트를 닫고 활성 상태의 연결을 모두 연결 해제합니다.
