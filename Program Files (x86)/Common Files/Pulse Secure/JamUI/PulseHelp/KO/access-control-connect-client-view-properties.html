연결 속성 보기

연결 속성 보는 방법:

  1. 연결 이름 옆에 있는 화살표를 클릭합니다. 연결 세부정보가 연결 아래에 나타납니다.

서버 URL이 임의로 선택됨으로 표시된다면 이 연결은 Pulse Secure 클라이언트 관리자가 정의한 것입니다. 이 연결은 하나의 대상 서버 대신 Pulse Secure 클라이언트 서버 목록으로 구성되고 Pulse Secure 클라이언트는 대상 서버를 임의로 선택합니다. 변경 버튼으로 목록에서 특정 서버를 선택할 수 있습니다. 변경 버튼은 서버 목록을 포함한 모든 연결에 나타납니다. Pulse Secure 클라이언트 관리자는 Pulse Secure 클라이언트가 항상 목록에 있는 첫 번째 서버나 가장 최근에 연결한 서버 또는 목록에서 임의로 선택된 서버에 연결되도록 목록을 구성할 수 있습니다.
