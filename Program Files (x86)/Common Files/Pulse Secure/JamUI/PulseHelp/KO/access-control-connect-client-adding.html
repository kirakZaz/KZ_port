연결 추가

Pulse Secure 클라이언트에서 각 연결은 보호된 네트워크를 나타냅니다. 일반적으로 네트워크 관리자는 사용자를 위해 연결을 정의하고 연결 추가 기능을 비활성화할 수 있습니다. 사용자가 새로운 연결을 만들어야 한다면, 어떤 설정을 사용해야 하는지 네트워크 관리자가 알려줍니다.

새 연결 추가 방법:

  1. 연결 추가 버튼 클릭:

    연결 추가 대화 상자가 열립니다.

  2. 유형의 경우, 일반적으로 네트워크 관리자가 선택할 옵션에 대해 알려줍니다. 다음 중 하나를 선택합니다.
    • 보안 정책(UAC) 또는 안전 연결(VPN)— Pulse Policy Secure 또는 Pulse Connect Secure에 연결하는 경우 이 네트워크 유형을 사용하십시오.
    • 방화벽(SRX)—주니퍼 네트웍스 SRX 시리즈 장치에 연결하는 경우 이 네트워크 유형을 사용합니다.
  3. 이름의 경우, 이 연결에 대한 설명이 포함된 이름을 지정하십시오. 지정한 이름은 Pulse Secure 클라이언트 인터페이스에 표시됩니다.
  4. 서버 URL의 경우, 연결하려는 네트워크를 지정하십시오. 다음 형식으로 서버 URL 값을 입력할 수 있습니다.
    • IP 주소(예: 10.204.71.86). Pulse Secure 클라이언트는 또한 IPv6 주소를 지원합니다. 일반적으로 IPv6 주소를 수동으로 입력하지는 않지만 필요할 경우 다음 형식을 사용하십시오. fd29:64fc:bd56:5dbd:3:23:202:1a31. IPv6 주소에는 0-9의 숫자와 a-f의 문자를 사용할 수 있습니다. 주소 시작 부분의 0은 생략할 수 있습니다.
    • DNS 이름(예: server.mycompany.net)
  5. 추가를 클릭하여 새로운 연결을 저장하고 대화 상자를 닫습니다. 연결을 클릭하여 새로운 연결을 저장하고 네트워크로의 연결을 시작합니다.
