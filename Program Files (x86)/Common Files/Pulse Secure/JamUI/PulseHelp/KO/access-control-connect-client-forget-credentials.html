저장된 설정 잊음

네트워크에 연결할 때, 설정 저장 확인란을 선택하여 Pulse Secure 클라이언트가 사용자의 로그인 자격 증명을 기억하도록 할 수 있습니다. (네트워크 관리자가 이 기능을 비활성화 할 수 있습니다.) 응답을 요청하는 메시지가 표시되는 각각의 다른 화면마다 설정 저장 확인란이 있습니다. 설정을 저장하면 이후에 로그인을 시도할 때 해당 정보를 제공하라는 메시지가 표시되지 않습니다. 그러나 로그인 자격 증명이 변경된 경우 새 자격 증명을 저장하기 전에 먼저 Pulse Secure 클라이언트에서 저장된 설정을 삭제해야 합니다.

연결이 활성 상태인 경우 저장된 설정 잊음을 선택하면 연결 일시 중단 옵션을 사용할 수 없습니다.

연결이 일시 중단 상태인 경우 저장된 설정 잊음을 선택하면 연결이 해제됩니다.

저장된 로그인 자격 증명을 지우는 방법:

  1. 아무 연결이나 마우스 오른쪽 버튼으로 클릭하여 팝업 메뉴를 표시합니다.
  2. 저장된 설정 잊음을 클릭합니다. Pulse Secure 클라이언트가 모든 구성된 연결에 대해 저장된 설정을 삭제합니다.

또는

  1. 파일 > 연결 > 저장된 설정 잊음을 클릭합니다.
