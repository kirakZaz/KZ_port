Pulse Secure 클라이언트 연결 상태

Pulse Secure 클라이언트는 연결 창과 시스템 트레이에 연결 상태를 표시합니다. 연결 창에 표시되는 상태는 다음과 같습니다.

연결 중. 연결이 실패하거나 성공할 때까지 이 상태를 유지합니다.

일시 중단.

연결되었으나 문제가 있습니다.

연결 실패.

연결됨.

로컬 네트워크에 연결되어 있으나 인터넷에 연결되어 있지 않습니다. 공용 WiFi 위치는 네트워크 액세스를 허용하기 전에 사용자에게 인증 정보를 입력하게 하거나 서비스 조건을 수락할 것을 요구하는 종속 포털을 사용하는 경우가 많습니다. Pulse Secure 클라이언트가 종속 포털이 있는지 검사하고 인터넷 액세스가 허용될 때까지 Pulse Secure 클라이언트 서버에 대한 연결을 시작하지 않습니다.

하나의 시스템 트레이 아이콘으로 모든 활성 Pulse Secure 클라이언트 연결의 상태를 표시합니다. 시스템 트레이 아이콘을 마우스 오른쪽 버튼으로 클릭하여 Pulse Secure 클라이언트 연결을 제어하고, Pulse Collaboration Suite 미팅 기능에 액세스하고, Pulse Secure 클라이언트 인터페이스를 열고, Pulse Secure 클라이언트를 종료할 수 있습니다. 다음 표는 시스템 트레이 아이콘으로 표시되는 연결 상태를 나타낸 것입니다.
