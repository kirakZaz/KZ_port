로그 파일 업로드

Pulse Secure 데스크톱 클라이언트를 사용하면 시스템 관리자가 분석해야 할 진단 로그 번들을 PCS 게이트웨이로 쉽게 전송할 수 있습니다. VPN 연결이 활성화 되었을 때 로그 번들을 PCS에 보내기 위해서는 데스크톱 클라이언트 사용자 인터페이스에서 다음을 실행합니다. 파일 > 로그 > 업로드

로그를 보낼 서버를 선택해야 합니다. 업로드 과정을 보여 주는 대화 상자가 나타납니다.

최종 사용자가 로그 파일을 Pulse Secure 게이트웨이에 업로드하려면 먼저 시스템 관리자가 서버측에서 이 기능을 활성화해야 합니다.

업로드 된 클라이언트 로그를 모두 합했을 때 서버의 최대 크기 한도는 200MB입니다.

Pulse Connect Secure 게이트웨이에서 업로드 된 로그 파일 저장을 위한 전용 디스크 파티션의 크기는 한정되어 있습니다. 게이트웨이에서 이 공간이 다른 로그 파일로 채워진 경우 업로드가 실패할 수 있습니다. 이 경우 시스템 관리자에게 로그 업로드를 위해 공간을 비워달라고 요청해야 합니다.
