Pulse Secure 클라이언트 연결 유형

새로운 연결 을 정의할 때 사용자가 선택한 연결 유형은 보호된 네트워크 리소스에 액세스하는 데 사용되는 장치의 유형과 연관되어 있습니다. 사용자가 수동으로 새 연결을 추가할 경우에는 네트워크 관리자가 선택해야 하는 연결 유형을 알려줄 수 있습니다.

  • 보안 정책(UAC) 또는 안전 연결(VPN)—Pulse Connect Secure 혹은 Pulse Policy Secure에 연결하는 경우 이 네트워크 유형을 사용하십시오.
  • 방화벽(SRX)—주니퍼 네트웍스 SRX 시리즈 서비스 게이트웨이에 연결할 경우 이 네트워크 유형을 사용합니다.
