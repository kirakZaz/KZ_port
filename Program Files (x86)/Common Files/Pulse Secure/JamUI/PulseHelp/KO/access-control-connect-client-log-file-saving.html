로그 파일 저장

문제 해결 과정에서 사용자는 Pulse Secure 클라이언트 로그 파일을 저장하도록 요청 받을 수 있습니다. Pulse Secure 클라이언트는 여러 구성 요소를 포함하고, 각 구성 요소는 하나 이상의 로그 파일을 생성합니다. 다른 이름으로 저장 작업은 모든 Pulse Secure 클라이언트 로그 파일과 다른 진단 파일을 이름이 LogsAndDiagnostics.zip인 하나의 파일로 결합합니다.

Pulse Secure 클라이언트 로그 파일 저장 방법:

  1. 파일 > 로그 > 다른 이름으로 저장을 클릭합니다. 다른 이름으로 저장 대화 상자가 나타납니다.
  2. 위치와 파일명에 대해 기본값을 적용하거나 값을 새로 지정한 다음 저장을 클릭합니다.
