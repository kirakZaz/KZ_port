연결을 위한 재구성 정보 보기

연결 시도가 실패하거나 활성 연결에 문제가 생긴 경우, Pulse Secure 클라이언트는 재구성 정보를 제시합니다. 재구성 정보는 문제 해결 방법에 대한 지침을 포함하거나 자동으로 시작된 재구성 절차의 상태를 보여줍니다.

재구성 정보를 보는 방법:

  1. 연결을 클릭하여 선택합니다.
  2. 파일 > 연결 > 재구성 정보를 클릭하여 재구성 대화 상자를 엽니다.

    활성 또는 보류 중인 재구성 작업이 없으면 재구성 메뉴는 희미하게 나타납니다.

  3. 대화 상자 상단 부분에서 문제를 설명하고, 하단 부분에서 문제 해결을 위한 단계를 설명합니다. 재구성 단계를 수행한 다음, 연결을 복원하기 위해 재시도를 클릭합니다.

또는

  1. 연결을 확대하고 연결 상태를 보려면 연결 옆에 있는 화살표를 클릭합니다.
  2. 연결 상태 내에서 세부정보 링크를 클릭하여 해당 문제에 대한 자세한 정보를 표시합니다.

또는

  1. 마우스 오른쪽 버튼으로 연결을 클릭하여 팝업 메뉴를 표시한 다음, 재구성 정보를 클릭하여 재구성 대화 상자를 엽니다.
  2. 재구성 단계를 수행한 다음, 연결을 복원하기 위해 재시도를 클릭합니다.
