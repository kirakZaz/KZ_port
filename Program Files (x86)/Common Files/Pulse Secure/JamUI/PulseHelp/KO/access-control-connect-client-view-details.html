고급 연결 세부정보 보기

고급 연결 세부정보 대화 상자는 선택한 연결에 대한 자세한 정보를 표시합니다.

고급 연결 세부정보 보는 방법:

  1. 연결을 클릭하여 선택합니다.
  2. 파일 > 연결 > 고급 연결 세부정보를 클릭합니다.
  3. 연결 세부정보는 자동으로 업데이트되지 않습니다. 예를 들어, 남은 세션 시간은 대화 상자를 열 때 남아있는 시간을 보여줍니다. 고급 세부정보를 업데이트하려면 새로 고침을 클릭하거나 자동 새로 고침 레이블이 있는 확인란을 클릭합니다.

고급 연결 세부정보 창에는 다음과 같은 정보가 제공됩니다.

필드 이름

설명

남아 있는 세션 시간

자격 증명을 다시 입력하거나 세션을 수동으로 연장해야 하기 전에 현재 VPN 세션이 활성화된 상태로 남아 있는 시간

세션 지속 시간

현재 VPN 세션이 활성화된 상태로 지속되는 시간

터널 유형

연결이 VPN 터널인지 SAM(Secure Access Manager)을 통한 포트/애플리케이션 매핑인지 설명합니다.

VPN 유형

터널 생성에 사용되는 프로토콜(SSL 또는 ESP).

할당된 IPv4

Pulse 가상 어댑터에 할당된 IPv4 주소.

유입 바이트

터널을 통해 수신된 바이트 수.

유출 바이트

터널을 통해 송신된 바이트 수.

연결 소스

Pulse 클라이언트가 연결 입력 항목을 수신하는 방법을 설명합니다.

  • 값이 사전 구성됨인 경우 연결 입력 항목은 게이트웨이에서 다운로드 된 연결 세트에서 옵니다.
  • 값이 사용자 추가됨인 경우 연결 입력 항목은 최종 사용자가 Pulse 클라이언트 UI를 사용하여 수동으로 추가합니다.
  • 값이 동적인 경우 연결 입력 항목은 웹 브라우저를 Pulse Secure 게이트웨이에 연결하고 웹 페이지의 "시작" 버튼을 누름으로써 Pulse 클라이언트를 실행한 결과입니다.

연결 세트 이름

이 연결 입력 항목을 포함하는 연결 세트의 이름입니다. 시스템 관리자는 연결 세트에 대한 자세한 정보를 보기 위해 이 이름을 사용하여 적절한 게이트웨이(다음 필드 참고)의 관리 콘솔에서 일치하는 연결 세트를 찾을 수 있습니다.

연결 세트 소유자

이 연결 항목을 가진 연결 세트를 가장 최근에 편집한 Pulse Secure 게이트웨이의 이름입니다. 이 게이트웨이와 위에서 열거한 연결 세트 이름은 여러 Pulse Secure 게이트웨이에서 클라이언트의 연결 세트 구성을 찾는 데 필요한 두 가지 정보입니다.

마지막으로 수정된 연결 세트

Pulse 클라이언트의 연결 세트가 게이트웨이에서 업데이트된 시각입니다.

연결 세트 서버 ID

Pulse 클라이언트가 처음으로 구축되면 특정 서버에 연결된 연결 세트를 받습니다. 게이트웨이에 대한 모든 후속 연결 시 클라이언트 연결 세트의 서버 ID와 일치하는 서버 ID를 게이트웨이의 연결 세트가 가진 경우에만 클라이언트는 업데이트된 연결 세트를 다운로드합니다.

연결 세트 다운로드 호스트

Pulse 클라이언트가 가장 최근에 연결 세트를 업데이트한 서버입니다.

연결 세트 다운로드 시간

Pulse 클라이언트가 가장 최근에 연결 세트를 업데이트한 시각입니다.
