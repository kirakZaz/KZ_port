Accesso alle funzioni di riunione di Pulse Secure Client Collaboration

Se è stata stabilita una connessione Pulse Secure Client a un server abilitato come server Pulse Collaboration Suite, è possibile accedere alle funzioni di Pulse Collaboration Suite dall'icona Pulse Secure Client della barra delle applicazioni e partecipare a una riunione, programmare una riunione o uscire da una riunione. In alternativa, è possibile accedere al server Pulse Secure Client tramite il portale Web e quindi fare clic sul pulsante Riunioni per accedere alle funzioni di Pulse Collaboration.

Per accedere alle opzioni di Pulse Collaboration:

  1. Fare clic con il pulsante destro del mouse sull'icona di Pulse Secure Client nell'area di notifica di Windows per visualizzare il menu a comparsa.
  2. Scegliere Pulse Collaboration, il server per le riunioni, quindi fare clic su Avvia riunione.
  3. Se richiesto, fornire le credenziali di accesso. Se è stata pianificata una riunione per l'orario corrente, Pulse apre una finestra del browser contenente il pulsante Avvia riunione. In caso contrario, visualizzare il programma delle riunioni. Selezionare la riunione nel programma, quindi fare clic su Avvia riunione per aprire il client Pulse Collaboration Suite e partecipare alla riunione. Quando si partecipa a una riunione Pulse Collaboration, il client Pulse Collaboration viene scaricato e aperto. Per i dettagli su come utilizzare Pulse Collaboration Suite, fare clic sul collegamento della Guida in linea nell'interfaccia di Pulse Collaboration.

    Se l'amministratore ha abilitato l'integrazione Microsoft Outlook, fare clic sul collegamento per scaricare e installare il plugin di Microsoft Outlook. Questo consente di pianificare le riunioni dall'interno di Outlook e di inviare gli inviti ai partecipanti utilizzando la posta elettronica di Outlook.

    Fare clic su Pianifica riunione per pianificare una riunione e invitare i partecipanti.

    Fare clic su Riunione immediata o su Riunione di supporto (se l'amministratore ha attivato questi tipi di riunione), a seconda della necessità. Una riunione immediata è simile a una riunione standard. Consente di creare rapidamente una riunione applicando automaticamente le impostazioni a livello di ruolo dell'utente, ad esempio requisiti di autenticazione, controllo remoto e conversazione sicura. Una riunione di supporto è una riunione di due persone, destinata principalmente alla risoluzione di problemi online. Questo tipo di riunione consente operazioni di condivisione del desktop e di controllo remoto.

    Il pulsante Riunioni nella parte superiore della finestra del browser riporta l'utente al programma delle riunioni da qualsiasi posizione si trovi all'interno del portale Web.

    Per ulteriori informazioni sulle funzioni di Pulse Collaboration, fare clic sul pulsante della Guida in linea nella finestra del browser.

Per disconnettersi da una riunione:

  1. Fare clic su Pulse Collaboration, il server per le riunioni, quindi fare clic su Disconnetti.

    La voce di menu di uscita dell'icona nell'area di notifica produce lo stesso risultato dell'uscita tramite l'interfaccia del client Pulse Collaboration.
