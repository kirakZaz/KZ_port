Visualizzazione delle proprietà di una connessione

Per visualizzare le proprietà di una connessione:

  1. Fare clic sulla freccia accanto al nome della connessione. I relativi dettagli vengono visualizzati al di sotto della connessione.

Se l'URL del server viene visualizzato come Selezionato a caso, significa che la connessione è stata definita dall'amministratore di Pulse Secure Client. Invece che con un server di destinazione, la connessione viene configurata con un elenco di server Pulse Secure Client che vengono selezionati in maniera casuale. Il pulsante Modifica consente di selezionare un determinato server nell'elenco. Il pulsante Modifica viene visualizzato per ogni connessione che include un elenco di server. L'amministratore di Pulse Secure Client può configurare l'elenco in modo che Pulse Secure Client si connetta sempre al primo server dell'elenco, al server a cui si è effettuato l'accesso più di recente o a un server scelto casualmente dall'elenco.
