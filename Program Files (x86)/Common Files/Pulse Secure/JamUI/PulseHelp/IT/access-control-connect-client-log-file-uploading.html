Caricamento dei file di log

Con il client desktop Pulse Secure è facile trasmettere bundle di log diagnostici ai gateway PCS per consentirne l'analisi da parte degli amministratori di sistema. Per inviare un bundle di log a PCS, quando è attiva una connessione VPN, nell'interfaccia utente del client desktop scegliere: File > Log > Carica.

Selezionare il server di destinazione dell'invio dei log. Viene visualizzata una finestra di dialogo che mostra l'avanzamento del caricamento.

L'amministratore di sistema deve attivare questa funzione sul lato server per consentire a un utente finale di caricare i file di registro nel gateway Pulse Secure.

Il server dispone di uno spazio massimo di 200 MB per tutti i log client caricati.

Il gateway Pulse Connect Secure ha una partizione del disco limitata dedicata per l'archiviazione dei file di log caricati. Se lo spazio sul gateway è occupato da altri file di log, il caricamento potrebbe non riuscire. In questo caso, è necessario chiedere all'amministratore di sistema di liberare spazio per il caricamento dei log.
