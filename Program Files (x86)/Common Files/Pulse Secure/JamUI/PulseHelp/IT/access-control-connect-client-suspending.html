Accesso alle risorse di rete locali (sospensione della connessione)

Quando è attiva una connessione VPN, potrebbe essere impossibile accedere a risorse di rete locali quali stampanti e unità condivise. È possibile sospendere la connessione VPN per connettersi a risorse locali e quindi riattivare la connessione VPN alla rete remota senza dover fornire le credenziali di accesso.

Se l'amministratore di Pulse Secure Client ha attivato l'accesso alla subnet locale, è possibile accedere a una risorsa, ad esempio una stampante, connessa alla stessa subnet del computer senza sospendere la connessione Pulse Secure Client.

Per sospendere la connessione VPN e accedere alle risorse di rete locali:

  1. Fare clic su una connessione attiva per selezionarla, quindi fare clic su File > Connessioni > Sospendi nella barra dei menu di Pulse Secure Client.

    Questa opzione non è disponibile se si sceglie Dimentica impostazioni salvate mentre una connessione è attiva.

    Oppure

    Fare clic con il pulsante destro del mouse sulla connessione attiva e fare clic su Sospendi.

    Oppure

    Fare clic sull'icona di Pulse Secure Client nell'area di notifica di Windows per visualizzare il menu a comparsa, fare clic sulla connessione attiva e quindi su Sospendi.

    Per una connessione sospesa, l'etichetta del pulsante nel pannello Connessioni di Pulse si trasforma in Riprendi e accanto alla connessione appare l'icona di connessione sospesa.

Per riattivare una connessione VPN sospesa:

  1. Selezionare la connessione sospesa, quindi fare clic su File > Connessioni > Riprendi.

    Oppure

    Fare clic su Riprendi.

    Oppure

    Fare clic sull'icona di Pulse Secure Client nell'area di notifica di Windows per visualizzare il menu a comparsa, quindi sulla connessione sospesa e infine su Riprendi.
