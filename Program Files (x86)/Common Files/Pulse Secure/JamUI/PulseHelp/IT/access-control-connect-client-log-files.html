File di log di Pulse Secure Client

Un file di log di Pulse Secure Client tiene traccia di informazioni che possono essere utili per la risoluzione dei problemi di connessione.

I file di log includono le connessioni Pulse Secure Client e le relative proprietà. Se si desidera salvare la configurazione, è sufficiente salvare i log. La configurazione corrente di Pulse Secure Client viene salvata nel pacchetto dei log. Se si disinstalla Pulse Secure Client, il programma richiede se si desidera salvare la configurazione di Pulse Secure Client in modo che le connessioni e le altre impostazioni di configurazione siano comunque disponibili quando si installa nuovamente l'applicazione.

L'operazione di registrazione avviene in background. Non è necessario apportare alcuna modifica nell'ambiente di registrazione, a meno che non vengano fornite istruzioni in tal senso nell'ambito di un'attività di risoluzione dei problemi.

Oltre che nei propri file di log, Pulse Secure Client scrive gli eventi nel registro di eventi delle applicazioni Windows (solo per i sistemi Windows Vista e Windows 7). Se si dispone dell'accesso amministratore al computer, è possibile usare il Visualizzatore eventi di Windows per visualizzare gli eventi Pulse Secure Client registrati nel registro Eventi. Le informazioni del Registro eventi vengono incluse quando si salvano i file di log di Pulse Secure Client.

Per agevolare la risoluzione dei problemi, potrebbe essere richiesto di eseguire le seguenti attività:

  • Annotare i log - Quando si annota un file di log, si inserisce un testo che contrassegna il file di log in una posizione specifica. Ad esempio, per risolvere un problema di connessione, potrebbe essere richiesto di annotare il file di log con un testo specifico, tentare la connessione che non riusciva e quindi annotare nuovamente il file di log. Questa sequenza di eventi consente a un rappresentante dell'assistenza di ricercare nel file di log il testo inserito. Il testo racchiude tra parentesi le voci del file di log che tengono traccia del problema di connessione.
  • Impostare il livello di log - Il livello di log predefinito è Normale. Per un'operazione di risoluzione dei problemi, potrebbe essere richiesto di cambiare il livello di log in Dettagli.
  • Salvare i log - L'operazione di salvataggio con nome raccoglie tutti i file di log in un singolo file .zip e consente di specificare il percorso in cui posizionare il file.
