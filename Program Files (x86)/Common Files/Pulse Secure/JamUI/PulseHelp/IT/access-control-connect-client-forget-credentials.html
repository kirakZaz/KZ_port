Dimentica impostazioni salvate

Quando si esegue la connessione a una rete, è possibile selezionare la casella di controllo Salva impostazioni in modo tale che Pulse Secure Client memorizzi le credenziali di accesso. L'amministratore di rete può disabilitare questa funzione. Ogni schermata in cui viene richiesta l'immissione di dati contiene la casella di controllo Salva impostazioni. Quando si salvano le impostazioni, ai successivi tentativi di accesso non verrà più richiesto di fornire le corrispondenti informazioni. Se le credenziali di accesso cambiano, è tuttavia necessario cancellare le impostazioni salvate in Pulse Secure Client prima di salvare le nuove credenziali.

L'opzione Sospendi connessione non è disponibile se si sceglie Dimentica impostazioni salvate in presenza di una connessione attiva.

Se si seleziona Dimentica impostazioni salvate mentre una connessione è in stato di sospensione, la connessione viene interrotta.

Per cancellare le credenziali di accesso salvate:

  1. Fare clic con il pulsante destro del mouse su una delle connessioni per visualizzare il menu a comparsa.
  2. Fare clic su Dimentica impostazioni salvate. Pulse Secure Client cancella le impostazioni salvate per tutte le connessioni configurate.

Oppure

  1. Fare clic su File > Connessioni > Dimentica impostazioni salvate.
