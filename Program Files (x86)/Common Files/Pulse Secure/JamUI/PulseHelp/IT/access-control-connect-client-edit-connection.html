Modifica delle proprietà di una connessione

Dopo aver creato una connessione, è possibile modificarne l'URL e il nome visualizzato nel pannello Connessioni. È possibile modificare una connessione solo se tale connessione non è attiva ed è stata creata manualmente. Se la connessione è stata creata e installata dall'amministratore di Pulse Secure Client, non è possibile modificare il campo URL.

Se l'URL del server viene visualizzato come Selezionato a caso, significa che la connessione è stata definita dall'amministratore di Pulse Secure Client. Invece che con un server di destinazione, la connessione viene configurata con un elenco di server Pulse Secure Client che vengono selezionati in maniera casuale. Un pulsante Modifica viene visualizzato nei dettagli della connessione per ogni connessione che include un elenco di server. L'amministratore di Pulse Secure Client può configurare l'elenco in modo che Pulse Secure Client si connetta sempre al primo server dell'elenco, al server a cui si è effettuato l'accesso più di recente o a un server scelto casualmente dall'elenco. Non è possibile modificare l'URL di destinazione per una connessione che include un elenco di server.

Per modificare una connessione:

  1. Fare clic con il pulsante destro del mouse sulla connessione per visualizzare il menu a comparsa, quindi fare clic su Modifica.

    Viene visualizzata la finestra di dialogo Modifica connessione.

  2. Modificare la connessione, quindi fare clic su Salva per salvare le modifiche e chiudere la finestra di dialogo o fare clic su Connetti per avviare una connessione e chiudere la finestra di dialogo.

Oppure

  1. Fare clic sulla connessione per selezionarla.
  2. Fare clic sul pulsante Modifica connessione:

    Viene visualizzata la finestra di dialogo Modifica connessione.

  3. Modificare la connessione, quindi fare clic su Salva per salvare le modifiche e chiudere la finestra di dialogo o fare clic su Connetti per avviare una connessione e chiudere la finestra di dialogo.
