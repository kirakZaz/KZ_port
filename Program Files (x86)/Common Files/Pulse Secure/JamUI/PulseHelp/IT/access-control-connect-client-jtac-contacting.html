Richiesta di assistenza tecnica

L'assistenza tecnica è disponibile tramite il Pulse Secure Global Support Center (PSGSC).

  • Per informazioni sulla garanzia dei prodotti, visitare il sito Web:
    https://www.pulsesecure.net/support/

I centri PSGSC hanno risorse disponibili 24 ore su 24, 7 giorni alla settimana, 365 giorni l'anno.

Strumenti e risorse di self-help online

Per una risoluzione dei problemi semplice e rapida, Pulse Secure ha progettato un portale di assistenza self-service online denominato Pulse Secure Global Support Center (PSGSC) in grado di offrire le seguenti funzionalità:

  • Per le offerte PSGSC, consultare il sito Web:
    https://www.pulsesecure.net/support/

  • Per la documentazione dei prodotti, consultare il sito Web:
    https://www.pulsesecure.net/techpubs/

  • Le ultimi versioni del software e le note di rilascio sono disponibili per il download all'indirizzo:
    https://www.pulsesecure.net/support/

  • Per cercare bollettini tecnici di importanti notifiche hardware e software, consultare il sito Web
    https://www.pulsesecure.net/support/

  • È possibile verificare i diritti del servizio in base al numero di serie del prodotto utilizzando il nostro strumento SNE (Serial Number Entitlement) sul sito Web
    https://www.pulsesecure.net/support/

Apertura di una richiesta di assistenza con PSGSC

È possibile aprire una richiesta di assistenza con PSGSC sul Web o per telefono.

  • Utilizzare lo strumento di Gestione casi nel PSGSC all'indirizzo:
    https://pulsesecure.net/support

  • Chiamare il numero 1-844 751 7629 (numero verde negli Stati Uniti)

  • Per le opzioni di chiamata diretta o internazionale nei paesi in cui non è disponibile un numero verde, consultare l'indirizzo:
    https://www.pulsesecure.net/support/support-contacts/
