Utilizzo dell'icona di Pulse Secure Client nell'area di notifica e visualizzazione dello stato della connessione

L'area di notifica di Pulse Secure Client indica lo stato corrente della connessione di rete. Anche in caso di più connessioni correnti, viene visualizzata una sola icona nell'area di notifica. Se si verifica un problema in una connessione attiva, lo stato dell'icona della barra delle applicazioni cambia. Le seguenti icone segnalano lo stato della connessione.

Nessuna connessione.

Connessione in corso. Una connessione rimane in questo stato fino a quando non giunge a un esito positivo o negativo.

Sospeso.

Connesso con errori.

Connessione non riuscita.

Connesso.

Connesso alla rete locale ma nessun accesso a Internet disponibile. Le postazioni Wi-Fi pubbliche spesso implementano un captive portal che richiede l'immissione delle informazioni di autenticazione o l'accettazione dei termini di servizio da parte dell'utente affinché l'accesso sia consentito. Pulse Secure Client rileva la presenza di captive portal e non avvia la connessione a un server Pulse Secure Client finché l'accesso non viene consentito.

L'icona di Pulse Secure Client nell'area di notifica può essere utilizzata anche per eseguire le seguenti azioni:

  • Apri Pulse Secure: consente di visualizzare l'interfaccia di Pulse Secure Client.
  • Pulse Secure Client Collaboration: consente di accedere alle funzioni di riunione di Pulse Secure Client Collaboration Suite. Questo elemento viene visualizzato solo se l'amministratore di Pulse Secure Client ha abilitato Pulse Secure Client Collaboration.
  • Elenco connessioni: elenca tutte le connessioni Pulse Secure Client. Fare clic sul nome di una connessione per visualizzare le azioni che è possibile eseguire su di essa:
    • Connetti: consente di avviare una connessione.
    • Disconnetti: consente di interrompere la connessione di rete selezionata.
    • Annulla: consente di annullare un'azione avviata dal server Pulse. L'opzione Annulla è disponibile quando il server ha avviato un'operazione di Host Checker. Se si annulla l'operazione, la connessione alla rete non viene eseguita.
    • Sospendi: consente di sospendere una connessione VPN per permettere l'accesso alle risorse di rete locali. Questa opzione non è disponibile se si sceglie Dimentica impostazioni salvate mentre una connessione è attiva.
    • Riprendi: consente di riprendere una connessione VPN sospesa.
    • Estendi sessione: consente di estendere una sessione di connessione attiva senza attendere la notifica di sessione in scadenza da parte di Pulse. Questa opzione può essere selezionata in qualsiasi momento per reimpostare i contatori della sessione e di inattività sui relativi valori massimi.
  • Esci: consente di chiudere Pulse Secure Client e interrompere ogni connessione attiva.
