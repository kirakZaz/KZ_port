Salvataggio dei file di log

Nell'ambito della procedura di risoluzione di un problema, è possibile che venga richiesto di salvare i file di log di Pulse Secure Client. Pulse Secure Client include vari possibili componenti e ognuno di essi genera uno o più file di log. L'operazione di salvataggio con nome riunisce tutti i file di log di Pulse Secure Client e altri file diagnostici in un singolo file denominato LogsAndDiagnostics.zip.

Per salvare i file di log di Pulse Secure Client:

  1. Fare clic su File > Log > Salva con nome. Viene visualizzata la finestra di dialogo Salva con nome.
  2. Accettare i valori predefiniti relativi al percorso e al nome file o specificare nuovi valori, quindi fare clic su Salva.
