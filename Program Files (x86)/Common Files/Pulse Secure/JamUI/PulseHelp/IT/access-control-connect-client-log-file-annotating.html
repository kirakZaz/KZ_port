Annotazione dei file di log

Quando si annota un file di log, si inserisce un testo che contrassegna il file di log in una posizione specifica. Ad esempio, per risolvere un problema di connessione, potrebbe essere richiesto di annotare il file di log con un testo specifico, tentare la connessione che non riusciva e quindi annotare nuovamente il file di log. Questa sequenza di eventi consente a un tecnico dell'assistenza di individuare facilmente le voci del file di log che tengono traccia del problema di connessione.

Per annotare i file di log di Pulse Secure Client:

  1. Fare clic su File > Log > Annota per aprire la finestra di dialogo Annota log.
  2. Immettere il testo di annotazione, quindi fare clic su OK.
