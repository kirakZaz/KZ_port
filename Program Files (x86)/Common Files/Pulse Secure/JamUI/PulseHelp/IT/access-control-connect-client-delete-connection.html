Eliminazione di una connessione

Per eliminare una connessione:

  1. Fare clic sulla connessione per selezionarla.
  2. Fare clic sul pulsante Elimina:
  3. Prima di eliminare la connessione, viene richiesto di confermare la scelta.

Oppure

  1. Fare clic con il pulsante destro del mouse sulla connessione per visualizzare il menu a comparsa, quindi fare clic su Elimina.
  2. Prima di eliminare la connessione, viene richiesto di confermare la scelta.
