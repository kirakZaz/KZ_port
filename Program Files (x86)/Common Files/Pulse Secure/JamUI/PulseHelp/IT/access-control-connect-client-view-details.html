Visualizzazione dei dettagli di connessione avanzata

La finestra di dialogo Dettagli avanzati visualizza le informazioni dettagliate su una connessione selezionata.

Per visualizzare i dettagli avanzati di una connessione:

  1. Fare clic sulla connessione per selezionarla.
  2. Fare clic su File > Connessioni > Dettagli connessione avanzata.
  3. Le informazioni dei dettagli di una connessione non vengono aggiornate automaticamente. Ad esempio, la durata rimanente della sessione si riferisce al tempo residuo al momento dell'apertura della finestra di dialogo. Per aggiornare le informazioni dei dettagli avanzati, fare clic su Aggiorna o selezionare la casella di controllo Aggiornamento automatico.

La finestra Dettagli avanzati fornisce le seguenti informazioni:

Nome campo

Descrizione

Tempo sessione rimanente

Il tempo in cui la sessione VPN corrente resta attiva prima che sia necessario immettere nuovamente le credenziali o estendere la sessione manualmente.

Durata sessione

Il tempo di attività della sessione VPN corrente.

Tipo di tunnel

Indica se la connessione è un tunnel VPN o una corrispondenza porta/applicazione mediante SAM (Secure Access Manager).

Tipo di VPN

Il protocollo utilizzato per creare il tunnel (SSL o ESP).

IPv4 assegnato

L'indirizzo IPv4 assegnato all'adattatore virtuale Pulse.

Byte in entrata

Numero di byte ricevuti tramite il tunnel.

Byte in uscita

Numero di byte inviati tramite il tunnel.

Origine connessione

Descrive il modo in cui il client Pulse ha ricevuto la connessione:

  • Se il valore è Preconfigurato, la connessione deriva da un set di connessione scaricato da un gateway.
  • Se il valore è Utente aggiunto, la connessione è stata aggiunta manualmente dall'utente finale con l'interfaccia utente del client Pulse.
  • Se, infine, il valore è Dinamico, la connessione è stata ottenuta avviando il client Pulse mediante connessione di un browser Web a un gateway Pulse Secure e premendo il pulsante "Avvia" sulla pagina Web.

Nome Connessione impostata

È il nome del set di connessioni contenente la connessione. L'amministratore del sistema può utilizzare questo nome per trovare il set di connessioni corrispondenti sulla console di amministrazione del gateway appropriato (vedere il campo successivo) per vedere i dettagli del set di connessioni.

Proprietario del set di connessioni

È il nome del gateway Pulse Secure che ha modificato più recentemente il set di connessioni con questa connessione. Questo gateway e il nome del set di connessioni elencato in precedenza sono le due informazioni richieste per individuare la configurazione del set di connessioni del client in più gateway Pulse Secure.

Ultima modifica del set di connessioni

È l'ora in cui il set di connessioni del client Pulse è stato aggiornato sul gateway.

ID server del set di connessioni

Quando il client Pulse viene distribuito per la prima volta, riceve un set di connessioni collegate a un determinato ID server. Per tutte le connessioni successive a un gateway, il client scaricherà un set di connessioni aggiornato solo se il set di connessioni del gateway ha un ID server che corrisponde all'ID server del set di connessioni del client.

Host di download del set di connessioni

È il server dal quale il client Pulse ha aggiornato più recentemente il proprio set di connessioni.

Tempo di download del set di connessioni

È l'ora in cui il client Pulse ha modificato più recentemente il proprio set di connessioni.

I suddetti campi vengono visualizzati anche nella console di amministrazione di Pulse Connect Secure.
