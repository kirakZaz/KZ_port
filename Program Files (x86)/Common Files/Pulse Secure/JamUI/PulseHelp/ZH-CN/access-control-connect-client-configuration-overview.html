Pulse Secure 客户端配置概述

当您连接到网络时，Pulse Secure 客户端会将设置下载到端点。大多数 Pulse Secure 客户端配置任务由网络管理员执行。网络管理员还可以启用或禁用 Pulse Secure 客户端功能。例如，管理员可以将 Pulse Secure 客户端配置为自动连接到特定网络或允许您选择所需连接。

通常，网络管理员对您将访问的网络进行定义。网络管理员还可以让您来定义其他连接。如果您在办公室、家中和路上使用便携式计算机，则很有可能从办公室连接时需要一个连接，而在远程位置访问时需要另一个连接。然而，当您建立一种网络连接类型后，Pulse Secure 客户端可以自动进行调整以使您保持连接状态。例如，如果从远程位置连接到企业网络，然后在未注销的情况下关闭便携式计算机并回到办公室，则不必重新登录就可以继续进行计算会话。这种称为会话迁移的可移植功能取决于管理员配置网络访问的方式。
