Pulse Secure 客户端界面

Pulse Secure 客户端界面列出了您的 Pulse Secure 客户端连接。通常，网络管理员定义连接并将其部署至 Pulse 客户端。管理员也可以允许您为 Pulse Secure 客户端服务器创建新的连接。连接到 Pulse Secure 客户端服务器的 Web 门户即可向您的 Pulse Secure 客户端界面添加到该服务器的连接。

Pulse Secure 客户端还会显示一个提供连接状态的系统托盘图标，可让您连接和断开网络与会议，并可快速访问程序界面。

已连接到本地网络，但没有互联网访问可用。公共 WiFi 位置通常部署要求用户输入认证信息或接受服务条款的捕获门户，然后再授予网络访问权限。Pulse Secure 客户端检测到捕获门户的存在，直到获得互联网访问权限才可发起 Pulse Secure 客户端服务器连接。

Pulse Secure 客户端支持联邦信息处理标准 (FIPS)，该标准定义了适用于美国政府的安全通信实践。如果您的 Pulse Secure 客户端管理员启用了 FIPS，Pulse Secure 客户端窗口底部附近会显示 FIPS 开启。
