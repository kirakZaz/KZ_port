延长连接会话

网络管理员可以定义网络连接的会话超时限制。管理员可以定义最大会话限制和会话空闲限制。在达到限制时，Pulse Secure 客户端显示一条消息。该消息显示网络会话还有多少时间，Pulse Secure 客户端便会断开连接。如果不执行任何任何操作，Pulse Secure 客户端在会话计数器达到零时断开连接。如果单击“延长”，Pulse Secure 客户端将计时器重置为网络管理员定义的最大值，然后继续会话，不会出现服务中断。

您也可以在 Pulse Secure 客户端通知您会话即将到期之前延长会话。您可以随时延长会话，以便将会话和空闲计时器重置为其最大值。

“延长会话”功能不适用于“防火墙”连接。

要延长会话，请执行以下操作：

  1. 单击要延长的连接。
  2. 单击文件 > 连接 > 延长会话。

或者

  1. 在系统托盘中，右键单击 Pulse Secure 客户端图标以显示弹出菜单，单击连接，然后单击延长会话。

如果发现会话频繁受到会话超时的影响，请要求网络管理员增加会话生存期选项。
