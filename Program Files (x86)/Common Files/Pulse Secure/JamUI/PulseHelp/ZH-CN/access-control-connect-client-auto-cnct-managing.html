删除“自动连接”上的“手动替代”

Pulse Secure 客户端管理员可提供能够自动连接的 Pulse Secure 客户端连接。例如，在办公室并连接到 LAN 时，您可能有一个激活的 Pulse Secure 客户端连接；当您离开办公室时，会出现另一个激活的 Pulse Secure 客户端连接。自动连接通过限制手动连接决策简化连接过程。Pulse Secure 客户端管理员可允许您替代自动连接。对于不再自动响应的自动连接，Pulse Secure 客户端将显示手动替代。

要恢复已设置为手动替代的连接的自动行为，请执行以下操作：

  1. 单击连接以将其选中。
  2. 单击文件 > 连接 > 删除替代（返回自动控制）。

或者

  1. 在 Pulse Secure 客户端的“连接”窗格中，右键单击连接以显示弹出菜单。
  2. 单击删除替代（返回自动控制）。
