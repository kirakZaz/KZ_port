调整 Pulse Secure 客户端窗口大小

您可以调整 Pulse Secure 客户端窗口大小，也可以将其移动到屏幕上的另一个位置。如果随后将窗口最小化，它将保留新的大小和位置。但是，如果关闭程序，则在下次启动 Pulse Secure 客户端时，窗口将恢复为其默认大小和位置。

要调整 Pulse Secure 客户端窗口大小，请执行以下操作：

  1. 单击 Pulse Secure 客户端窗口右下角的大小调整手柄：
  2. 按住鼠标按钮并拖动窗口边缘，直到获得所需的窗口大小，然后松开鼠标按钮。
