Pulse Secure 客户端连接状态

Pulse Secure 客户端在“连接”窗格和系统托盘中显示连接状态。连接窗格可显示以下状态：

正在连接。连接将保持此状态，直到失败或成功为止。

已挂起。

已连接，但存在问题。

连接失败。

已连接。

已连接到本地网络，但没有互联网访问可用。公共 WiFi 位置通常部署要求用户输入认证信息或接受服务条款的捕获门户，然后再授予网络访问权限。Pulse Secure 客户端检测到捕获门户的存在，直到获得互联网访问权限才可发起 Pulse Secure 客户端服务器连接。

一个系统托盘图标指示所有活动 Pulse Secure 客户端连接的状态。您可右键单击该系统托盘图标，以控制 Pulse Secure 客户端连接、访问 Pulse 协作套件会议功能、打开 Pulse Secure 客户端界面或退出 Pulse Secure 客户端。下表显示了该系统托盘图标所指示的连接状态。

无连接。

正在连接。连接将保持此状态，直到失败或成功为止。

已挂起。

已连接，但存在问题。

连接失败。

已连接。

已连接到本地网络，但没有互联网访问可用。公共 WiFi 位置通常部署要求用户输入认证信息或接受服务条款的捕获门户，然后再授予网络访问权限。Pulse Secure 客户端检测到捕获门户的存在，直到获得互联网访问权限才可发起 Pulse Secure 客户端服务器连接。
