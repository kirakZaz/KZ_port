查看高级连接详细信息

“高级连接详细信息”对话框显示有关所选连接的详细信息。

查看高级连接详细信息：

  1. 单击连接以将其选中。
  2. 单击文件 > 连接 > 高级连接详细信息。
  3. 连接详细信息不会自动更新。例如，剩余会话时间显示打开对话框时所剩的时间量。要更新高级详细信息，请单击刷新或单击标签为自动刷新的复选框。

“高级连接详细信息”窗口提供以下信息：

字段名称

说明

剩余会话时间

在重新输入凭据或手动延长会话时间之前，当前 VPN 会话的剩余时间。

会话持续时间

当前 VPN 会话处于活动状态的持续时间。

隧道类型

说明连接是 VPN 隧道还是通过 SAM（安全访问管理器）进行的端口/应用程序映射。

VPN 类型

用于创建隧道的协议（SSL 或 ESP）。

分配的 IPv4

分配给 Pulse 虚拟适配器的 IPv4 地址。

传入字节数

通过隧道接收的字节数。

发出字节数

通过隧道发送的字节数。

连接源

说明 Pulse 客户端如何接收连接条目：

  • 如果是预配置的值，则连接条目来自从网关下载的连接集。
  • 如果是用户添加的值，则连接条目是由最终用户通过 Pulse 客户端 UI 手动添加的。
  • 如果是动态值，则意味着连接条目是在如下启动 Pulse 客户端时生成的：将网络浏览器连接到 Pulse Secure 网关，并按网页上的“开始”按钮。

连接集名称

这是包含所用连接条目的连接集的名称。系统管理员可以使用此名称在相应网关（参见下一个字段）的管理控制台上查找匹配的连接集，以查看连接集的详细信息。

连接集所有者

这是 Pulse Secure 网关的名称，该网关最近编辑了包含此连接条目的连接集。要在多个 Pulse Secure 网关上查找客户端的连接集配置，需要使用这个网关名称以及前面列出的连接集名称。

连接集上次修改时间

这是 Pulse 客户端的连接集上一次在网关上更新的时间。

连接集服务器 ID

Pulse 客户端首次部署时，会收到与特定服务器 ID 绑定的连接集。连接到网关后，只有该网关的连接集与客户端的连接集具有一致的服务器 ID，客户端才会下载更新的连接集。

连接集下载主机

这是 Pulse 客户端最近在其中更新了连接集的服务器。

连接集下载时间

这是 Pulse 客户端最近更新连接集的时间。
