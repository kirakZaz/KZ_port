编辑连接属性

创建连接后，可以对出现在“连接”窗格中的 URL 和名称进行编辑。仅当连接当前处于非活动状态且该连接必须是您手动创建的连接时，才能对其进行编辑。如果连接是由 Pulse Secure 客户端管理员创建并安装，则不可对 URL 字段进行编辑。

如果服务器 URL 显示为随机选择，则表示此连接已由 Pulse Secure 客户端管理员定义。此连接配置了一系列 Pulse Secure 客户端服务器（而非一台目标服务器），Pulse Secure 客户端将随机选择目标服务器。包含服务器列表的所有连接都将在连接详细信息中显示更改按钮。Pulse Secure 客户端管理员可配置该列表，以便 Pulse Secure 客户端始终连接到列表中的第一台服务器、您最近连接的服务器，或者连接到从列表中随机选择的服务器。对于包含服务器列表的连接，您无法编辑其目标 URL。

要编辑连接，请执行以下操作：

  1. 右键单击连接以显示弹出菜单，然后单击编辑。

    将打开“编辑连接”对话框。

  2. 编辑连接，然后单击保存以保存更改并关闭对话框，或者单击连接以发起连接并关闭对话框。

或者

  1. 单击连接以将其选中。
  2. 单击“编辑连接”按钮：

    将打开“编辑连接”对话框。

  3. 编辑连接，然后单击保存以保存更改并关闭对话框，或者单击连接以发起连接并关闭对话框。
