查看连接属性

要查看连接属性，请执行以下操作：

  1. 单击连接名称旁边的箭头。连接详细信息将出现在连接的下方。

如果服务器 URL 显示为随机选择，则表示此连接已由 Pulse Secure 客户端管理员定义。此连接配置了一系列 Pulse Secure 客户端服务器（而非一台目标服务器），Pulse Secure 客户端将随机选择目标服务器。更改按钮允许您从列表中选择特定服务器。包含服务器列表的所有连接都将显示更改按钮。Pulse Secure 客户端管理员可配置该列表，以便 Pulse Secure 客户端始终连接到列表中的第一台服务器、您最近连接的服务器，或者连接到从列表中随机选择的服务器。
