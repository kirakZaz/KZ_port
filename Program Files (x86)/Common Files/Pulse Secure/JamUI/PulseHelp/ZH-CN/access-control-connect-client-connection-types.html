Pulse Secure 客户端连接类型

定义新连接时 选择的连接类型与提供受保护网络资源访问权限的设备类型相关。手动添加新连接时，网络管理员可以告诉您选择哪种类型。

  • Policy Secure (UAC) 或 Connect Secure (VPN)—如果您正在连接 Pulse Connect Secure 或 Pulse Policy Secure，请使用此网络类型。
  • 防火墙 (SRX)—要连接到 Juniper Networks SRX 系列服务网关，请使用此网络类型。
