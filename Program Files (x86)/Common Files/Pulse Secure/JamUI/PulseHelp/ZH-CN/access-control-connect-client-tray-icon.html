使用 Pulse Secure 客户端托盘图标并查看连接状态

Pulse Secure 客户端托盘图标指示网络连接的当前状态。即使当前有多个连接，托盘中也只会显示一个连接图标。如果任何一个活动连接出现问题，系统托盘图标都会更改其状态。下列图标指示连接状态。

无连接。

正在连接。连接将保持此状态，直到失败或成功为止。

已挂起。

已连接，但存在问题。

连接失败。

已连接。

已连接到本地网络，但没有互联网访问可用。公共 WiFi 位置通常部署要求用户输入认证信息或接受服务条款的捕获门户，然后再授予网络访问权限。Pulse Secure 客户端检测到捕获门户的存在，直到获得互联网访问权限才可发起 Pulse Secure 客户端服务器连接。

您也可使用 Pulse Secure 客户端托盘图标来执行以下操作：

  • 打开 Pulse Secure 客户端 - 显示 Pulse Secure 客户端界面。
  • Pulse Secure 客户端协作 - 访问 Pulse Secure 客户端协作套件会议功能。此项目仅当 Pulse Secure 客户端管理员启用 Pulse Secure 客户端协作后显示。
  • 连接列表 - 列出每个 Pulse Secure 客户端连接。单击连接名称可显示您可以针对该连接执行的操作：
    • 连接 - 发起连接。
    • 断开连接 - 断开所选的网络连接。
    • 取消 - 取消由 Pulse Secure 客户端服务器发起的操作。当服务器发起“主机检查程序”操作时，“取消”选项可用。如果取消，则 Pulse Secure 客户端不会连接到网络。
    • 挂起 - 挂起 VPN 连接以便访问本地网络资源。注意，当您在具有活动连接时选择“忘记保存的设置”时，此选项则不可用。
    • 恢复 - 恢复已挂起 VPN 连接。
    • 延长会话 - 延长活动连接会话，而不等待 Pulse 通知您会话即使到期。您可以随时选择此选项，以便将会话和空闲计时器重置为其最大值。
  • 退出 - 关闭 Pulse Secure 客户端并断开任何活动连接的连接。
