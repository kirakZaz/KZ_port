访问 Pulse Secure 客户端协作会议功能

如果与已启用为 Pulse 协作套件服务器的服务器之间建立了 Pulse Secure 客户端连接，则可从 Pulse Secure 客户端托盘图标访问 Pulse 协作套件功能并加入会议、安排会议或退出会议。或者，也可通过 Web 门户登录到 Pulse Secure 客户端服务器，然后单击“会议”按钮以访问 Pulse 协作功能。

要访问 Pulse 协作选项，请执行以下操作：

  1. 右键单击 Windows 系统托盘中的 Pulse Secure 客户端图标以显示弹出菜单。
  2. 依次单击 Pulse 协作、会议服务器、启动会议。
  3. 如果出现提示，请提供登录凭证。如果此时存在已安排的会议，则 Pulse Secure 客户端将打开显示启动会议按钮的浏览器窗口。否则，请查看会议安排。在会议安排中选择会议，然后单击启动会议以打开 Pulse 协作套件客户端并参与会议。参与 Pulse 协作会议时，将下载和打开 Pulse 协作客户端。有关如何使用 Pulse 协作套件的详情，请单击 Pulse 协作界面中的“帮助”链接。

    如果您的管理员已启用 Microsoft Outlook 集成，单击链接以下载和安装 Microsoft Outlook 插件。插件允许您从 Outlook 中安排会议和使用 Outlook 电子邮件发送邀请给参加者。

    单击安排会议以安排会议和邀请参加者。

    按需单击即时会议 或支持会议 （如果管理员已启用这些会议类型）。即时会议与标准会议类似。它允许您通过自动应用角色级设置（例如，认证要求、远程控制和安全交谈）快速创建会议。支持会议是两个人参与的会议，主要是为了进行在线故障排除。支持会议允许桌面共享和远程控制操作。

    浏览器窗口顶部的“会议”按钮允许您从 Web 门户中的任意位置返回到会议安排。

    有关 Pulse 协作功能的更多信息，请点击浏览器窗口中的“帮助”按钮。

要与会议断开连接，请执行以下操作：

  1. 依次单击 Pulse 协作和会议服务器，然后单击登出。

    使用托盘图标登出菜单项目与通过 Pulse 协作客户端界面登出所完成的操作相同。
