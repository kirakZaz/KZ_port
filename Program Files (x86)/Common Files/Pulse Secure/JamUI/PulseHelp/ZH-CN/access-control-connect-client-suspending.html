访问本地网络资源（挂起连接）

具有活动 VPN 连接时，可能无法访问本地网络资源（诸如打印机和共享驱动器）。您可挂起 VPN 连接以连接至本地资源，然后在无需提供登录凭证的条件下重新激活远程网络的 VPN 连接。

如果 Pulse Secure 客户端管理员已启用当地子网访问权限，您无需挂起 Pulse Secure 客户端连接即可访问与您计算机连接同一子网的资源，如打印机。

要挂起 VPN 连接并访问本地网络资源，请执行以下操作：

  1. 单击活动连接以将其选中，然后单击 Pulse Secure 客户端菜单栏上的文件 > 连接 > 挂起。

    具有活动连接时，如果您选择“忘记保存的设置”，则此连接选项不可用。

    或者

    右键单击活动的连接，然后单击挂起。

    或者

    单击 Windows 系统托盘中的 Pulse Secure 客户端图标以显示弹出菜单，单击活动的连接，然后单击挂起。

    对于已挂起的连接，“Pulse Secure 客户端连接”窗格中的按钮标签变为恢复，“已挂起”图标显示在连接旁边：

    连接挂起后，托盘图标显示“已挂起”状态：

要重新激活已挂起的 VPN 连接，请执行以下操作：

  1. 选择已挂起连接，然后单击文件 > 连接 > 恢复。

    或者

    单击恢复。

    或者

    单击 Windows 系统托盘中的 Pulse Secure 客户端图标以显示弹出菜单，单击已挂起连接，然后单击恢复。
