辅助功能 / 第 508 节合规性

Pulse Secure 桌面客户端中包含多种有助于身体、感官或认知残障人士浏览该桌面客户端用户界面的功能。

其中的某些功能需要设置注册表项才能使用。应在以下位置设置这些项：

HKEY_CURRENT_USER\Software\Pulse Secure\Pulse\Accessibility

辅助功能包括：

  • 支持使用 Tab 切换 - 这款桌面客户端的 UI 支持使用键盘上的 Tab 键在 UI 小组件之间切换。所以无需使用鼠标，即可轻松地浏览用户界面。
  • 其他导航键 -
    • 同时按 Alt 键和快捷键可访问大多数数据输入字段和对话框控件。按下然后松开 Alt 键可显示要使用的特定快捷键。
    • 还可同时按 Alt 键和菜单快捷键显示 Pulse 菜单。进入菜单系统后，可使用 Alt 快捷组合键或箭头键和 Enter 键浏览和选择菜单项。
    • 在对话框中，Tab + 箭头键这个常用组合可用于导航。
  • 支持屏幕阅读器 - 可使用 JAWS 屏幕阅读器读出连接状态。要启用屏幕阅读器功能，您或系统管理员必须将注册表项 ScreenreaderMode设置为 1。’
  • 使用系统颜色设置 - Pulse 桌面客户端的 UI 支持以高对比度模式显示 Windows 操作系统的系统颜色。要启用此功能，您或系统管理员必须将注册表项 UseSystemColors 设置为 00000001。
