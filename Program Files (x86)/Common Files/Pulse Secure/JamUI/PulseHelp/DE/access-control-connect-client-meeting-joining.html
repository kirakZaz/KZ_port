Auf Konferenzfunktionen von Pulse Secure Client zugreifen

Wenn der für die Pulse Secure Client-Verbindung verwendete Server als Pulse Secure Client Collaboration Suite-Server aktiviert wurde, können Sie über das Taskleistensymbol auf die Funktionen der Pulse Secure Client Collaboration Suite zugreifen und an einer Konferenz teilnehmen, diese planen oder sich von ihr abmelden. Alternativ können Sie sich über ein Webportal am Pulse Secure Client-Server anmelden und anschließend auf die Schaltfläche „Konferenzen“ klicken, um auf die Funktionen von Pulse Secure Client Collaboration-Konferenzen zuzugreifen.

So greifen Sie auf die Konferenzoptionen von Pulse Secure Client zu:

  1. Klicken Sie mit der rechten Maustaste in der Windows-Taskleiste auf das Pulse Secure Client-Symbol, um das Popupmenü aufzurufen.
  2. Klicken Sie auf Pulse Secure Client Collaboration, den Konferenzserver und anschließend auf Konferenz beginnen.
  3. Wenn Sie dazu aufgefordert werden, geben Sie Ihre Anmeldeinformationen ein. Wenn für diese Zeit eine Konferenz geplant ist, öffnet Pulse Secure Client ein Browser-Fenster, in dem die Schaltfläche Konferenz beginnen angezeigt wird. Wenn dies nicht der Fall ist, zeigen Sie den Konferenzplan an. Wählen Sie im Konferenzplan die Konferenz aus, und klicken Sie anschließend auf Konferenz beginnen, um den Pulse Secure Client Collaboration Suite-Client zu öffnen und an der Konferenz teilzunehmen. Wenn Sie an einer Pulse Secure Client-Konferenz teilnehmen, wird der Pulse Secure Client Collaboration-Client heruntergeladen und geöffnet. Um Einzelheiten zur Bedienung der Pulse Secure Client Collaboration Suite zu erhalten, klicken Sie in der Pulse Collaboration-Benutzeroberfläche auf den Link „Hilfe“.

    Wenn Ihr Administrator die Microsoft Outlook-Integration aktiviert hat, klicken Sie auf den Link, um das Microsoft Outlook-Plugin herunterzuladen und zu installieren. Mit dem Plugin können Sie Konferenzen in Outlook planen und den Teilnehmern per Outlook E-Mail-Einladungen zusenden.

    Klicken Sie auf Konferenz planen, um eine Konferenz zu planen und Teilnehmer einzuladen.

    Klicken Sie je nach Bedarf auf Sofortkonferenz oder Supportkonferenz (wenn Ihr Administrator diese Konferenztypen aktiviert hat). Eine Sofortkonferenz gleicht einer Standardkonferenz. Sie können dabei schnell eine Konferenz organisieren, indem Sie automatisch Ihre Einstellungen auf Rollenebene anwenden, z. B. Authentifizierungsanforderungen, Remotesteuerung und sicheren Chat. Eine Supportkonferenz ist eine Konferenz für zwei Personen, die in erster Linie einer Online-Fehlerbehebung dient. In einer Supportkonferenz können Sie Ihren Desktop freigeben und ihn fernsteuern lassen.

    Über die Schaltfläche „Konferenzen“ oben im Browser-Fenster können Sie von überall im Webportal zum Konferenzplan zurückkehren.

    Wenn Sie weitere Informationen zu Pulse Collaboration-Funktionen benötigen, klicken Sie im Browserfenster auf die Schaltfläche „Hilfe“.

So trennen Sie die Verbindung zu einem Meeting:

  1. Klicken Sie auf Pulse Collaboration und den Konferenzserver und anschließend auf Abmelden.

    Sie können sich auch von der Pulse Secure Client Collaboration Client-Benutzeroberfläche abmelden, indem Sie das Symbol auf der Taskleiste im Abmeldemenü verwenden.
