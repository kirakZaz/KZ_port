Gespeicherte Einstellungen vergessen

Beim Herstellen einer Verbindung zu einem Netzwerk können Sie das Kontrollkästchen „Einstellungen speichern“ aktivieren, damit Pulse Secure Client Ihre Anmeldeinformationen speichert. (Beachten Sie, dass der Netzwerkadministrator diese Funktion deaktivieren kann.) Jeder Bildschirm mit einer Eingabeaufforderung verfügt über ein eigenes Kontrollkästchen „Einstellungen speichern“. Sie werden nach dem Speichern der Einstellungen aufgefordert, Informationen zu aufeinander folgenden Anmeldeversuchen bereitzustellen. Wenn sich allerdings Ihre Anmeldeinformationen ändern, können Sie die neuen Anmeldeinformationen erst speichern, nachdem Sie die gespeicherten Einstellungen in Pulse Secure Client gelöscht haben.

Die Option „Verbindung unterbrechen“ ist nicht verfügbar, wenn Sie bei einer aktiven Verbindung „Gespeicherte Einstellungen vergessen“ auswählen.

Wenn Sie „Gespeicherte Einstellungen vergessen“ auswählen, während eine Verbindung unterbrochen ist, wird die Verbindung getrennt.

So löschen Sie gespeicherte Anmeldeinformationen:

  1. Klicken Sie mit der rechten Maustaste auf eine beliebige Verbindung, um das Popupmenü anzuzeigen.
  2. Klicken Sie auf Gespeicherte Einstellungen vergessen. Pulse Secure Client löscht die gespeicherten Einstellungen für alle konfigurierten Verbindungen.

Oder

  1. Klicken Sie auf Datei > Verbindungen > Gespeicherte Einstellungen vergessen.
