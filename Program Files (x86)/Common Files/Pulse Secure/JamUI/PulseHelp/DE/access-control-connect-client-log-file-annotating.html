Protokolldateien mit Anmerkungen versehen

Wenn Sie einer Protokolldatei eine Anmerkung hinzufügen, fügen Sie Text als Markierung an einer bestimmten Position in der Protokolldatei ein. Zur Behebung eines Verbindungsproblems können Sie beispielsweise aufgefordert werden, spezifischen Text als Anmerkung in die Protokolldatei einzufügen, die fehlerhafte Verbindung zu testen und erneut eine Anmerkung in die Protokolldatei einzufügen. Die Ereignisabfolge ermöglicht einem Supporttechniker das einfache Finden der Protokolldateieinträge, in denen das Verbindungsproblem protokolliert wird.

So fügen Sie Pulse Secure Client-Protokolldateien Anmerkungen hinzu:

  1. Klicken Sie auf Datei > Protokolle > Mit Anmerkungen versehen, um das Dialogfeld „Mit Anmerkungen versehen“ zu öffnen.
  2. Geben Sie den Anmerkungstext ein, und klicken Sie dann auf OK.
