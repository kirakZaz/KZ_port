Protokolldateien speichern

Als Teil des Fehlerbehebungsprozesses werden Sie gegebenenfalls aufgefordert, die Pulse Secure Client-Protokolldateien zu speichern. Pulse Secure Client enthält zahlreiche mögliche Komponenten. Jede Komponente erstellt eine oder mehrere Protokolldateien. Durch „Speichern unter“ werden alle Log-Dateien und sonstige Diagnosedateien von Pulse Secure in einer Datei mit dem Namen LogsAndDiagnostics.zip gespeichert.

So speichern Sie Pulse Secure Client-Protokolldateien:

  1. Klicken Sie auf Datei > Protokolle > Speichern unter. Das Dialogfeld „Speichern unter“ wird angezeigt.
  2. Akzeptieren Sie entweder die Standardwerte für den Standort und Dateinamen, oder geben Sie neue Werte an, und klicken Sie auf Speichern.

Verwandte Themen

  • Protokolldateien mit Anmerkungen versehen
  • Protokollebene festlegen
  • Protokolldateien hochladen
