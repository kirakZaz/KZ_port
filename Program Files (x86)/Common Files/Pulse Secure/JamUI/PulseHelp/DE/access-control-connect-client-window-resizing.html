Größe des Pulse Secure Client-Fensters anpassen

Sie können die Größe des Pulse Secure Client-Fensters anpassen und es an eine andere Position auf Ihrem Bildschirm verschieben. Wenn Sie das Fenster dann minimieren, behält es seine neue Größe und Position bei. Sollten Sie das Programm jedoch schließen, kehrt das Fenster beim nächsten Start von Pulse Secure Client zu seiner Standardgröße und -position zurück.

So ändern Sie die Größe des Pulse Secure Client-Fensters:

  1. Klicken Sie auf den Anfasser für Größe unten rechts im Pulse Secure Client-Fenster:
  2. Halten Sie die Maustaste gedrückt, und ziehen Sie den Rand des Fensters, bis Sie die gewünschte Größe erreicht haben. Lassen Sie die Maustaste dann los.
