Informationen zur Reparatur einer Verbindung anzeigen

Wenn ein Verbindungsversuch fehlschlägt oder bei einer aktiven Verbindung ein Problem auftritt, zeigt Pulse Secure Client Informationen zur Reparatur an. Die Informationen zur Reparatur enthalten Anweisungen zum Beheben des Problems oder geben die automatisch initiierten Prozeduren zur Reparatur an.

So zeigen Sie Informationen zur Reparatur an:

  1. Klicken Sie auf eine Verbindung, um sie auszuwählen.
  2. Klicken Sie auf Datei > Verbindungen > Informationen zur Korrektur, um das Dialogfeld „Informationen zur Korrektur“ anzuzeigen.

    Liegen keine aktiven oder ausstehenden Reparaturaktionen an, ist das Menü „Reparatur“ abgeblendet.

  3. Der obere Teil des Dialogfelds beschreibt das Problem, der untere Teil die Schritte zum Beheben des Problems. Führen Sie die Schritte zur Reparatur aus, und klicken Sie auf Erneut versuchen, um die Verbindung wiederherzustellen.

Oder

  1. Klicken Sie auf den Pfeil neben einer Verbindung, um die Verbindung zu erweitern und den Verbindungsstatus anzuzeigen.
  2. Klicken Sie im Verbindungsstatus auf den Link Details, um detailliertere Informationen zum Problem anzuzeigen.

Oder

  1. Klicken Sie mit der rechten Maustaste auf die Verbindung, um das Popupmenü anzuzeigen, und klicken Sie auf Informationen zur Korrektur, um das Reparaturdialogfeld anzuzeigen.
  2. Führen Sie die Schritte zur Reparatur aus, und klicken Sie auf Erneut versuchen, um die Verbindung wiederherzustellen.
