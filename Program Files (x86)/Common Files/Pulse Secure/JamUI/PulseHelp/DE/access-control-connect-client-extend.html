Verbindungssitzung verlängern

Der Netzwerkadministrator definiert das Sitzungszeitlimit der Netzwerkverbindungen. Der Administrator definiert das maximale Sitzungslimit und das Leerlaufzeitlimit. Sie werden von Pulse Secure Client über eine Meldung benachrichtigt, wenn Sie ein Limit erreichen. Die Meldung enthält die verbleibende Zeit in der Netzwerksitzung, bevor Pulse Secure Client die Verbindung trennt. Wenn Sie keine Aktion ausführen, trennt Pulse Secure Client die Verbindung, wenn der Sitzungszähler Null erreicht. Wenn Sie auf „Verlängern“ klicken, wird der Zähler auf den vom Administrator definierten Wert zurückgesetzt, und die Verbindung wird ohne Dienstunterbrechung fortgesetzt.

Sie können eine Sitzung auch verlängern, ohne darauf zu warten, dass Pulse Secure Client Sie über eine ablaufende Sitzung benachrichtigt. Sie können die Sitzung jederzeit verlängern, um das Sitzungs- und Leerlaufzeitlimit auf die Maximalwerte zurückzusetzen.

Die Funktion „Sitzung verlängern“ gilt nicht für Firewall-Verbindungen.

So verlängern Sie eine Sitzung:

  1. Klicken Sie auf die Verbindung, die Sie verlängern möchten.
  2. Klicken Sie auf Datei > Verbindungen > Sitzung verlängern.

Oder

  1. Klicken Sie in der Taskleiste mit der rechten Maustaste auf das Pulse Secure Client-Symbol, um das Popupmenü zu öffnen. Klicken Sie anschließend auf Sitzung verlängern.

Wenn Ihre Sitzungen häufig durch Zeitüberschreitungen unterbrochen werden, bitten Sie Ihren Netzwerk-Administrator, die Optionen für die Sitzungslebensdauer zu verlängern.
