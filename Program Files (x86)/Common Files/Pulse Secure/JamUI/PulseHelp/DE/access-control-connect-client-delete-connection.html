Verbindung löschen

So löschen Sie eine Verbindung:

  1. Klicken Sie auf eine Verbindung, um sie auszuwählen.
  2. Klicken Sie auf die Schaltfläche „Löschen“.
  3. Sie werden aufgefordert, Ihre Entscheidung zu bestätigen, bevor die Verbindung gelöscht wird.

Oder

  1. Klicken Sie mit der rechten Maustaste auf die Verbindung, um das Popupmenü anzuzeigen, und klicken Sie dann auf Löschen.
  2. Sie werden aufgefordert, Ihre Entscheidung zu bestätigen, bevor die Verbindung gelöscht wird.
