Pulse Secure Client-Verbindungsstatus

In Pulse Secure Client wird der Status einer Verbindung im Verbindungsbereich und in der Taskleiste angezeigt. Im Verbindungsbereich können die folgenden Status angezeigt werden:

Verbindung wird hergestellt. Eine Verbindung bleibt in diesem Zustand, bis das Herstellen der Verbindung fehlschlägt oder erfolgreich durchgeführt wurde.

Unterbrochen.

Verbunden mit Problemen.

Verbindungsherstellung fehlgeschlagen.

Verbunden.

Verbunden mit dem lokalen Netzwerk, aber kein Internetzuggriff verfügbar. An öffentlichen WLAN-Standorten wird häufig ein Captive-Portal bereitgestellt, in dem der Benutzer zunächst Informationen zur Authentifizierung eingeben oder die Servicebedingungen akzeptieren muss, bevor ein Zugriff auf das Netzwerk gewährt wird. Pulse Secure Client erkennt vorhandene Captive-Portale und startet erst dann eine Verbindung mit einem Pulse Secure Client-Server, wenn der Internetzugriff gewährt wurde.

Ein einzelnes Taskleistensymbol zeigt den Status aller aktiven Pulse Secure Client-Verbindungen an. Klicken Sie mit der rechten Maustaste auf das Taskleistensymbol, um Pulse Secure Client-Verbindungen zu steuern, auf die Funktionen der Pulse Secure Client Collaboration Suite-Sitzung zuzugreifen, die Pulse Secure Client-Benutzeroberfläche zu öffnen oder Pulse Secure Client zu beenden.
