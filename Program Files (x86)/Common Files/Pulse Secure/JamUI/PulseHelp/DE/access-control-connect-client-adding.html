Verbindung hinzufügen

Jede Verbindung in Pulse Secure Client stellt ein geschütztes Netzwerk dar. In der Regel definiert der Netzwerkadministrator die Verbindungen für Sie und deaktiviert möglicherweise die Funktion „Verbindung hinzufügen“. Falls Sie neue Verbindungen erstellen müssen, erfahren Sie die zu verwendenden Einstellungen vom Netzwerkadministrator.

So fügen Sie eine neue Verbindung hinzu:

  1. Klicken Sie auf die Schaltfläche Verbindung hinzufügen:

    Das Dialogfeld „Verbindung hinzufügen“ wird geöffnet.

  2. Der Netzwerkadministrator teilt Ihnen in der Regel mit, welche Option Sie für „Typ“ auswählen sollen. Wählen Sie eine der folgenden Optionen aus:
    • Policy Secure (UAC) oder Connect Secure (VPN) – Verwenden Sie diesen Netzwerktyp beim Herstellen einer Verbindung mit Pulse Policy Secure oder Pulse Connect Secure.
    • Firewall SRX – Verwenden Sie diesen Netzwerktyp beim Herstellen einer Verbindung zu einem Pulse Secure SRX Series-Gerät.
  3. Geben Sie unter „Name“ einen beschreibenden Namen für diese Verbindung ein. Der angegebene Name wird auf der Benutzeroberfläche von Pulse Secure Client angezeigt.
  4. Geben Sie als Server-URL das Netzwerk an, zu dem Sie eine Verbindung herstellen möchten. Sie können den Wert der Server-URL in einem der folgenden Formate eingeben:
    • Eine IP-Adresse, zum Beispiel 10.204.71.86. Pulse Secure Client unterstützt auch IPv6-Adressen. IPv6-Adressen werden in der Regel nicht manuell eingegeben. Wenn dies aber doch der Fall ist, verwenden Sie das folgende Format: fd29:64fc:bd56:5dbd:3:23:202:1a31. Eine IPv6-Adresse kann die Zahlen 0 bis 9 und die Buchstaben a bis f enthalten. Führende Nullen können ausgelassen werden.
    • Ein DNS-Name, zum Beispiel server.meinunternehmen.net.
  5. Klicken Sie auf Hinzufügen, um Ihre Verbindung zu speichern und das Dialogfeld zu schließen. Klicken Sie auf Verbinden, um die neue Verbindung zu speichern und eine Verbindung zum Netzwerk zu initiieren.
