Von einer aktiven Verbindung trennen

So trennen Sie eine Netzwerkverbindung:

  1. Klicken Sie für die Verbindung, die Sie trennen möchten, auf Verbindung trennen.

    Oder

    Klicken Sie in der Taskleiste mit der rechten Maustaste auf das Pulse Secure Client-Symbol, um das Popupmenü zu öffnen. Klicken Sie anschließend auf Verbindung trennen.
