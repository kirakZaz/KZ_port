Pulse Secure Client-Übersicht

Pulse Secure Client ermöglicht sichere, authentifizierte Verbindungen zu geschützten Netzwerkressourcen über LANs und WANs. Pulse Secure Client umfasst zudem die Pulse Secure Client Collaboration Suite für Onlinekonferenzen. Pulse Secure Client vereinfacht Netzwerkverbindungen, da die Netzwerkzugriffsumgebung durch den Netzwerkadministrator konfiguriert wird.

In einer sicheren Verbindung wird der Netzwerkverkehr verschlüsselt, so dass nur das Endgerät und der Server, mit denen kommuniziert wird, den Inhalt entschlüsseln können. Ihr Netzwerkadministrator definiert die Sicherheitseinstellungen, die von Pulse Secure Client beim Herstellen einer Netzwerkverbindung verwendet werden.

Eine authentifizierte Verbindung bedeutet, dass das Endgerät und der Server ihre Identität gegenseitig nachgewiesen haben. Der Server verwendet ein Zertifikat von einer vertrauenswürdigen Zertifizierungsstelle zur eigenen Authentifizierung. Das Endgerät unterbreitet dem Server Ihre Anmeldeinformationen. Ihr Netzwerkadministrator kann Ihre Umgebung so konfigurieren, dass die Authentifizierung automatisch erfolgt und Sie dabei keine Informationen eingeben müssen. Typischerweise werden Sie beim Einrichten einer Verbindung jedoch zur Eingabe Ihres Benutzernamens und Ihres Passworts aufgefordert.

Bei einer geschützten Netzwerkressource handelt es sich um ein beliebiges Netzwerkgerät oder eine Anwendung, auf das/die Sie über eine authentifizierte Verbindung zugreifen. Um z. B. auf das E-Mail-System des Unternehmens zuzugreifen, melden Sie sich erst am Netzwerk an. In Pulse Secure Client können Sie darüber hinaus den Pulse Collaboration Suite Client starten, um an Konferenzen teilzunehmen. Der Netzwerkadministrator organisiert geschützte Netzwerkressourcen in Bereichen und autorisiert Ihre Nutzung dieser Ressourcen, indem er Ihnen Zugriff auf ausgewählte Bereiche gewährt. Wenn Sie Zugriff auf mehrere Bereiche haben, können Sie beim Einrichten einer Netzwerkverbindung über Pulse Secure Client auswählen, welchen Bereich Sie verwenden wollen. Der Netzwerkadministrator kann außerdem die Sicherheitseigenschaften definieren, die für den Zugriff auf einen Bereich erforderlich sind, indem er das Host Checker-Programm aktiviert. Host Checker wird ausgeführt, wenn Sie über Pulse Secure Client eine Verbindung zum Netzwerk herstellen. Das Programm untersucht Ihr Endgerät, um sicherzustellen, dass er über die erforderliche Software verfügt, wie z. B. aktuelle Antivirusdefinitionen und Malwareschutz. Der Administrator kann Ihre Pulse Secure Client-Verbindungen darüber hinaus so konfigurieren, dass nur Zugriff auf bestimmte netzwerkbasierte Anwendungen gewährt wird.

Der Netzwerkadministrator kann Zugriffsrichtlinien so konfigurieren, dass Ihre Verbindung nach dem Herstellen der Netzwerkverbindung portabel ist. Falls Sie z. B. mit Ihrem Laptop zuhause eine Verbindung zum Unternehmensnetzwerk herstellen, müssen Sie diese nicht zuerst trennen und dann eine neue Verbindung herstellen, wenn Sie den Laptop wieder im Büro verwenden. Sie schließen den Laptop einfach zuhause. Wenn Sie ihn im Büro dann wieder öffnen, erkennt Pulse Secure Client die veränderte Netzwerkumgebung. Pulse Secure verwendet die Informationen in Ihrem Verbindungsprofil, um den passenden Zugriff für Ihre Verbindung zu konfigurieren und authentifiziert Sie in der neuen Umgebung. Wenn Ihr Administrator die Standorterkennungsfunktion von Pulse Secure Client aktiviert hat, werden Sie beim Start Ihres Endgeräts automatisch mit dem passenden Netzwerk verbunden. Wenn Ihr Gerät z. B. außerhalb des Büros gestartet wird, richtet Pulse Secure Client automatisch eine Verbindung zu einem Gerät mit WLAN-Zugriff ein.

Mit Pulse Secure Client kann der Netzwerkadministrator neue Software und wichtige Softwareaktualisierungen an das Endgerät „pushen“, wenn eine Netzwerkverbindung hergestellt wird.
