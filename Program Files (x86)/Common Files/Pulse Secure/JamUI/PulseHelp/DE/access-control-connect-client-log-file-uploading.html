Protokolldateien hochladen

Mit dem Pulse Secure-Desktopclient können Pakete mit Diagnoseprotokollen problemlos zur Analyse durch einen Systemadministrator an PCS-Gateways übertragen werden. Zum Senden eines Protokollpakets bei einer aktiven VPN-Verbindung klicken Sie in der Benutzeroberfläche des Desktopclients auf: Datei > Protokolle > Hochladen.

Sie müssen den Server, an den die Protokolle gesendet werden sollen, auswählen. Ein Dialog mit dem Fortschritt des Vorgangs zum Hochladen wird angezeigt.

Ihr Systemadministrator muss diese Funktion auf der Serverseite aktivieren, bevor ein Endbenutzer Protokolldateien in das Pulse Secure-Gateway hochladen kann.

Der Server verfügt über eine begrenzte Größe von 200 MB für die gesamten hochgeladenen Client-Protokolle.

Das Pulse Connect Secure-Gateway verfügt über eine begrenzte Festplattenpartition zum Speichern von hochgeladenen Protokolldateien. Wenn dieser Speicherplatz mit anderen Protokolldateien belegt ist, können Sie möglicherweise keine Dateien hochladen. In diesem Fall müssen Sie sich an Ihren Systemadministrator wenden, damit dieser zum Hochladen Ihrer Protokolle Speicherplatz freigeben kann.

Verwandte Themen

  • Protokolldateien mit Anmerkungen versehen
  • Protokollebene festlegen
  • Protokolldateien speichern
