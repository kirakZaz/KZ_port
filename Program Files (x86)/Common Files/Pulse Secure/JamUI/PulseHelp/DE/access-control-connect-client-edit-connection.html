Verbindungseigenschaften bearbeiten

Nach dem Herstellen einer Verbindung können Sie die URL und den Namen bearbeiten, der im Verbindungsbereich angezeigt wird. Verbindungen können nur bearbeitet werden, wenn sie gerade nicht aktiv sind und Sie sie manuell erstellt haben. Wurde eine Verbindung von Ihrem Pulse Secure Client-Administrator erstellt und installiert, können Sie das URL-Feld nicht bearbeiten.

Wenn für die Server-URL Zufällig ausgewählt angezeigt wird, bedeutet das, dass diese Verbindung vom Pulse Secure Client-Administrator definiert wurde. Für diese Verbindung wird anstelle eines einzelnen Zielservers eine Liste von Pulse Secure Client-Servern konfiguriert. Pulse Secure Client wählt dann per Zufallsauswahl den Zielserver aus. Für jede Verbindung mit einer Liste von Servern wird die Schaltfläche Ändern angezeigt. Ihr Pulse Secure Client-Administrator kann die Liste so konfigurieren, dass Pulse Secure Client immer eine Verbindung zum ersten Server in der Liste einrichtet, oder zum Server, mit dem Sie zuletzt verbunden waren, oder zu einem nach dem Zufallsprinzip ausgewählten Server. Sie können die Ziel-URL einer Verbindung mit einer Liste von Servern nicht ändern.

So bearbeiten Sie eine Verbindung:

  1. Klicken Sie mit der rechten Maustaste auf die Verbindung, um das Popupmenü anzuzeigen, und klicken Sie dann auf Bearbeiten.

    Das Dialogfeld „Verbindung bearbeiten“ wird geöffnet.

  2. Bearbeiten Sie die Verbindung, und klicken Sie dann auf Speichern, um die Änderungen zu speichern und das Dialogfeld zu schließen, oder klicken Sie auf Verbinden, um eine Verbindung herzustellen und das Dialogfeld zu schließen.

Oder

  1. Klicken Sie auf eine Verbindung, um sie auszuwählen.
  2. Klicken Sie auf die Schaltfläche „Verbindung bearbeiten“:

    Das Dialogfeld „Verbindung bearbeiten“ wird geöffnet.

  3. Bearbeiten Sie die Verbindung, und klicken Sie dann auf Speichern, um die Änderungen zu speichern und das Dialogfeld zu schließen, oder klicken Sie auf Verbinden, um eine Verbindung herzustellen und das Dialogfeld zu schließen.
