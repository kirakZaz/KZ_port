Pulse Secure Client-Taskleistensymbol verwenden und Verbindungsstatus anzeigen

Das Taskleistensymbol von Pulse Secure Client gibt den aktuellen Status der Netzwerkverbindung an. Es erscheint immer nur ein Verbindungssymbol in der Taskleiste, auch wenn derzeit mehr als eine Verbindung besteht. Bei einem Problem mit einer aktiven Verbindung ändert das Taskleistensymbol seinen Status. Die folgenden Symbole zeigen den Verbindungsstatus an.

Keine Verbindung.

Verbindung wird hergestellt. Eine Verbindung bleibt in diesem Zustand, bis das Herstellen der Verbindung fehlschlägt oder erfolgreich durchgeführt wurde.

Unterbrochen.

Verbunden mit Problemen.

Verbindungsherstellung fehlgeschlagen.

Verbunden.

Verbunden mit dem lokalen Netzwerk, aber kein Internetzuggriff verfügbar. An öffentlichen WLAN-Standorten wird häufig ein Captive-Portal bereitgestellt, in dem der Benutzer zunächst Informationen zur Authentifizierung eingeben oder die Servicebedingungen akzeptieren muss, bevor ein Zugriff auf das Netzwerk gewährt wird. Pulse Secure Client erkennt vorhandene Captive-Portale und startet erst dann eine Verbindung mit einem Pulse Secure Client-Server, wenn der Internetzugriff gewährt wurde.

Sie können über das Taskleistensymbol von Pulse Secure Client auch folgende Aktionen durchführen:

  • Pulse Secure Client öffnen – Zeigt die Pulse Secure Client-Benutzeroberfläche an.
  • Pulse Secure Client Collaboration – Ruft die Konferenzfunktionen von Pulse Secure Client Collaboration Suite auf. Diese Option wird nur angezeigt, wenn Pulse Secure Client Collaboration vom Pulse Secure Client-Administrator aktiviert wurde.
  • Verbindungsliste – Führt alle Pulse Secure Client-Verbindungen auf. Klicken Sie auf einen Verbindungsnamen, um die Aktionen anzuzeigen, die Sie für diese Verbindung ausführen können:
    • Verbinden – Stellt eine Verbindung her.
    • Trennen – Trennt die ausgewählte Netzwerkverbindung.
    • Abbrechen – Bricht eine Aktion ab, die von einem Pulse Secure Client-Server eingeleitet wurde. Die Option „Abbrechen“ steht zur Verfügung, wenn der Server einen Host Checker-Vorgang eingeleitet hat. Wenn Sie abbrechen, baut Pulse Secure Client keine Verbindung mit dem Netzwerk auf.
    • Unterbrechen – Unterbricht eine VPN-Verbindung, um den Zugriff auf lokale Netzwerkressourcen zu erlauben. Beachten Sie, dass diese Option nicht verfügbar ist, wenn Sie bei einer aktiven Verbindung „Gespeicherte Einstellungen vergessen“ auswählen.
    • Fortsetzen – Setzt eine unterbrochene VPN-Verbindung fort.
    • Sitzung verlängern – Verlängert eine aktive Verbindungssitzung, ohne darauf zu warten, dass Pulse Secure Client Sie über eine ablaufende Sitzung benachrichtigt. Sie können diese Option jederzeit auswählen, um das Sitzungs- und Leerlaufzeitlimit auf die Maximalwerte zurückzusetzen.
  • Beenden – Schließt Pulse Secure Client und trennt alle aktiven Verbindungen.

Verwandte Themen

  • Pulse Secure Client-Benutzeroberfläche
