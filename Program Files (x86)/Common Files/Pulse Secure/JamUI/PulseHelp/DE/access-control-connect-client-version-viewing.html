Versionsinformationen zu einer Komponente anzeigen

Im Rahmen einer Fehlerbehebung werden Sie vom Netzwerkadministrator möglicherweise nach den Versionsnummern der Pulse Secure Client-Programme gefragt. So zeigen Sie Pulse Secure Client-Versionsinformationen an:

  1. Klicken Sie auf Hilfe > Info, um das Info-Dialogfeld zu öffnen.
  2. Klicken Sie auf Versionsdetails um das Dialogfeld „Pulse Secure Client-Versionsdetails“ zu öffnen.
