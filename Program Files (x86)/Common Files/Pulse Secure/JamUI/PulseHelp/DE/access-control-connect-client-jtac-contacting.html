Technischen Support anfordern

Technischen Produktsupport können Sie über das Pulse Secure Global Support Center (PSGSC) anfordern.

  • Ausführliche Informationen zum Thema Garantie finden Sie unter:
    https://www.pulsesecure.net/support/

Die Ressourcen der PSGSC-Zentren sind rund um die Uhr 7 Tage die Woche und 365 Tage im Jahr verfügbar.

Onlinetools und -ressourcen zur Selbsthilfe

Zur schnellen und einfachen Problembehebung hat Pulse Secure ein Online-Self-Service-Portal namens Customer Support Center (CSC) bereitgestellt, das Ihnen die folgenden Funktionen bietet:

  • PSGSC Angebote finden Sie unter:
    http://www.pulsesecure.net/support/

  • Die Produktdokumentation finden Sie unter:
    https://www.pulsesecure.net/techpubs/

  • Die neuesten Softwareversionen und Versionshinweise sind verfügbar unter:
    https://www.pulsesecure.net/support/

  • Durchsuchen Sie technische Merkblätter nach relevanten Benachrichtigungen zu Hardware und Software unter:
    https://www.pulsesecure.net/support/

  • Prüfen Sie mit dem SNE-Tool (Serial Number Entitlement) anhand der Produktseriennummer, ob Sie zur Inanspruchnahme von Supportdiensten berechtigt sind:
    https://www.pulsesecure.net/support/

Einen Fall im PSGSC eröffnen

Sie können einen Fall im PSGSC über das Internet oder per Telefon eröffnen.

  • Verwenden Sie das Case Manager-Tool im PSGSC unter:
    https://pulsesecure.net/support

  • Rufen Sie uns an unter: 1-844 751 7629 (gebührenfrei in den USA)

  • Internationale oder Direktwahl-Optionen für Länder ohne gebührenfreie Rufnummern finden Sie unter:
    https://www.pulsesecure.net/support/support-contacts/
