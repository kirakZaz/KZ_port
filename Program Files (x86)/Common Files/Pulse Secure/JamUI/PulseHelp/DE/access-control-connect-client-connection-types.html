Pulse Secure Client-Verbindungstypen

Der beim Definieren einer neuen Verbindung ausgewählte Verbindungstyp bezieht sich auf den Typ des Geräts, das Zugriff auf die geschützten Netzwerkressourcen bietet. Der Netzwerkadministrator kann Ihnen erläutern, welchen Typ Sie auswählen müssen, wenn Sie manuell eine neue Verbindung hinzufügen.

  • Policy Secure (UAC) oder Connect Secure (VPN) – Verwenden Sie diesen Netzwerktyp beim Herstellen einer Verbindung mit Pulse Connect Secure oder Pulse Policy Secure.
  • Firewall (SRX) – Verwenden Sie diesen Netzwerktyp beim Herstellen einer Verbindung zu einem Pulse Secure SRX Series Service-Gateway.

Verwandte Themen

  • Verbindung hinzufügen
  • Verbindung mit einem Netzwerk herstellen
