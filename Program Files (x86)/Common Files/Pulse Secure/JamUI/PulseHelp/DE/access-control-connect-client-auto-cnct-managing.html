Manuelles Außerkraftsetzen bei einer automatischen Verbindung entfernen

Ihr Pulse Secure Client-Administrator kann Pulse Secure Client-Verbindungen bereitstellen, die automatisch verbunden werden. Sie können beispielsweise eine Pulse Secure Client-Verbindung haben, die aktiviert wird, während Sie sich im Büro befinden und mit dem LAN verbunden sind, und eine andere, die außerhalb des Büros aktiviert wird. Automatische Verbindungen vereinfachen die Konnektivität, da weniger manuelle Verbindungsentscheidungen getroffen werden müssen. Ihr Pulse Secure Client-Administrator kann Ihnen gestatten, automatische Verbindungen außer Kraft zu setzen. Pulse Secure Client zeigt Manuelles Außerkraftsetzen für automatische Verbindungen an, die nicht länger automatisch reagieren.

So stellen Sie das automatische Verhalten einer Verbindung wieder her, für die „Manuelles Außerkraftsetzen“ aktiviert wurde:

  1. Klicken Sie auf eine Verbindung, um sie auszuwählen.
  2. Klicken Sie auf Datei > Verbindungen > Außerkraftsetzen entfernen (zur automatischen Kontrolle zurückkehren).

Oder

  1. Klicken Sie im Verbindungsbereich von Pulse Secure Client mit der rechten Maustaste auf die Verbindung, um das Popupmenü anzuzeigen.
  2. Klicken Sie auf Außerkraftsetzen entfernen (zur automatischen Kontrolle zurückkehren).

Verwandte Themen
