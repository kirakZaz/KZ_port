Verbindungseigenschaften anzeigen

So zeigen Sie die Eigenschaften einer Verbindung an:

  1. Klicken Sie auf den Pfeil neben dem Verbindungsnamen. Die Verbindungsdetails werden neben der Verbindung angezeigt.

Wenn für die Server-URL Zufällig ausgewählt angezeigt wird, wurde diese Verbindung vom Pulse Secure Client-Administrator definiert. Für diese Verbindung wird anstelle eines einzelnen Zielservers eine Liste von Pulse Secure Client-Servern konfiguriert. Pulse Secure Client wählt dann per Zufallsauswahl den Zielserver aus. Über die Schaltfläche Ändern können Sie einen beliebigen Server aus der Liste auswählen. Die Schaltfläche Ändern wird für jede Verbindung angezeigt, die eine Liste von Servern hat. Ihr Pulse Secure Client-Administrator kann die Liste so konfigurieren, dass Pulse Secure Client immer eine Verbindung zum ersten Server in der Liste einrichtet, oder zum Server, mit dem Sie zuletzt verbunden waren, oder zu einem nach dem Zufallsprinzip ausgewählten Server.

Verwandte Themen

  • Pulse Secure Client-Benutzeroberfläche
  • Pulse Secure Client-Taskleistensymbol verwenden und Verbindungsstatus anzeigen
