Pulse Secure Client – Konfigurationsübersicht

Pulse Secure Client lädt Einstellungen auf Ihr Endgerät herunter, wenn Sie eine Verbindung mit einem Netzwerk herstellen. Die meisten Konfigurationsaufgaben werden von Ihrem Netzwerkadministrator ausgeführt. Ihr Netzwerkadministrator kann zudem Pulse Secure Client-Funktionen aktivieren oder deaktivieren. Beispielsweise kann Ihr Administrator Pulse Secure Client so konfigurieren, dass automatisch eine Verbindung zu einem bestimmten Netzwerk eingerichtet wird oder dass Sie die gewünschte Verbindung auswählen können.

In der Regel definiert der Netzwerkadministrator die Netzwerke, auf die Sie zugreifen. Der Netzwerkadministrator ermöglicht Ihnen möglicherweise die Definition zusätzlicher Verbindungen. Falls Sie einen Laptop im Büro, zuhause und unterwegs verwenden, benötigen Sie höchstwahrscheinlich eine Verbindung für den Zugriff im Büro und eine andere Verbindung für den Zugriff an Remotestandorten. Nach dem Einrichten eines Typs einer Netzwerkverbindung kann Pulse Secure Client jedoch automatisch angepasst werden, damit die Verbindung bestehen bleibt. Falls Sie beispielsweise an einem Remotestandort eine Verbindung zum Unternehmensnetzwerk herstellen, den Laptop dann ohne Abmelden schließen und ins Büro zurückkehren, können Sie die Computersitzung ohne erneutes Anmelden fortsetzen. Die Portabilitätsfunktion, die als Sitzungsmigration bezeichnet wird, ist von der Konfiguration des Netzwerkzugriffs durch den Administrator abhängig.
