Cargar archivos de registro

El cliente de escritorio Pulse Secure facilita la transmisión de paquetes de archivos de diagnóstico a puertas de enlace PCS para el análisis por parte de los administradores de sistemas. Para enviar un paquete de registros a PCS, ejecute lo siguiente desde la interfaz de usuario del cliente de escritorio cuando haya una conexión VPN activa: Archivo > Registros > Cargar.

Debe seleccionar el servidor al que desea enviar los archivos. Aparecerá un cuadro de diálogo que muestra el progreso de la carga.

El administrador de sistemas debe habilitar esta función en el servidor para que el usuario final pueda cargar archivos de registro a la puerta de enlace de Pulse Secure.

El tamaño máximo del servidor para la combinación de todos los registros de cliente cargados es de 200 MB.

La puerta de enlace Pulse Connect Secure tiene una partición de disco limitada para el almacenamiento de los archivos de registro cargados. Si este espacio de la puerta de enlace se llena con otros archivos de registro, es posible que la carga falle. En este caso, debe pedir al administrador de sistema que libere espacio para la carga de registros.
