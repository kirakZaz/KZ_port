Añadir una conexión

Cada conexión de Pulse Secure Client representa una red protegida. Generalmente, el administrador de la red define las conexiones por usted y puede deshabilitar la función Añadir una conexión. Si necesita crear nuevas conexiones, el administrador de red le indicará los ajustes que debe usar.

Para añadir una conexión nueva:

  1. Haga clic en el botón Añadir una conexión:

    Se abre el cuadro de diálogo Añadir una conexión.

  2. Para Tipo, generalmente el administrador de red le indicará qué opción debe seleccionar. Elija una de las siguientes:
    • UAC o VPN SSL: use este tipo de red si se va a conectar a Pulse Policy Secure o a Pulse Connect Secure.
    • Cortafuegos (SRX): use este tipo de red si se conecta a un dispositivo de la serie SRX de Juniper Networks.
  3. En Nombre, especifique un nombre descriptivo para esta conexión. El nombre que especifique va a aparecer en la interfaz de Pulse Secure Client.
  4. En la URL del servidor, especifique la red a la que desea conectarse. Puede introducir la URL del servidor en cualquiera de los siguientes formatos:
    • Una dirección IP, por ejemplo, 10.204.71.86. El cliente Pulse Secure también es compatible con direcciones IPv6. Aunque no es habitual introducir direcciones IPv6 manualmente, si necesita hacerlo, utilice este formato: fd29:64fc:bd56:5dbd:3:23:202:1a31. Las direcciones IPv6 pueden contener números del 0 al 9 y letras de la "a" a la "f". Puede omitir los ceros iniciales.
    • Un nombre DNS, por ejemplo, server.mycompany.net.
  5. Haga clic en Añadir para guardar la nueva conexión y cerrar el cuadro de diálogo. Haga clic en Conectar para guardar la nueva conexión e iniciar una conexión a la red.
