Ver propiedades de conexión

Para ver las propiedades de una conexión:

  1. Haga clic en la flecha que se encuentra junto al nombre de la conexión. Los detalles de la conexión aparecen bajo la conexión.

Si la URL del servidor aparece como Seleccionada al azar, significa que esta conexión fue definida por el administrador de Pulse Secure Client. En lugar de un servidor de destino, esta conexión se configura con una lista de servidores de Pulse Secure Client y Pulse Secure Client selecciona al azar el servidor de destino. El botón Cambiar le permite seleccionar un servidor determinado en la lista. El botón Cambiar aparece para cada conexión que incluye una lista de servidores. Su administrador de Pulse Secure Client puede configurar la lista de forma que Pulse Secure Client se conecte siempre al primer servidor de la lista, al servidor al que se había conectado más recientemente o a un servidor seleccionado al azar en la lista.
