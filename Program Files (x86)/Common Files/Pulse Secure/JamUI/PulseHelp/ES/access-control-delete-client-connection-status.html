Estado de conexión de Pulse Secure Client

Pulse Secure Client muestra el estado de una conexión en el panel de conexiones y en la bandeja del sistema. El panel de conexiones puede mostrar los siguientes estados:

Conectando. La conexión permanece en este estado hasta que se produce un error o logra conectarse.

En suspensión.

Con conexión, pero con problemas.

Error en la conexión.

Con conexión.

Conectado a la red local, pero sin acceso a Internet disponible. Las redes Wi-Fi públicas suelen implementar un portal cautivo que requiere que el usuario introduzca la información de autenticación o acepte los términos del servicio antes de concederle acceso a la red. Pulse Secure Client detecta la presencia de portales cautivos y no inicia una conexión con un servidor de Pulse Secure Client hasta que no se otorga el acceso a Internet.

El estado de todas las conexiones activas de Pulse Secure Client se indica mediante un único icono de la bandeja del sistema. Puede hacer clic con el botón secundario en el icono de la bandeja del sistema para controlar las conexiones de Pulse Secure Client, acceder a las funciones de reunión de Pulse Collaboration Suite, abrir la interfaz de Pulse Secure Client o salir de Pulse Secure Client.
