Suprimir el control manual en una conexión automática

Su administrador de Pulse Secure Client puede proporcionarle conexiones que se establezcan de forma automática. Por ejemplo, puede tener una conexión de Pulse Secure Client que se active cuando se encuentre en la oficina y esté conectado a la LAN, así como una conexión de Pulse Secure Client diferente que se active cuando se encuentre fuera de la oficina. Las conexiones automáticas simplifican la conectividad mediante la limitación de las decisiones de conexión manual. Su administrador de Pulse Secure Client puede permitirle suprimir las conexiones automáticas. Pulse Secure Client muestra el mensaje omisión manual para las conexiones automáticas que ya no responden automáticamente.

Para restaurar el comportamiento automático en una conexión que se ha establecido en control manual:

  1. Haga clic en la conexión para seleccionarla.
  2. Haga clic en Archivo > Conexiones > Retirar omisión (volver al control automático).

O bien:

  1. En el Panel de conexiones de Pulse Secure Client, haga clic con el botón secundario en la conexión para mostrar el menú emergente.
  2. Haga clic en Retirar omisión (volver al control automático).
