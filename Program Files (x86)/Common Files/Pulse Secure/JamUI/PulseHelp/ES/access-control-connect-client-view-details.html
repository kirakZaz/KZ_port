Ver detalles avanzados de conexión

El cuadro de diálogo Detalles avanzados de la conexión muestra información detallada acerca de la conexión seleccionada.

Para ver detalles avanzados de conexión

  1. Haga clic en la conexión para seleccionarla.
  2. Haga clic en Archivo > Conexiones > Detalles avanzados de conexión.
  3. La información de los detalles de la conexión no se actualiza automáticamente. Por ejemplo, el tiempo de sesión restante corresponde al tiempo restante en el momento en que se abrió el cuadro de diálogo. Para actualizar la información de los detalles avanzados, haga clic en Actualizar o marque la casilla de verificación con la etiqueta Actualizar automáticamente.

La ventana Detalles avanzados de la conexión proporciona la siguiente información:

Nombre del campo

Descripción

Tiempo de sesión restante

La duración del tiempo que la sesión VPN actual permanecerá activa antes de que sea necesario volver a introducir las credenciales o ampliar manualmente la sesión.

Duración de la sesión

La duración del tiempo de la sesión VPN actual se encuentra activa.

Tipo de túnel

Describe si la conexión es un túnel de VPN o una asignación de puerto/aplicación mediante SAM (Secure Access Manager).

Tipo de VPN

El protocolo usado para crear el túnel (SSL or ESP).

IPv4 asignada

La dirección IPv4 asignada al adaptador virtual Pulse.

Entrada de bytes

Números de bytes recibidos a través del túnel

Salida de bytes

Número bytes enviados a través del túnel.

Fuente de conexión

Describe cómo el cliente de Pulse recibió la entrada de conexión:

  • Si el valor es Preconfigurado, la entrada de conexión fue desde un grupo de conexiones descargado desde una puerta de enlace.
  • Si el valor es Agregado por el usuario, la entrada de conexión fue agregada manualmente por el usuario final con la interfaz de usuario del cliente Pulso.
  • Y si el valor es Dinámico, entonces significa que la entrada de conexión fue resultado de iniciar el cliente de Pulse mediante la conexión de un navegador web con una puerta de enlace de Pulso de Secure y pulsar el botón "Inicio" de la página web.

Nombre del grupo de conexiones

Nombre del grupo de conexiones que contiene esta entrada de conexión. Su administrador del sistema puede utilizar este nombre para encontrar el grupo de conexiones que coincide en la consola de administración de la puerta de enlace correspondiente (consulte el siguiente campo) para ver los detalles del grupo de conexiones.

Propietario del grupo de conexiones

Este es el nombre de la puerta de enlace de Pulse Secure que ha editado más recientemente el conjunto de conexiones que tiene esta entrada de conexión. Esta puerta de enlace, junto con el nombre del conjunto de conexiones que aparece más arriba, son dos datos necesarios para localizar la configuración del conjunto de conexiones del cliente en diferentes puertas de enlace de Pulse Secure.

Última modificación del conjunto de conexiones

Hora en la que se actualizó el conjunto de conexiones del cliente de Pulse en la puerta de enlace.

ID de servidor del conjunto de conexiones

Cuando se implementa por primera vez un cliente de Pulse, recibe un conjunto de conexiones vinculado a un ID de servidor específico. Después de las siguientes conexiones a una puerta de enlace, el cliente descargará un conjunto de conexiones actualizado solo si el conjunto de conexiones de la puerta de enlace tiene un ID de servidor que coincide con el ID de servidor del conjunto de conexiones del cliente.

Host de descarga del conjunto de conexiones

Servidor desde el que el cliente de Pulse ha actualizado más recientemente su conjunto de conexiones.

Tiempo de descarga del conjunto de conexiones

Hora a la que el cliente de Pulse ha actualizado más recientemente su conjunto de conexiones.

Los campos anteriores aparecen en la consola de administración de Pulse Connect Secure.
