Tipos de conexión de Pulse Secure Client

El tipo de conexión que elija cuando defina una conexión nueva está relacionado con el tipo de dispositivo que le proporcione acceso a recursos de red protegidos. El administrador de red le puede indicar qué tipo debe seleccionar al agregar manualmente una conexión nueva.

  • UAC o VPN SSL: use este tipo de red si se va a conectar a Pulse Connect Secure o a Pulse Policy Secure.
  • Cortafuegos (SRX): use este tipo de red si se va a conectar a la puerta de enlace de los servicios de la serie SRX de Pulse Secure.
