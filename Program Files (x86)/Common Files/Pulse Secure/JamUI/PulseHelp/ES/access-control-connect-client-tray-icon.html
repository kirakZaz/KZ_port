Usar el icono de la bandeja de Pulse Secure Client y Ver el estado de conexión

El icono de la bandeja de Pulse Secure Client indica el estado actual de la conexión a la red. Verá un icono de conexión en la bandeja, incluso cuando tenga más de una conexión actual. Cuando alguna conexión activa tenga algún problema, el icono de la bandeja del sistema cambiará su estado. Los iconos siguientes indican el estado de la conexión.

Sin conexión.

Conectando. La conexión permanece en este estado hasta que se produce un error o logra conectarse.

En suspensión.

Con conexión, pero con problemas.

Error en la conexión.

Con conexión.

Conectado a la red local, pero sin acceso a Internet disponible. Las redes Wi-Fi públicas suelen implementar un portal cautivo que requiere que el usuario introduzca la información de autenticación o acepte los términos del servicio antes de concederle acceso a la red. Pulse Secure Client detecta la presencia de portales cautivos y no inicia una conexión con un servidor de Pulse Secure Client hasta que no se otorga el acceso a Internet.

También puede utilizar el icono de la bandeja de Pulse Secure Client para realizar las siguientes acciones:

  • Abrir Pulse Secure Client: muestra la interfaz de Pulse Secure Client.
  • Pulse Secure Client Collaboration: accede a las funciones de reunión de Pulse Secure Client Collaboration Suite. Este elemento solo aparece si el administrador de Pulse Secure Client ha habilitado Pulse Secure Client Collaboration.
  • Lista de conexiones: enumera cada conexión de Pulse Secure Client. Haga clic en el nombre de la conexión para mostrar las acciones que puede realizar en ella:
    • Conectar: inicia una conexión.
    • Desconectar: desconecta la conexión de red seleccionada.
    • Cancelar: cancela una acción que inició el servidor de Pulse Secure Client. La opción Cancelar está disponible cuando el servidor ha iniciado una operación de Host Checker. Si utiliza la opción Cancelar, Pulse Secure Client no realizará la conexión a la red.
    • Suspender: suspende la conexión VPN para permitir el acceso a los recursos de red locales. Tenga en cuenta que esta opción no estará disponible si selecciona Olvidar configuraciones guardadas mientras dispone de una conexión activa.
    • Reanudar: reanuda una conexión VPN suspendida.
    • Ampliar la sesión: amplía una sesión de conexión activa sin esperar a que Pulse Secure Client le notifique que ha caducado una sesión. Seleccione esta opción en cualquier momento para restablecer los valores máximos de los temporizadores de sesión e inactividad.
  • Salir: cierra Pulse Secure Client y desconecta cualquier conexión activa.
