Establecer nivel de registro

Para establecer el nivel de registro:

  1. Haga clic en Archivo > Registros > Nivel de registro > Detallado o en Archivo > Registros> Nivel de registro > Normal. Una marca de verificación indica la opción que se encuentra activa.

El registro Normal es el valor predeterminado. Debe establecer el nivel de registro en Normal a menos que deba solucionar un problema de conexión.

El registro Detallado crea un mayor número de entradas de registro y aumenta el tamaño del archivo de registro. El nivel de registro Detallado se activa generalmente solo cuando se debe solucionar un problema.
