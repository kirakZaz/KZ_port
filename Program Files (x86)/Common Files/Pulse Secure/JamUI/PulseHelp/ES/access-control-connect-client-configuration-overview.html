Descripción general de la configuración de Pulse Secure Client

Pulse Secure Client descarga ajustes al punto final cuando se conecta a una red. La mayoría de las tareas de configuración de Pulse Secure Client las realiza el administrador de la red. El administrador de la red también puede habilitar o deshabilitar las funciones de Pulse Secure Client. Por ejemplo, el administrador puede configurar Pulse Secure Client para que se conecte automáticamente a una red en particular o para que permita que usted elija qué conexión desea utilizar.

Normalmente, el administrador de red define las redes a las que tendrá acceso. El administrador de la red también podría habilitarlo para que defina conexiones adicionales. Si tiene un equipo portátil que utiliza en la oficina, en casa y mientras se traslada, probablemente necesite una conexión para cuando se conecte desde su oficina y una conexión diferente para acceder cuando se encuentre en una ubicación remota. No obstante, después de haber establecido un tipo de conexión a una red, Pulse Secure Client puede adaptarse automáticamente para mantenerle conectado. Por ejemplo, si se conecta a la red corporativa desde una ubicación remota, cierra el equipo portátil sin cerrar la sesión y después se dirige a su oficina, puede reanudar la sesión sin tener que iniciar sesión de nuevo. La función de transferencia, que se denomina migración de sesión, depende de la forma en la que el administrador haya configurado el acceso a la red.
