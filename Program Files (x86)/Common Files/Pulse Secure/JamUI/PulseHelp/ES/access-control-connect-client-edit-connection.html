Editar propiedades de conexión

Después de crear una conexión, puede editar la URL y el nombre que aparece en el Panel de conexiones. Puede editar una conexión solo si no está activa y la ha creado manualmente. Si un administrador de Pulse Secure Client ha creado e instalado la conexión, no puede editar el campo URL.

Si la URL del servidor aparece como Seleccionada al azar, significa que esta conexión fue definida por el administrador de Pulse Secure Client. En lugar de un servidor de destino, esta conexión se configura con una lista de servidores de Pulse Secure Client y Pulse Secure Client selecciona al azar el servidor de destino. El botón Cambiar aparece en los detalles de la conexión para cada conexión que incluye una lista de servidores. Su administrador de Pulse Secure Client puede configurar la lista de forma que Pulse Secure Client se conecte siempre al primer servidor de la lista, al servidor al que se había conectado más recientemente o a un servidor seleccionado al azar en la lista. No puede editar la URL de destino para una conexión que incluye una lista de servidores.

Para editar una conexión:

  1. Haga clic con el botón secundario en la conexión para mostrar el menú emergente y, a continuación, haga clic en Editar.

    Se abrirá el cuadro de diálogo Editar conexión.

  2. Edite la conexión y luego haga clic en Guardar para guardar los cambios y cierre el cuadro de diálogo o haga clic en Conectar para iniciar una conexión y cierre el cuadro de diálogo.

O bien:

  1. Haga clic en la conexión para seleccionarla.
  2. Haga clic en el botón Editar conexión:

    Se abrirá el cuadro de diálogo Editar conexión.

  3. Edite la conexión y luego haga clic en Guardar para guardar los cambios y cierre el cuadro de diálogo o haga clic en Conectar para iniciar una conexión y cierre el cuadro de diálogo.
