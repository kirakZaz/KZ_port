Ver información de corrección para una conexión

Cuando falla un intento de conexión o cuando una conexión activa experimenta un problema, Pulse Secure Client muestra información de corrección. Entre la información de corrección, se encuentran instrucciones sobre cómo resolver el problema o se muestra el estado de los procedimientos de corrección que se iniciaron de manera automática.

Para ver la información de corrección:

  1. Haga clic en la conexión para seleccionarla.
  2. Haga clic en Archivo> Conexiones> Información de corrección para abrir el cuadro de diálogo Corrección.

    El menú Corrección se atenúa si no hay acciones de corrección activas o pendientes.

  3. La parte superior del cuadro de diálogo describe el problema y la parte inferior los pasos para resolver el problema. Realice los pasos de corrección y luego haga clic en Reintentar para restablecer la conectividad.

O bien:

  1. Haga clic en la flecha que se encuentra junto a una conexión para ampliarla y mostrar su estado de conexión.
  2. En el estado de conexión, haga clic en el enlace Detalles para ver información más detallada sobre el problema.

O bien:

  1. Haga clic con el botón secundario en la conexión para mostrar el menú emergente y, a continuación, haga clic en Información sobre la corrección para abrir el cuadro de diálogo de Corrección.
  2. Realice los pasos de corrección y luego haga clic en Reintentar para restablecer la conectividad.
