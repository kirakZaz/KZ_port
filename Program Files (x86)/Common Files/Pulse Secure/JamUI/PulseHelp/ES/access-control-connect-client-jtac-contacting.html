Solicitud de asistencia técnica

Encontrará asistencia técnica en el Centro de asistencia técnica global de Pulse Secure (PSGSC).

  • Para obtener información sobre la garantía de producto, visite:
    https://www.pulsesecure.net/support/

Los centros PSGSC ofrecen recursos de forma ininterrumpida.

Recursos y herramientas de autoayuda en línea

Para solucionar un problema de una forma rápida y sencilla, Pulse Secure ha diseñado un portal de autoservicio en línea denominado Centro de asistencia técnica global de Pulse Secure (PSGSC), que le ofrece las siguientes funciones:

  • Encuentre ofertas de PSGSC en:
    https://www.pulsesecure.net/support/

  • Consulte la documentación del producto en:
    https://www.pulsesecure.net/techpubs/

  • Descargue las versiones más recientes del software y las notas de publicación en:
    https://www.pulsesecure.net/support/

  • Realice búsquedas en boletines técnicos para encontrar las notificaciones pertinentes de hardware y software en:
    https://www.pulsesecure.net/support/

  • Verifique la autorización de servicio por número de serie de producto. Para ello, utilice la herramienta de Autorización de número de serie (SNE) en:
    https://www.pulsesecure.net/support/

Abrir un caso en el Centro de asistencia técnica global de Pulse Secure

Puede abrir un caso con PSGSC a través de la web o por teléfono.

  • Utilice la herramienta de Gestión de casos de PSGSC disponible en:
    https://pulsesecure.net/support

  • Llame a 1-844 751 7629 (llamada gratuita en EE. UU.)

  • Para consultar las opciones de llamada internacional o marcación directa en los paíss con llamadas gratuitas, visítenos en:
    https://www.pulsesecure.net/support/support-contacts/
