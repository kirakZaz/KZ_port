Accesibilidad/Cumplimiento con la sección 508

El cliente de escritorio Pulse Secure incluye numerosas funciones que facilitan la navegación por la interfaz del usuario del cliente de escritorio para los usuarios con discapacidades físicas, sensoriales o cognitivas.

Para habilitar algunas de estas funciones debe establecer una clave de registro. Se deben establecer estas claves:

HKEY_CURRENT_USER\Software\Pulse Secure\Pulse\Accessibility

Las funciones de accesibilidad incluyen:

  • Compatibilidad con tabulación: la interfaz de usuario del cliente de escritorio gestiona el movimiento de objetivo de teclado de un widget de interfaz de usuario a otro usando la tecla de tabulación. Esto permite navegar fácilmente por la interfaz de usuario sin usar un ratón.
  • Otras teclas de navegación:
    • Se puede acceder a la mayoría de los campos de entrada de datos y los controles de diálogo manteniendo pulsada la tecla Alt y pulsando la tecla de acceso directo. El acceso directo específico que se va a usar se puede mostrar pulsando y soltando la tecla Alt.
    • También se puede acceder a los menús de Pulse mediante la tecla Alt y el acceso directo de menú. Una vez que se accede al sistema de menús, se pueden usar los accesos directo de la tecla Alt, las teclas de flecha o Intro para navegar y seleccionar las opciones de menú.
    • Dentro de los cuadros diálogos, se pueden usar para navegar la combinación normal de la tecla Tab y las teclas de flecha.
  • Compatibilidad con lectores de pantalla: se puede usar el lector de pantalla JAWS para leer el estado de las conexiones. Para habilitar la funcionalidad de lector de pantalla, usted o el administrador del sistema deben establecer la clave de registro ScreenreaderMode DWORD en 1.
  • Uso de configuración de color del sistema: la interfaz de cliente de escritorio de Pulse puede adoptar los colores de sistema del sistema operativo Windows en el modo de alto contraste. Para habilitar esta funcionalidad, usted o el administrador del sistema deben establecer la clave de registro UseSystemColors REG_BINARY en 00000001.

La guía de usuario del cliente de escritorio de Pulse Secure está disponible en formato PDF en diferentes idiomas. Para acceder a esta guía del usuario en formato PDF, póngase en contacto con el Centro de soporte global de Pulse Secure utilizando uno de los métodos enumerados en la sección Solicitar soporte técnico.
