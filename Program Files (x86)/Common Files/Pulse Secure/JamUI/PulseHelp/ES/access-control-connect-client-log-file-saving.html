Guardar archivos de registro

Como parte de un proceso de solución de problemas, se le podría pedir que guarde los archivos de registro de Pulse Secure Client. Pulse Secure Client incluye una cantidad de posibles componentes y cada componente genera uno o más archivos de registro. La operación Guardar como combina todos los archivos de registro de Pulse Secure Client y otros archivos de diagnóstico en un solo archivo llamado LogsAndDiagnostics.zip.

Para guardar archivos de registro de Pulse Secure Client:

  1. Haga clic en Archivo > Registros > Guardar como. Se mostrará el cuadro de diálogo Guardar como.
  2. Puede aceptar los valores predeterminados para la ubicación y el nombre del archivo o puede especificar nuevos valores y después hacer clic en Guardar.
