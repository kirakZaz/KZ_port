Archivos de registro de Pulse Secure Client

Un archivo de registro de Pulse Secure Client almacena información que puede ayudarle a resolver problemas de conexión.

Los archivos de registro incluyen conexiones de Pulse Secure Client y sus propiedades. Si desea guardar la configuración, basta con guardar los registros. Su configuración actual de Pulse Secure Client se guarda en el paquete de registro. Si desinstala Pulse Secure Client, el programa le pregunta si desea guardar su configuración de Pulse Secure Client para que, cuando vuelva a instalar Pulse Secure Client, estén disponibles sus conexiones de Pulse Secure Client y otros ajustes de configuración.

El registro es una operación que se ejecuta en segundo plano. No es necesario que haya ningún cambio en su entorno de registro a menos que se le indique que lo haga como parte de un intento de solución de problemas.

Además de los archivos de registro de Pulse Secure Client, Pulse Secure Client escribe eventos en la aplicación de registro de eventos de Windows (solo en sistemas con Windows Vista y Windows 7). Si tiene acceso como administrador en el equipo, puede usar el visor de eventos de Windows para ver los eventos de Pulse Secure Client que se han incluido en el registro de eventos. Al guardar los archivos de registro de Pulse Secure Client, se incluye la información de registro de eventos.

Para ayudar en la solución de problemas, puede que se le solicite que realice las siguientes tareas:

  • Anotar registros: cuando anota un archivo de registro, inserta texto que marca el archivo de registro en una ubicación específica. Por ejemplo, para solucionar un problema de conexión, puede que le soliciten anotar el archivo de registro con texto específico, intentar establecer la conexión que ha estado fallando y luego anotar el archivo de registro de nuevo. Esta secuencia de eventos permite que un representante de asistencia busque el archivo de registro para ver el texto que insertó. El texto incluye entre corchetes las entradas del archivo de registro que rastrean el problema de conexión.
  • Establecer nivel de registro: el nivel de registro predeterminado es Normal. Para una operación de solución de problemas, se le puede pedir que cambie el nivel de registro a Detallado.
  • Guardar registros: la operación Guardar como reúne todos los archivos de registro en un archivo .zip y le permite especificar dónde desea guardar el archivo.
