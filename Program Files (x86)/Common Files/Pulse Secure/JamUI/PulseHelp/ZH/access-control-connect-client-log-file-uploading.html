上傳日誌檔

Pulse Secure 桌面用戶端可輕鬆將診斷日誌連結群組傳送至 PCS 閘道，供系統管理員作為分析之用。若要在 VPN 連線啟用時，將日誌連結群組傳送至 PCS，請從桌面用戶端使用者介面執行下列程序：[檔案] > [日誌] > [上傳]。

您必須選取日誌要傳送至哪個伺服器。然後畫面會出現有上傳進度的對話方塊。

您的系統管理員必須在伺服器端啟用此功能，一般使用者才能上傳日誌檔到 Pulse Secure 閘道。

伺服器的大小上限為 200MB (所有已上傳用戶端日誌的總計)。

Pulse Connect Secure 閘道專門用於儲存已上傳日誌檔的磁碟分割空間有限。如果閘道上的空間存滿其他日誌檔，您的上傳可能會失敗。在此情況下，您需要請系統管理員清理空間來讓您上傳日誌。
