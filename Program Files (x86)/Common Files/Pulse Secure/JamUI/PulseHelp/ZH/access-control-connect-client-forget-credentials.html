忘記儲存的設定

連線至網路時，您可以核取 [儲存設定] 核取方塊，讓 Pulse Secure 用戶端記住您的登入驗證資料。(請注意，您的網路管理員可以停用此功能)。提示您做出回應的每個不同畫面，都有其自己的 [儲存設定] 核取方塊。儲存設定後，在後續登入嘗試中，系統便不會提示您提供該資訊。不過，如果您的登入驗證資料有所變更，則您必須先清除 Pulse Secure 用戶端中所儲存的設定，然後才能儲存新的驗證資料。

如果您在有作用中的連線時，選擇 [忘記儲存的設定]，則 [暫停] 連線選項無法使用。

如果您在連線處於暫停狀態時，選擇 [忘記儲存的設定]，則連線會中斷。

若要清除儲存的登入驗證資料：

  1. 在任何連線上按一下滑鼠右鍵，以顯示快顯功能表。
  2. 按一下忘記儲存的設定。Pulse Secure 用戶端便會清除所有已設定之連線的已儲存設定。

或者

  1. 按一下檔案 > 連線 > 忘記儲存的設定。
