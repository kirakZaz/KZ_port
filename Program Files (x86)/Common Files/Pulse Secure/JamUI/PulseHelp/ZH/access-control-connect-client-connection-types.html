Pulse Secure 用戶端連線類型

定義新連線時所選擇的連線類型，與提供受保護網路資源存取權的裝置類型有關。當您手動新增連線時，網路管理員會告知您應選取哪一種類型。

  • Policy Secure (UAC) 或 Connect Secure (VPN) - 如果您要連線至 Pulse Connect Secure 或 Pulse Policy Secure，請使用此網路類型。
  • 防火牆 (SRX) - 如果您是連線至 Juniper Networks SRX 系列服務閘道，請使用此網路類型。
