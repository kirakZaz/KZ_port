Pulse Secure 用戶端綜述

Pulse Secure 用戶端可透過區域網路和廣域網路，啟用受保護網路資源的安全驗證連線。Pulse Secure 用戶端也與 Pulse Collaboration Suite 整合，以提供線上會議功能。Pulse Secure 用戶端可藉由讓網路管理員設定您的網路存取環境來簡化網路連線功能。

安全連線會加密網路通訊流量，以便只有您的端點以及與之通訊的伺服器能夠瞭解通訊內容。當 Pulse Secure 用戶端建立網路連線時，網路管理員會定義 Pulse Secure 用戶端所採用的安全性設定。

已經過驗證的連線表示您的端點與伺服器已向彼此證明其身份。伺服器會使用來自信任的認證授權單位之認證來驗證其自身，而端點會向伺服器提供您的登入驗證資料。網路管理員可以設定您的環境，以在不提示您輸入任何資訊的情況下自動進行驗證 (雖然當您連線時，系統通常仍會要求您輸入使用者名稱和密碼)。

受保護的網路資源是指使用經過驗證之連線存取的任何網路裝置或應用程式。例如，若要存取企業的電子郵件系統，首先要登入您的網路。Pulse Secure 用戶端還可讓您啟動 Pulse Collaboration Suite 用戶端並參與會議。網路管理員會管理領域中受保護的網路資源，並透過授予對所選領域的存取權限來賦予您對這些資源的存取權。如果您擁有超過一個領域的存取權限，您可以在透過 Pulse Secure 用戶端連線至網路時，選取您要的領域。網路管理員也可以透過啟用主機檢查器程式，定義存取領域所需的安全屬性。當您透過 Pulse Secure 用戶端連線至網路時，主機檢查器便會執行。它會檢查您的端點，以確保它擁有所需的軟體，例如針對最新病毒與惡意程式的保護軟體。管理員還可以將您的 Pulse Secure 用戶端連線設定為僅允許存取特定網路專有的應用程式。

網路管理員可以設定存取政策，以便在您連線至網路之後，該連線可成為可攜式連線。例如，如果您從家中將筆記型電腦連線至公司網路，那麼當您將筆記型電腦帶到辦公室時，便無需先中斷連線再重新連線。您在家中只需將筆記型電腦關閉即可。當您到達辦公室並開啟筆記型電腦時，Pulse Secure 用戶端會偵測到網路環境的變更。Pulse Secure 用戶端會使用您的連線設定檔資訊來設定連線，以在新環境中為您提供適當的存取與驗證。如果管理員已設定 Pulse Secure 用戶端的位置檢測功能，您將會在電腦啟動時連線至適當的網路。例如，如果您在遠端位置啟動了端點，Pulse Secure 用戶端會自動與 WAN 存取裝置建立連線。

當您連線至網路時，Pulse Secure 用戶端可讓網路管理員向您的端點「推送」新軟體以及重要的軟體更新。
