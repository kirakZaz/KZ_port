使用 Pulse Secure 用戶端系統匣圖示並檢視連線狀態

Pulse Secure 用戶端系統匣圖示可指出網路連線的目前狀態。您只會在系統匣中看到一個連線圖示，即使目前有多個連線也是如此。有任何作用中的連線發生問題時，系統匣圖示便會變更其狀態。下列圖示代表連線狀態。

無連線。

正在連線。連線會維持此狀態，直到失敗或成功。

已暫停。

已連線，但發生問題。

連線失敗。

已連線。

已連線到區域網路，但沒有網際網路存取。公共 WiFi 位置通常會部署網頁驗證入口，要求使用者輸入驗證資訊或接受服務條款，才授予網路存取權。Pulse Secure 用戶端會偵測到網頁驗證入口的存在，而且在獲授予網際網路存取權限之前，不會初始化與 Pulse Secure 用戶端伺服器的連線。

您也可以使用 Pulse Secure 用戶端系統匣圖示執行下列動作：

  • 開啟 Pulse Secure 用戶端 - 顯示 Pulse Secure 用戶端介面。
  • Pulse Secure 用戶端 Collaboration - 存取 Pulse Secure 用戶端 Collaboration Suite 會議功能。只有在 Pulse Secure 用戶端管理員已啟用 Pulse Secure 用戶端 Collaboration 時，此項目才會出現。
  • 連線清單 - 列出每個 Pulse Secure 用戶端連線。按一下連線名稱，會顯示可對該連線執行的動作：
    • 連線 - 初始化連線。
    • 中斷連線 - 中斷所選網路連線。
    • 取消 - 取消 Pulse Secure 用戶端伺服器已初始化的動作。若伺服器已初始化主機檢查器操作，[取消] 選項便可使用。如果您取消，Pulse Secure 用戶端便不會連線至網路。
    • 暫停 - 暫停 VPN 連線以允許存取區域網路資源。請注意，如果您在有作用中的連線時，選擇 [忘記儲存的設定]，則此選項無法使用。
    • 恢復 - 恢復已暫停的 VPN 連線。
    • 延長工作階段 - 不等候 Pulse Secure 用戶端通知您有工作階段即將到期，而延長作用中的連線工作階段。您可以隨時選取此選項，將工作階段及閒置的計時器重設為最大值。
  • 退出 - 關閉 Pulse Secure 用戶端並中斷任何作用中的連線。
