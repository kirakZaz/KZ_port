延長連線工作階段

您的網路管理員可為網路連線定義工作階段逾時限制。您的管理員可定義工作階段上限及工作階段閒置限制。到達限制時，Pulse Secure 用戶端會顯示訊息。此訊息會顯示 Pulse Secure 用戶端中斷連線之前，網路工作階段中所剩餘的時間量。如果您未採取動作，Pulse Secure 用戶端便會在工作階段計數器到達零時中斷連線。如果您按一下 Extend [延長]，Pulse Secure 用戶端便會將計時器重設為網路管理員所定義的最大值，並繼續進行該工作階段而不中斷服務。

您也可以延長工作階段，而不需等候 Pulse Secure 用戶端通知您有工作階段即將到期。您可以隨時延長工作階段，以將工作階段及閒置的計時器重設為最大值。

Extend Session [延長工作階段] 功能不適用於防火牆連線。

若要延長工作階段：

  1. 按一下您要延長時間的連線。
  2. 按一下 File > Connections > Extend Session ([檔案] > [連線] > [延長工作階段])。

或者

  1. 在系統匣中的 Pulse Secure 用戶端圖示上按一下滑鼠右鍵以顯示快顯功能表、按一下該連線，再按一下 Extend Session [延長工作階段]。

如果您發現您的工作階段經常受到工作階段逾時的影響，請要求網路管理員提高工作階段生命週期選項。
