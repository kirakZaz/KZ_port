輔助工具 / 508 條款合規

Pulse Secure 桌面用戶端包含許多功能，讓肢體、感官或認知障礙使用者能輕鬆瀏覽桌面用戶端使用者介面。

部分功能需要設定登錄機碼才能啟用。應在以下位置設定機碼：

HKEY_CURRENT_USER\Software\Pulse Secure\Pulse\Accessibility

輔助工具功能包括：

  • 支援 Tab 鍵 - 桌面用戶端使用者介面可透過 Tab 鍵將鍵盤焦點從一個使用者介面小工具移動到另一個。如此一來，不使用滑鼠也能輕鬆瀏覽使用者介面。
  • 其他導覽按鍵 -
    • 按住 Alt 鍵並按下快速鍵可存取大部分的資料輸入欄位和對話方塊控制項。按下並放開 Alt 鍵可顯示能使用的特定快速鍵。
    • 按下 Alt 鍵及功能表快速鍵也可存取 Pulse 功能表。一旦進入功能表系統後，使用 Alt 鍵和快速鍵，或使用方向鍵和 Eenter 鍵，皆能導覽及選取功能表選項。
    • 在對話方塊中，可使用 Tab 和方向鍵的一般按鍵組合來進行導覽。
  • 支援螢幕閱讀器 - 可以使用 JAWS 螢幕閱讀器來讀取連線狀態。若要啟用螢幕閱讀器功能，您或您的系統管理員必須將登錄機碼 ScreenreaderMode DWORD 設為 1。
  • 使用系統色彩設定 - Pulse 桌面用戶端使用者介面可在高對比度模式中使用 Windows 作業系統的系統色彩。若要啟用此功能，您或您的系統管理員必須將登錄機碼 UseSystemColors REG_BINARY 設為 00000001。
