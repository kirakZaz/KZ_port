調整 Pulse Secure 用戶端視窗的大小

您可以調整 Pulse Secure 用戶端視窗的大小，並可將其移至螢幕上的其他位置。如果您之後將視窗最小化，它會記住新的大小與位置。但是，如果您關閉程式，視窗會在您下次啟動 Pulse Secure 用戶端時恢復其預設大小與位置。

若要調整 Pulse Secure 用戶端視窗的大小：

  1. 按一下 Pulse Secure 用戶端視窗右下方調整大小的操作框：
  2. 按住滑鼠鍵並拖曳視窗的邊緣，直到您看到想要的視窗大小，然後再釋放滑鼠鍵。
