存取 Pulse Secure 用戶端 Collaboration 會議功能

當您的 Pulse Secure 用戶端連線到作為 Pulse Collaboration Suite 伺服器使用的伺服器時，您可以從 Pulse Secure 用戶端系統匣圖示存取 Pulse Collaboration Suite 功能，並加入會議、排定會議或退出會議。或者，您也可以透過 Web 入口登入 Pulse Secure 用戶端伺服器，然後按一下 [會議] 按鈕以存取 Pulse Collaboration 功能。

若要存取 Pulse Collaboration 功能：

  1. 在 Windows 系統匣中的 Pulse Secure 用戶端圖示上按一下滑鼠右鍵，以顯示快顯功能表。
  2. 依序按一下 Pulse Collaboration、會議伺服器，以及啟動會議。
  3. 若出現提示，請提供您的登入驗證資料。若此時有排定會議，Pulse Secure 用戶端會開啟瀏覽器視窗，其中會顯示啟動會議按鈕。若無，則請檢視會議排程。選取會議排程中的會議，然後按一下啟動會議以開啟 Pulse Collaboration Suite 用戶端並加入會議。當您加入 Pulse Collaboration 會議時，系統會下載並開啟 Pulse Collaboration 用戶端。如需如何使用 Pulse Collaboration Suite 的詳細資訊，請按一下 Pulse Collaboration 介面中的 [說明] 連結。

    如果您的管理員已啟用 Microsoft Outlook 整合功能，請按一下連結，下載並安裝 Microsoft Outlook 外掛程式。此外掛程式可讓您在 Outlook 內排定會議，並使用 Outlook 電子郵件將邀請傳送給出席者。

    按一下排定會議以排定會議並邀請出席者。

    視需要按一下即時會議或支援會議 (如果您的管理員已啟用這些會議類型)。即時會議與標準會議相似。此功能可以讓您透過自動套用您的角色權限設定 (例如驗證需求、遠端控制和安全聊天) 的方式，快速建立會議。支援會議是一種雙人會議，主要是用來進行線上疑難排解。支援會議提供桌面分享和遠端控制等操作。

    瀏覽器視窗頂端的 [會議] 按鈕能讓您從 Web 入口內的任意位置返回會議排程。

    如需 Pulse Collaboration 功能的詳細資訊，請按一下瀏覽器視窗中的 [說明] 按鈕。

若要中斷會議連線：

  1. 依序按一下 Pulse Collaboration、會議伺服器，以及登出。

    使用系統匣圖示登出功能表項目這項動作的作用，與透過 Pulse Collaboration 用戶端介面進行登出的動作相同。
