儲存日誌檔

作為疑難排解程序的一部分，系統可能會要求您儲存 Pulse Secure 用戶端日誌檔。Pulse Secure 用戶端會加入許多可能的元件，且每個元件都會產生一或多個日誌檔。[另存新檔] 操作會將所有 Pulse Secure 用戶端日誌檔與其他診斷檔案結合成一個名為 LogsAndDiagnostics.zip 的檔案。

若要儲存 Pulse Secure 用戶端日誌檔：

  1. 按一下檔案 > 日誌 > 另存新檔。[另存新檔] 對話方塊隨即出現。
  2. 接受位置與檔案名稱的預設值，或者指定新值，然後按一下儲存。
