為日誌檔加上註解

為日誌檔加上註解時，您可以在特定位置插入標記日誌檔的文字。例如，若要對連線問題進行疑難排解，系統可能會要求您使用特定文字為日誌檔加上註解、嘗試已失敗的連線，然後再次為日誌檔加上註解。此種事件順序可讓技術支援人員輕鬆找出記錄了您連線問題的日誌檔項目。

若要為 Pulse Secure 用戶端日誌檔加上註解：

  1. 按一下檔案 > 日誌 > 註解，開啟 [為日誌加上註解] 對話方塊。
  2. 輸入您的註解文字，然後按一下確定。
