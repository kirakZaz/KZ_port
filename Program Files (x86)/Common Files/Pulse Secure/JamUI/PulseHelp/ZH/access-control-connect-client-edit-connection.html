編輯連線屬性

建立連線後，您可以編輯 URL 及顯示在 Connections [連線] 窗格中的名稱。只有在連線目前並未處於作用中的狀態，而且該連線是由您手動建立時，您才能編輯該連線。如果連線是您的 Pulse Secure 用戶端管理員建立與安裝的，您就不能編輯 URL 欄位。

如果 Server URL [伺服器 URL] 顯示為 Selected at random [隨機選取]，表示此連線是 Pulse Secure 用戶端管理員定義的。此連線並未設定一個目標伺服器，而是設定列成清單的多個 Pulse Secure 用戶端伺服器，並由 Pulse Secure 用戶端隨機選取目標伺服器。每個包含多個伺服器的連線都會在連線詳細資訊中顯示 Change [變更] 按鈕。您的 Pulse Secure 用戶端管理員能設定清單，使 Pulse Secure 用戶端一律連線至清單中第一個伺服器、您最近連線的伺服器，或是從清單中隨機選取的伺服器。您無法編輯包含多個伺服器之連線的目標 URL。

若要編輯連線：

  1. 以滑鼠右鍵按一下連線來顯示快顯功能表，然後按一下 Edit [編輯]。

    Edit Connection [編輯連線] 對話方塊隨即開啟。

  2. 編輯連線，然後按一下 Save [儲存] 以儲存變更並關閉對話方塊，或者按一下 Connect [連線] 以初始化連線並關閉對話方塊。

或者

  1. 按一下該連線加以選取。
  2. 按一下 Edit Connection [編輯連線] 按鈕：

    Edit Connection [編輯連線] 對話方塊隨即開啟。

  3. 編輯連線，然後按一下 Save [儲存] 以儲存變更並關閉對話方塊，或者按一下 Connect [連線] 以初始化連線並關閉對話方塊。
