要求技術支援

技術產品支援可透過 Pulse Secure Global Support Center (PSGSC) 取得。

  • 如需產品保固資訊，請造訪：
    https://www.pulsesecure.net/support/

PSGSC 中心 24 小時全天候提供服務，全年無休。

線上自助工具與資源

為了快速輕鬆地解決問題，Pulse Secure 已設計出一種稱為「Pulse Secure Global Support Center (CSC)」的線上自助服務入口，可為您提供下列功能：

  • 尋找 PSGSC 提供的內容，網址為：
    https://www.pulsesecure.net/support/

  • 尋找產品文件，網址為：
    http://www.pulsesecure.net/techpubs/

  • 若要下載最新版本的軟體及版本說明，網址為：
    https://www.pulsesecure.net/support/

  • 若要在技術布告欄中搜尋相關硬體與軟體通知，網址為：
    https://www.pulsesecure.net/support/

  • 若要使用此「序號權利」(SNE) 工具，依據產品序號確認服務權利，網址為：
    https://www.pulsesecure.net/support/

使用 PSGSC 建立案例

您可以使用 PSGSC 在線上建立案例，或透過電話建立案例。

  • 如需在 PSGSC 使用 Case Manager 工具，網址為：
    https://pulsesecure.net/support

  • 撥打 1-844 751 7629 (美國境內免付費)

  • 若要在沒有免付費電話號碼的國家/地區撥打國際或直撥電話，請造訪：
    https://www.pulsesecure.net/support/support-contacts/
