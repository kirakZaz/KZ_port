Pulse Secure 用戶端連線狀態

Pulse Secure 用戶端會在 Connections [連線] 窗格與系統匣中顯示連線狀態。[連線] 窗格會顯示下列狀態：

正在連線。連線會維持此狀態，直到失敗或成功。

已暫停。

已連線，但發生問題。

連線失敗。

已連線。

已連線到區域網路，但沒有網際網路存取。公共 WiFi 位置通常會部署網頁驗證入口，要求使用者輸入驗證資訊或接受服務條款，才授予網路存取權。Pulse Secure 用戶端會偵測到網頁驗證入口的存在，而且在獲授予網際網路存取權限之前，不會初始化與 Pulse Secure 用戶端伺服器的連線。

單一系統匣圖示即表示所有作用中的 Pulse Secure 用戶端連線的狀態。您可以在系統匣圖示上按一下滑鼠右鍵來控制 Pulse Secure 用戶端連線、存取 Pulse Collaboration Suite 會議功能、開啟 Pulse Secure 用戶端介面，或退出 Pulse Secure 用戶端。下表指出系統匣圖示所代表的連線狀態。

無連線。

正在連線。連線會維持此狀態，直到失敗或成功。

已暫停。

已連線，但發生問題。

連線失敗。

已連線。
