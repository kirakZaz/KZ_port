存取區域網路資源 (暫停連線)

當您有作用中的 VPN 連線時，可能會無法存取區域網路資源，例如印表機和共用磁碟機。您可以暫停 VPN 連線以連接至本機資源，然後重新啟動至遠端網路的 VPN 連線，而無需提供登入驗證資料。

如果 Pulse Secure 用戶端管理員已啟用本機子網路存取權，則您可以存取資源，例如，連線至與您電腦相同的子網路且沒有暫停 Pulse Secure 用戶端連線的印表機。

若要暫停 VPN 連線並存取區域網路資源：

  1. 按一下作用中的連線加以選取，然後在 Pulse Secure 用戶端功能表列上按一下檔案 > 連線 > 暫停。

    如果您在有作用中的連線時，選擇 [忘記儲存的設定]，則此選項無法使用。

    或者

    以滑鼠右鍵按一下作用中的連線，然後按一下 Suspend [暫停]。

    或者

    在 Windows 系統匣中按一下 Pulse Secure 用戶端圖示以顯示快顯功能表，接著按一下作用中的連線，然後按一下暫停。

    對於已暫停的連線，Pulse Secure 用戶端 [連線] 窗格中的按鈕標籤會變更為恢復，且暫停圖示會出現在連線的旁邊：

    連線已暫停時，系統匣圖示會顯示暫停狀態：

若要重新啟動已暫停的 VPN 連線：

  1. 選取已暫停的連線，然後按一下檔案 > 連線 > 恢復。

    或者

    按一下恢復。

    或者

    在 Windows 系統匣中按一下 Pulse Secure 用戶端圖示以顯示快顯功能表，接著按一下已暫停的連線，然後按一下恢復。
