Pulse Secure 用戶端設定綜述

當您連線至網路時， Pulse Secure 用戶端會將設定下載至您的端點。大多數 Pulse Secure 用戶端設定工作都是由網路管理員執行。網路管理員也可以啟用或停用 Pulse Secure 用戶端功能。例如，管理員可以設定 Pulse Secure 用戶端，使其自動連線至特定網路，或者允許您選擇想要的連線。

一般而言，網路管理員會定義您將要存取的網路。網路管理員也可能會允許您定義其他連線。如果您有一台在辦公室、家中以及路上都會使用的筆記型電腦，則極有可能在從辦公室連線時需要一種連線，而在遠端位置時需要另一種連線來進行存取。然而，在您與網路建立一種連線之後，Pulse Secure 用戶端便能自動調整以使您保持連線狀態。例如，如果您從遠端位置連線至企業網路，然後在沒有登出的情況下關閉筆記型電腦並前往辦公室，則在辦公室您便可以繼續運算工作階段，而無需再次登入。這種稱為工作階段移轉的可攜功能將會視管理員設定網路存取的方式而定。
