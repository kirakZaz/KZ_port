移除自動連線的手動優先

您的 Pulse Secure 用戶端管理員可以提供自動連線的 Pulse Secure 用戶端連線。舉例來說，您可能擁有一個 Pulse Secure 用戶端連線，當您在辦公室連線至 LAN 時會啟動，另一個 Pulse Secure 用戶端連線則是在您離開辦公室時啟動。自動連線透過限制手動連線決定來簡化連線方式。您的 Pulse Secure 用戶端管理員可讓您覆寫自動連線。Pulse Secure 用戶端會針對無法再自動回應的自動連線顯示手動優先。

若要將已設定為手動優先的連線還原為自動運作：

  1. 按一下該連線加以選取。
  2. 按一下檔案 > 連線 > 移除優先 (恢復自動控制)。

或者

  1. 在 Pulse Secure 用戶端的 [連線] 窗格上，在該連線上按一下滑鼠右鍵，以顯示快顯功能表。
  2. 按一下移除優先 (恢復自動控制)。
