Pulse Secure 用戶端日誌檔

Pulse Secure 用戶端日誌檔可追蹤能夠協助解決連線問題的資訊。

日誌檔包含 Pulse Secure 用戶端連線及其屬性。若您要儲存設定，只要儲存日誌即可。您目前的 Pulse Secure 用戶端設定便會儲存於日誌封裝中。如果解除安裝 Pulse Secure 用戶端，程式會詢問您是否要儲存 Pulse Secure 用戶端設定。如此一來，當您重新安裝 Pulse Secure 用戶端時，系統仍會保留您的 Pulse Secure 用戶端連線及其他設定。

日誌工作是一項背景操作。除非系統指示您變更日誌環境，以作為疑難排解工作的一部分，否則您無需對日誌環境進行任何變更。

除了 Pulse Secure 用戶端日誌檔外，Pulse Secure 用戶端會將事件寫入 Windows 應用程式事件記錄檔 (僅限 Windows Vista 與 Windows 7 系統)。如果您擁有電腦的管理員存取權，您可以使用 Windows 事件檢視器來檢視記錄至事件記錄檔的 Pulse Secure 用戶端事件。當您儲存 Pulse Secure 用戶端日誌檔時，也會同時儲存事件日誌資訊。

為了協助您進行疑難排解，系統可能會要求您執行下列工作：

  • 為日誌加上註解 - 當您為日誌檔加上註解時，會在特定位置插入標記日誌檔的文字。例如，若要對連線問題進行疑難排解，系統可能會要求您使用特定文字為日誌檔加上註解、嘗試已失敗的連線，然後再次為日誌檔加上註解。此種事件順序可讓支援代表在日誌檔中搜尋您插入的文字。該文字會為追蹤連線問題的日誌檔項目加上括弧。
  • 設定日誌層級 - 預設日誌層級為 [正常]。針對某個疑難排解操作，系統可能會要求您將日誌層級變更為 [詳細]。
  • 儲存日誌 -[另存新檔] 操作會將所有日誌檔收集到一個 ZIP 檔案中，並可讓您指定放置檔案的位置。
