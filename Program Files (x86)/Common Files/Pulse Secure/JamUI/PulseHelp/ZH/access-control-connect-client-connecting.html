連線至網路

連線至網路前，您必須至少有一個已設定的連線。您看到的提示會根據您的網路存取環境而有所不同。

若要將 Pulse Secure 用戶端與無線網路搭配使用，您可能需要先透過 Windows 或安裝在端點上的無線裝置軟體，來設定端點的無線網路設定。或者，您的網路管理員可能會定義您的無線連線與掃描清單，並將其包括在 Pulse Secure 用戶端安裝中。

如果 [伺服器 URL] 顯示隨機選取，表示此連線是 Pulse Secure 用戶端管理員定義的。此連線並未設定一個目標伺服器，而是設定列成清單的多個 Pulse Secure 用戶端伺服器，並由 Pulse Secure 用戶端隨機選取目標伺服器。每個包含多個伺服器的連線都會在連線詳細資訊中顯示變更按鈕。按一下變更按鈕，以從清單中選取目標伺服器。您的 Pulse Secure 用戶端管理員能設定清單，使 Pulse Secure 用戶端一律連線至清單中第一個伺服器、您最近連線的伺服器，或是從清單中隨機選取的伺服器。

若要使用定義的連線來連線至網路：

  1. 針對您要建立的連線按一下連線。

    或者

    在系統匣中的 Pulse Secure 用戶端圖示上按一下滑鼠右鍵以顯示快顯功能表、按一下該連線，再按一下連線。

  2. 對要求您輸入使用者名稱、密碼、領域及角色等資訊的提示做出回應。您看到的提示將根據網路管理員設定存取環境的方式而有所不同。

  3. 連線狀態對話方塊隨即出現。如果連線嘗試成功，您可以按一下關閉，或等待對話方塊自動關閉。

    如果連線嘗試發生問題，系統會出現矯正對話方塊，顯示該問題解決方法的相關資訊；或者，會有另外的矯正對話方塊出現，並顯示 Pulse Secure 用戶端自動開始修正問題時所採取之措施的進度。

按一下連線按鈕後，您可能需要對下列提示做出回應：

  • 認證 - 如果 Pulse Secure 用戶端需要與認證伺服器通訊，且您的網路管理員已設定多個伺服器，則系統會提示您選擇其中一個伺服器。認證授權單位發行的認證會確認您要連線的網路資源是否有效。如果認證是由受信任的來源發行，則系統會自動接受，您不會看到認證提示。如果認證存在問題，系統可能會詢問您是否要接受認證並繼續連線。
  • 領域 - 領域會定義您可以存取的網路資源。如果您擁有多個領域的存取權，則需要選取一個領域。
  • 驗證資料 - 您的使用者名稱與密碼或使用者名稱與符記代碼，用於建立您存取裝置的識別身份。系統也可能會提示您輸入次要使用者名稱與密碼，及 Proxy 伺服器的使用者名稱與密碼。您的驗證環境可能會定期提示您變更密碼或符記 PIN 號碼。
  • 角色 - 角色會定義工作階段參數 (逾時值、允許的同時工作階段的數量等)、個人化設定及啟用的存取功能。您的網路管理員可讓您選擇在連線時要扮演的角色。
  • 同時工作階段 - 如果您多次登入系統 (例如使用相同的驗證資料從兩部不同的電腦登入)，那麼系統可能會提示您結束目前工作階段，再開始新工作階段。
  • 矯正動作 - 系統可能會提示您執行一些動作，以使您的端點符合安全性政策。

[儲存設定] 核取方塊會出現於每個登入畫面上。(您的管理員可以停用此功能)。如果您啟用此核取方塊，則在下次登入時，系統將不會提示您該資訊。如果您儲存設定，可以使用 [忘記儲存的設定] 功能來恢復登入時提示儲存資訊。請注意，如果您在有作用中的連線時，選擇 [忘記儲存的設定] 則會無法使用 [暫停] 連線選項。[儲存設定] 功能可讓您儲存下列資訊：

  • 認證接受
  • 認證選擇
  • 領域
  • 使用者名稱與密碼
  • Proxy 使用者名稱/密碼
  • 次要使用者名稱/密碼
  • 角色

對所有連線提示做出回應後所發生的步驟，取決於您的網路管理員設定的存取政策及網路存取裝置的類型。連線過程可包含下列工作：

  • 軟體更新 - Pulse Secure 用戶端可於連線時自動更新。您的系統可能會收到更新的 Pulse Secure 用戶端軟體，或您可能會收到其他軟體模組以支援擴充的服務，例如當您第一次連線至新類型的網路存取裝置時。
  • 主機檢查器 - 主機檢查器會檢查您的端點，以確定其是否符合您組織的安全性政策。例如，主機檢查器可能會檢查防毒軟體、最新病毒碼及個人防火牆狀態，並會檢查是否存在惡意程式與間諜軟體。它也可能會執行其他工作，例如停用瀏覽器自動完成功能，以及清空您的資源回收筒。這些設定由您的網路管理員定義。如果主機檢查器發現問題，它會顯示一則訊息，說明要修正問題您必須執行的步驟，或主機檢查器可自動初始化矯正操作。
  • 快取清潔器 - 快取清潔器會移除 Internet Explorer 所儲存的暫時檔案。快取清潔器會作為主機檢查器的選項來執行。
