檢視進階連線詳細資訊

[進階連線詳細資訊] 對話方塊會顯示有關所選連線的詳細資訊。

若要檢視進階連線詳細資訊：

  1. 按一下該連線加以選取。
  2. 按一下檔案 > 連線 > 進階連線詳細資訊。
  3. 連線詳細資訊不會自動更新。例如，工作階段剩餘時間會顯示您開啟對話方塊時還剩多少時間。若要更新進階詳細資訊，請按一下重新整理或是按一下標示為自動重新整理的核取方塊。

[進階連線詳細資訊] 視窗會提供以下資訊：

欄位名稱

說明

工作階段剩餘時間

目前 VPN 工作階段將維持在作用期間，直到必須再次輸入驗證資料或手動延長工作階段為止。

工作階段期間

目前 VPN 工作階段在作用期間。

通道類型

這會說明連線是使用 VPN 通道或是透過對應至 SAM (Secure Access Manager) 的連接埠/應用程式進行。

VPN 類型

建立通道 (SSL 或 ESP) 所使用的通訊協定。

指派 IPv4

指派給 Pulse 虛擬介面卡的 IPv4 位址。

位元組輸入

透過通道所接收的位元組數。

位元組輸出

透過通道所傳送的位元組數。

連線來源

這會說明 Pulse 用戶端如何接收連線項目：

  • 如果值是 Preconfigured，則連線項目是來自從閘道下載的連線集。
  • 如果值是 User-added，則連線項目是使用者透過 Pulse 用戶端使用者介面手動新增的。
  • 而如果值是 Dynamic，則表示連線項目是藉由啟動 Pulse 用戶端所產生，方法是將網頁瀏覽器連線至 Pulse Secure 閘道並按下網頁上的 [開始] 按鈕。

連線集名稱

這是有包含此連線項目之連線集的名稱。您的系統管理員可使用此名稱在適當閘道 (請見下個欄位) 的管理主控台上尋找相符的連線集，藉此查看連線集的詳細資訊。

連線集擁有者

這是最近編輯過具有此連線項目之連線集的 Pulse Secure 閘道名稱。此閘道及上方所列的連線集名稱，是跨多個 Pulse Secure 閘道尋找用戶端連線集設定所需的兩組資訊位元。

連線集上次修改時間

這是 Pulse 用戶端的連線集在閘道上更新的時間。

連線集伺服器 ID

當 Pulse 用戶端第一次進行部署時，會接收已繫結至特定伺服器 ID 的連線集。依據所有與閘道的後續連線，用戶端只有在閘道連線集的伺服器 ID 與用戶端連線集伺服器 ID 相符的情況下，才會下載更新的連線集。

連線集下載主機

Pulse 用戶端最近從此伺服器更新其連線集。

連線集下載時間

Pulse 用戶端上次更新其連線集的時間。

上方欄位也會顯示在 Pulse Connect Secure 系統管理主控台上。
