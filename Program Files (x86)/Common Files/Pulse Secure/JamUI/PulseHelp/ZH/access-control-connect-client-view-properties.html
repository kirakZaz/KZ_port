檢視連線屬性

若要檢視連線的屬性：

  1. 按一下連線名稱旁邊的箭頭。連線詳細資訊會顯示在連線的下方。

如果 [伺服器 URL] 顯示為隨機選取，表示此連線是 Pulse Secure 用戶端管理員定義的。此連線並未設定一個目標伺服器，而是設定列成清單的多個 Pulse Secure 用戶端伺服器，並由 Pulse Secure 用戶端隨機選取目標伺服器。變更按鈕可讓您從清單中選取特定的伺服器。每個包含多個伺服器的連線都會顯示變更按鈕。您的 Pulse Secure 用戶端管理員能設定清單，使 Pulse Secure 用戶端一律連線至清單中第一個伺服器、您最近連線的伺服器，或是從清單中隨機選取的伺服器。
