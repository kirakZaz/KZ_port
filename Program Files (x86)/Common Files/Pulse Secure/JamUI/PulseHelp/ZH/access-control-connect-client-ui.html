Pulse Secure 用戶端介面

Pulse Secure 用戶端介面會列出您的 Pulse Secure 用戶端連線。一般而言，網路管理員會定義連線，並將連線部署到您的 Pulse 用戶端。管理員也可以讓您建立連至 Pulse Secure 用戶端伺服器的新連線。連線至 Pulse Secure 用戶端伺服器的 Web 入口時，可能會新增連至該伺服器的連線至您的 Pulse Secure 用戶端介面。

Pulse Secure 用戶端也會顯示系統匣圖示，該圖示會提供連線狀態、讓您與網路和會議進行連線及中斷連線，並能快速存取程式介面。

已連線到區域網路，但沒有網際網路存取。公共 WiFi 位置通常會部署網頁驗證入口，要求使用者輸入驗證資訊或接受服務條款，才授予網路存取權。Pulse Secure 用戶端會偵測到網頁驗證入口的存在，而且在獲授予網際網路存取權限之前，不會初始化與 Pulse Secure 用戶端伺服器的連線。

Pulse Secure 用戶端支援「聯邦資訊處理標準」(FIPS)，該標準定義美國政府的安全通訊實務。若您的 Pulse Secure 用戶端管理員已啟用 FIPS，FIPS 開啟會出現在 Pulse Secure 用戶端視窗下方附近。
