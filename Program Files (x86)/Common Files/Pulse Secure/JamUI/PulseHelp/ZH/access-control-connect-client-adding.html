新增連線

Pulse Secure 用戶端中的每個連線都代表一個受保護的網路。一般而言，您的網路管理員會為您定義連線，且可能會停用 [新增連線] 功能。如果您需要建立新連線，您的網路管理員會告知您必須使用的設定。

若要新增連線：

  1. 按一下新增連線 按鈕：

    [新增連線] 對話方塊隨即開啟。

  2. 針對 [類型]，您的網路管理員通常會告知您應選取的選項。請選擇下列其中一項：
    • Policy Secure (UAC) 或 Connect Secure (VPN) - 如果您要連線至 Pulse Policy Secure 或 Pulse Connect Secure，請使用此網路類型。
    • 防火牆 (SRX) - 如果您要連線至 Juniper Networks SRX 系列裝置，請使用此網路類型。
  3. 針對 [名稱]，請為此連線指定一個說明性的名稱。您指定的名稱會顯示在 Pulse Secure 用戶端介面中。
  4. 針對 [伺服器 URL]，請指定您要連線的目標網路。您可以使用下列任一種格式輸入 [伺服器 URL] 值：
    • IP 位址，例如 10.204.71.86。Pulse Secure 用戶端亦支援 IPv6 位址。雖然手動輸入 IPv6 位址不是典型作法，但若您必須這麼做，請使用下列格式：fd29:64fc:bd56:5dbd:3:23:202:1a31。IPv6 位址可包含數字 0 到 9 和字母 a 到 f。您可以省略前置零。
    • DNS 名稱，例如 server.mycompany.net。
  5. 按一下新增，儲存您的新連線並關閉對話方塊。按一下連線，儲存您的新連線並初始化與網路的連線。
