コンポーネント バージョン情報の表示

トラブルシューティングでは、ネットワーク管理者から、Pulse プログラムのバージョン番号を確認するよう依頼される場合があります。Pulse のバージョン情報を表示するには、次の手順に従います。

  1. [Help] (ヘルプ) > [About] (バージョン情報) をクリックして [About] (バージョン情報) ダイアログ ボックスを開きます。
  2. [version details] (バージョン詳細) をクリックして、[Pulse Version Details] (Pulse バージョンの詳細) ダイアログ ボックスを表示します。
