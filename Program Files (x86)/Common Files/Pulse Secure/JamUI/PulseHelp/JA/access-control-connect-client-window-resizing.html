Pulse ウィンドウのサイズ変更

Pulse ウィンドウのサイズを変更して、画面上で別の場所に移動できます。続いてウィンドウを最小化しても、その新しいサイズと位置が保持されます。ただし、プログラムを閉じると、次に Pulse を起動するときには、ウィンドウはデフォルトのサイズと位置に戻ります。

Pulse ウィンドウのサイズを変更するには、次の手順に従います。

  1. Pulse ウィンドウの右下のサイズ変更ハンドルをクリックします。
  2. マウス ボタンを押したままウィンドウの端をドラッグし、希望のウィンドウ サイズになったら離します。
