接続プロパティの編集

接続を作成した後、URL および [Connect] (接続) ペインに表示される名前を編集できます。接続が現在アクティブでなく、手動で作成した場合のみ、接続を編集することができます。Pulse 管理者が作成してインストールした接続の場合、ユーザーが URL フィールドを編集することはできません。

サーバー URL に [Selected at random] (ランダムに選択済み) と表示されている場合、この接続は Pulse 管理者によって定義されていることを意味します。この接続では、1 台のターゲット サーバーではなく、Pulse サーバーのリストが構成されていて、ターゲット サーバーがランダムに選択されます。サーバーのリストを含む各接続の詳細には、[Change] (変更) ボタンが表示されます。Pulse 管理者は、リストの最初にあるサーバー（直近で接続したサーバー）に常に接続されるようにするか、リストからランダムに選択されたサーバーに接続されるように構成できます。サーバーのリストを含む接続用のターゲット URL は編集できません。

接続を編集するには、次の手順に従います。

  1. 接続を右クリックして、ポップアップ メニューを表示し、[Edit] (編集) をクリックします。

    [Edit Connection] (接続を編集) ダイアログ ボックスが開きます。

  2. 接続を編集し、[Save] (保存) をクリックして変更を保存し、ダイアログ ボックスを閉じます。または、[Connect] (接続) をクリックして接続を開始し、ダイアログ ボックスを閉じます。

または

  1. 接続をクリックして選択します。
  2. [Edit Connection] (接続を編集) ボタンをクリックします。

    [Edit Connection] (接続を編集) ダイアログ ボックスが開きます。

  3. 接続を編集し、[Save] (保存) をクリックして変更を保存し、ダイアログ ボックスを閉じます。または、[Connect] (接続) をクリックして接続を開始し、ダイアログ ボックスを閉じます。
