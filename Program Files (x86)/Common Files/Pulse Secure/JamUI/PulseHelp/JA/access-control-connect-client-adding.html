接続の追加

Pulse での各接続は、保護されたネットワークです。通常、ネットワーク管理者がユーザーの接続を定義し、[Add a Connection] (接続を追加) 機能を無効にしている場合があります。新しい接続の作成が必要な場合には、ネットワーク管理者から使用する設定の指示があります。

新しい接続を追加するには、次の手順に従います。

  1. [Add a Connection] (接続を追加) ボタンをクリックします。

    [Add a Connection] (接続を追加) ダイアログ ボックスが開きます。

  2. [Type] (タイプ) については通常、ネットワーク管理者から、どのオプションを選択するか指示があります。次の中から 1 つを選択します。
    • [Policy Secure（UAC）] または [Connect Secure（VPN）] — Pulse Policy Secure または Pulse Connect Secure に接続している場合は、このネットワーク タイプを使用します。
    • ファイアウォール（SRX） — ジュニパーネットワークス SRX シリーズのデバイスに接続する場合は、このネットワーク タイプを使用します。
  3. [Name] (名前) には、この接続の記述名を指定します。指定した名前は、Pulse インターフェイスに表示されます。
  4. [Server URL] (サーバー URL) には、接続先のネットワークを指定します。サーバー URL の値は、以下の形式のどちらでも入力できます。
    • IP アドレス（例：10.204.71.86）。Pulse では、IPv6 アドレスもサポートしています。IPv6 アドレスを手動で入力するのは一般的ではありませんが、そうする必要がある場合は、「fd29:64fc:bd56:5dbd:3:23:202:1a31」の形式で入力します。IPv6 アドレスには、数字の 0～9 および文字 a～f を使用できます。先頭の 0 は省略できます。
    • DNS 名（例：server.mycompany.net）
  5. [Add] (追加) をクリックして、新しい接続を保存し、ダイアログ ボックスを閉じます。[Connect] (接続) をクリックして、新しい接続を保存し、ネットワークへの接続を開始します。
