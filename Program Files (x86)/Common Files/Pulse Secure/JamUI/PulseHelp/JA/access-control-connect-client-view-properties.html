接続プロパティの表示

接続のプロパティを表示するには、次の手順に従います。

  1. 接続名の隣にある矢印をクリックします。接続の詳細が、接続の下に表示されます。

サーバー URL に [Selected at random] (ランダムに選択済み) と表示されている場合、この接続は Pulse 管理者によって定義されていることを意味します。この接続では、1 台のターゲット サーバーではなく、Pulse サーバーのリストが構成されていて、ターゲット サーバーがランダムに選択されます。[Change] (変更) ボタンを使用すると、特定のサーバーをリストから選択できます。[Change] (変更) ボタンは、サーバーのリストを含む各接続に表示されます。Pulse 管理者は、リストの最初にあるサーバー（直近で接続したサーバー）に常に接続されるようにするか、リストからランダムに選択されたサーバーに接続されるように構成できます。
