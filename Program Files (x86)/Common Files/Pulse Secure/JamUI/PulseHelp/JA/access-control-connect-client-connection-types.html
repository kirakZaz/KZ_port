Pulse 接続タイプ

新しい接続を定義するときに選択する接続タイプは、保護されたネットワーク リソースへのアクセスを提供するデバイスのタイプに関連しています。ユーザーが新しい接続を手動で追加するときに、どのタイプを選択するべきかは、ネットワーク管理者から指示を受けます。

  • [Policy Secure（UAC）] または [Connect Secure（VPN）] — Pulse Connect Secure または Pulse Policy Secure に接続している場合、このネットワーク タイプを使用します。
  • ファイアウォール（SRX） — ジュニパーネットワークス SRX シリーズのサービス ゲートウェイに接続する場合は、このネットワーク タイプを使用します。
