Pulse Collaboration ミーティング機能へのアクセス

Pulse Collaboration Suite サーバーとして有効になっているサーバーに Pulse 接続している場合は、トレイの Pulse アイコンから Pulse Collaboration Suite 機能にアクセスでき、ミーティングへの参加、ミーティングのスケジュール設定、ミーティングからの退出などが可能です。また、Web ポータルから Pulse サーバーにログインし、[Meetings] (ミーティング) ボタンをクリックして Pulse Collaboration 機能にアクセスすることもできます。

Pulse Collaboration のオプションにアクセスするには、次の手順に従います。

  1. Windows のシステム トレイにある Pulse のアイコンを右クリックしてポップアップ メニューを表示します。
  2. ミーティング サーバーとして [Pulse Collaboration] をクリックし、続いて [Start Meeting] (ミーティングの開始) をクリックします。
  3. ログイン認証情報を求められた場合は、該当の認証情報を入力します。現在の時間にスケジュールが設定されているミーティングがあると、[Start Meeting] (ミーティングの開始) ボタンがあるブラウザー ウィンドウが表示されます。そのようなミーティングがない場合は、ミーティングのスケジュールが表示されます。ミーティングのスケジュールでミーティングを選択し、[Start Meeting] (ミーティングの開始) をクリックすると Pulse Collaboration Suite クライアントが開くので、選択したミーティングに参加します。Pulse Collaboration ミーティングに参加すると、Pulse Collaboration クライアントがダウンロードされて開きます。Pulse Collaboration Suite の使用方法の詳細については、Pulse Collaboration インターフェイスの [Help] (ヘルプ) リンクをクリックしてください。

    管理者が Microsoft Outlook との統合を有効にしている場合は、リンクをクリックして Microsoft Outlook プラグインをダウンロードおよびインストールします。このプラグインによって、Outlook 内からミーティングのスケジュールを設定したり、Outlook メールを使用して参加者に招待状を送信したりすることができます。

    ミーティングのスケジュールを設定して参加者に招待状を送信するには、[Schedule Meeting] (ミーティングをスケジュール) をクリックします。

    [Instant Meeting] (インスタント ミーティング) または [Support Meeting] (サポート ミーティング) のいずれかを必要に応じてクリックします（管理者がミーティングの種類として、これらを有効にしている場合）。インスタント ミーティングは、普通の打ち合わせなどのようなミーティングです。ユーザーのロール レベル設定（認証要件、リモート管理、安全なチャットなど）が自動的に適用されることで、ミーティングを迅速に作成できます。サポート ミーティングは、2 人の参加者によるミーティングであり、主にオンラインでのトラブルシューティングを目的としたものです。サポート ミーティングでは、デスクトップを共有して、リモート制御による操作が可能です。

    ブラウザー ウィンドウの上部にある [Meetings] (ミーティング) ボタンを使用すると、Web ポータルのどこからでもミーティングのスケジュールに戻ることができます。

    Pulse Collaboration 機能の詳細については、ブラウザー ウィンドウの [Help] (ヘルプ) ボタンをクリックしてください。

ミーティングから切断するには、次の手順に従います。

  1. [Pulse Collaboration] とミーティング サーバーをクリックし、続いて [Sign Out] (サインアウト) をクリックします。

    Pulse Collaboration クライアントのインターフェイスからサインアウトするときに、トレイ アイコンのサインアウト メニュー項目を使用しても、同様の操作が可能です。
