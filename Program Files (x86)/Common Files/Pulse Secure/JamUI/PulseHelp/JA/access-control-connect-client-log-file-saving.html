ログ ファイルの保存

トラブルシューティング プロセスの一部として、Pulse ログ ファイルの保存を要求される場合があります。Pulse には複数のコンポーネントが含まれ、各コンポーネントが 1 つ以上のログ ファイルを生成します。[Save As] (名前を付けて保存) 操作を実行すると、すべての Pulse ログ ファイルおよび他の診断ファイルが LogsAndDiagnostics.zip という 1 つのファイルにまとめられます。

Pulse ログ ファイルを保存するには、次の手順に従います。

  1. [File] (ファイル) > [Logs] (ログ) > [Save As] (名前を付けて保存) をクリックします。[Save As] (名前を付けて保存) ダイアログ ボックスが表示されます。
  2. デフォルトの位置とファイル名を確定するか、新しい位置とファイル名を指定して、[Save] (保存) をクリックします。
