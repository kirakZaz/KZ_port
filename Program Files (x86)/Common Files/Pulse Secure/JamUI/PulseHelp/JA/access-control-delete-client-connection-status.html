Pulse 接続ステータス

Pulse では、[Connect] (接続) ペインとシステム トレイに接続ステータスが表示されます。[Connect] (接続) ペインには、以下のステータスが表示されます。

接続中。失敗または成功するまで、接続はこの状態を維持します。

中断中。

接続に問題があります。

接続は失敗しました。

接続済み。

ローカル ネットワークに接続していますが、インターネットにはアクセスできません。パブリック WiFi ロケーションには、多くの場合、キャプティブ ポータルが展開されており、ユーザーがネットワークへのアクセスを承認されるには、認証情報の入力または条件の同意が必要です。Pulse がキャプティブ ポータルの存在を検出すると、インターネット アクセスが承認されるまで Pulse サーバーへの接続は開始されません。

1 つのシステム トレイ アイコンが、アクティブな Pulse 接続すべてのステータスを示します。システム トレイ アイコンを右クリックすると、Pulse 接続の制御、Pulse Collaboration Suite のミーティング機能へのアクセス、Pulse インターフェイスの表示、Pulse の終了などの操作が可能です。次の表は、システム トレイ アイコンが表示する接続ステータスを示します。

接続がありません。

接続中。失敗または成功するまで、接続はこの状態を維持します。

中断中。

接続に問題があります。

接続は失敗しました。
