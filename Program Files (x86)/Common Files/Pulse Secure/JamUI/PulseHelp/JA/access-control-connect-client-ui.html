Pulse インターフェイス

Pulse インターフェイスには、Pulse 接続の一覧が表示されます。通常、ネットワーク管理者が接続を定義して、ユーザーの Pulse クライアントに配備します。また、管理者は、Pulse サーバーへの新しい接続の作成を許可することもできます。Pulse サーバーの Web ポータルに接続すると、Pulse インターフェイスにそのサーバーへの接続を追加することができます。

Pulse では、システム トレイ アイコンも表示されます。このアイコンを使用して、接続ステータスの確認、ネットワークやミーティングからの接続/切断、プログラム インターフェイスへのクイック アクセスが可能です。

接続中断

ローカル ネットワークに接続していますが、インターネットにはアクセスできません。パブリック WiFi ロケーションには、多くの場合、キャプティブ ポータルが展開されており、ユーザーがネットワークへのアクセスを承認されるには、認証情報の入力または条件の同意が必要です。Pulse がキャプティブ ポータルの存在を検出すると、インターネット アクセスが承認されるまで Pulse サーバーへの接続は開始されません。

Pulse は、米国政府の安全な通信の履行を定義している、連邦情報処理規格（FIPS）に対応しています。Pulse の管理者が FIPS を有効にしている場合、Pulse ウィンドウの下部付近に [FIPS On] (FIPS オン) と表示されます。
