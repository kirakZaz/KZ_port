アクセシビリティ/セクション 508 のコンプライアンス

Pulse Secure デスクトップ クライアントには、身体、知覚または認識障害があるユーザーがデスクトップ クライアントのユーザー インターフェイスを簡単に操作する、いくつかの機能が搭載されています。

一部の機能を有効にするには、レジストリ キーの設定が必要です。このキーは次で設定します。

HKEY_CURRENT_USER\Software\Pulse Secure\Pulse\Accessibility

以下のアクセシビリティ機能があります。

  • タブのサポート - デスクトップ クライアントの UI で、UI ウィジェットに対するキーボード フォーカスを移動するには、タブ キーを使用します。これにより、マウスを使わなくても、ユーザー インターフェイスの操作がしやすくなります。
  • 他のナビゲーション キー -
    • 大半のデータ エントリ フィールドやダイアログ コントロールにアクセスするには、Alt キーを押しながらショートカット キーを押します。Alt キーを押して放すと、使用するショートカット キーを表示できます。
    • Pulse メニューにアクセスするには、Alt キーを押して、メニュー ショートカットを押します。メニュー システムに入った後、Alt ショートカット キー、または矢印と Enter キーを使用して、メニュー オプションを移動/選択できます。
    • ダイアログ内で、Tab キーと矢印キーを組み合わせて移動します。
  • スクリーン リーダーのサポート - JAWS スクリーンリーダーを使用して、接続状態を読み上げられます。スクリーン リーダー機能を有効にするには、ユーザーまたはシステム管理者がレジストリ キー ScreenreaderMode DWORD を 1 に設定する必要があります。
  • システム カラー設定の使用 - Pulse Desktop クライアント UI は、高コントラスト モードの Windows OS システム カラーを適用できます。この機能を有効にするには、ユーザーまたはシステム管理者がレジストリ キー UseSystemColors REG_BINARY を 00000001 に設定する必要があります。

この Pulse Secure デスクトップ クライアント ユーザー ガイドは、さまざまな言語で PDF 形式で用意されています。PDF 形式のユーザー ガイドにアクセスするには、「技術サポートの要求」セクションに記載されている方法で Pulse Secure Global Support Center にアクセスします。
