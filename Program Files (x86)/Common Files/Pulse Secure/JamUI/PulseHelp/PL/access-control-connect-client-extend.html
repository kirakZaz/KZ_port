Wydłużanie sesji połączenia

Administrator sieci może określić limity czasu trwania sesji połączeń sieciowych. Może zostać określony maksymalny limit czasu sesji oraz limit bezczynności sesji. Zbliżanie się wyznaczonego limitu jest sygnalizowane przez aplikację Pulse Secure. Komunikat zawiera ilość pozostałego czasu sesji przed rozłączeniem aplikacji Pulse Secure. W przypadku braku działań aplikacja Pulse Secure rozłączy się, gdy licznik czasu sesji wskaże wartość zerową. Jeśli klikniesz opcję Extend (Wydłuż), aplikacja Pulse Secure zresetuje zegar do wartości maksymalnej określonej przez administratora sieci i umożliwi kontynuowanie sesji bez przerwy w dostawie usług.

Sesję można również wydłużyć, nie czekając na powiadomienie aplikacji Pulse Secure o jej wygasaniu. Wydłużenie sesji jest możliwe w dowolnym momencie i powoduje zresetowanie licznika czasu sesji i licznika czasu bezczynności do wartości maksymalnych.

Funkcja wydłużania sesji nie ma zastosowania w przypadku połączeń z zaporą.

Aby wydłużyć sesję:

  1. Kliknij połączenie, którego czas chcesz wydłużyć.
  2. Kliknij kolejno opcje File > Connections > Extend Session (Plik > Połączenia > Wydłuż sesję).

Lub

  1. Kliknij prawym przyciskiem myszy ikonę aplikacji Pulse Secure w zasobniku systemowym, z rozwiniętego menu podręcznego wybierz połączenie, a następnie kliknij polecenie Extend Session (Wydłuż sesję).

Jeśli stwierdzisz, że sesje są często przerywane przez zbyt krótkie limity czasu, poproś administratora sieci o wydłużenie czasu trwania sesji.
