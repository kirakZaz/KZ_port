Dodawanie połączenia

Każde połączenie w aplikacji Pulse Secure oznacza sieć chronioną. Zwykle administrator sieci definiuje połączenia wymagane przez użytkownika i może wyłączyć funkcję Add a Connection (Dodaj połączenie). W przypadku konieczności utworzenia nowych połączeń administrator sieci przekaże konieczne ustawienia.

Aby dodać nowe połączenie:

  1. Kliknij przycisk Add a Connection (Dodaj połączenie):

    Zostanie otwarte okno dialogowe Add a Connection (Dodaj połączenie).

  2. Administrator sieci wskaże opcję, którą należy wybrać w polu Type (Typ). Wybierz jedno z następujących ustawień:
    • Policy Secure (UAC) lub Connect Secure (VPN) — z tego typu sieci należy korzystać w przypadku nawiązywania połączenia z usługą Pulse Policy Secure lub Pulse Connect Secure.
    • Firewall (SRX) — z tego typu sieci należy korzystać w przypadku nawiązywania połączenia z urządzeniem Juniper Networks SRX Series.
  3. W polu Name (Nazwa) wpisz nazwę opisującą połączenie. Wprowadzona nazwa pojawi się w oknie aplikacji Pulse Secure.
  4. W polu Server URL (Adres URL serwera) określ sieć, z którą chcesz się łączyć. W polu tym można stosować dowolny z następujących formatów:
    • Adres IP, na przykład 10.204.71.86. Aplikacja Pulse Secure obsługuje również adresy IPv6. Mimo iż zwykle adresów IPv6 nie wpisuje się ręcznie, jeśli zaistnieje taka potrzeba, użyj następującego formatu: fd29:64fc:bd56:5dbd:3:23:202:1a31. Adres IPv6 może zawierać cyfry 0–9 i litery a–f. Można ominąć zera wiodące.
    • Nazwa DNS, na przykład serwer.mojafirma.net.
  5. Kliknij opcję Add (Dodaj), aby zapisać nowe połączenie i zamknąć okno dialogowe. Kliknij opcję Connect (Połącz), aby zapisać nowe połączenie i zainicjować połączenie z siecią.
