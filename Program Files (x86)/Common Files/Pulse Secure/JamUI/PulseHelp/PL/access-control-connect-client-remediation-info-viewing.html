Wyświetlanie informacji naprawczych dla połączenia

Jeśli nie udało się nawiązać połączenia lub wystąpił problem z aktywnym połączeniem, aplikacja Pulse Secure wyświetli informacje dotyczące naprawy. Obejmują one instrukcje umożliwiające rozwiązanie problemu lub dane o stanie podjętych automatycznie procedur naprawczych.

Aby wyświetlić informacje naprawcze:

  1. Zaznacz połączenie, klikając je.
  2. Kliknij kolejno opcje File > Connections > Remediation Info (Plik > Połączenia > Informacje naprawcze), aby otworzyć okno dialogowe Remediation (Naprawa).

    Menu Remediation (Naprawa) jest wyszarzone, jeśli nie ma żadnych aktywnych ani oczekujących działań naprawczych.

  3. Górna część okna dialogowego zawiera opis problemu, natomiast część dolna zawiera opis czynności mających na celu jego rozwiązanie. Wykonaj czynności naprawcze, a następnie kliknij opcję Retry (Spróbuj ponownie), aby przywrócić połączenie.

Lub

  1. Kliknij strzałkę obok połączenia, aby rozwinąć to połączenie i wyświetlić jego stan.
  2. W obszarze stanu połączenia kliknij łącze Details (Szczegóły), aby wyświetlić szczegółowe informacje o problemie.

Lub

  1. Kliknij prawym przyciskiem myszy połączenie, aby wyświetlić menu podręczne, a następnie kliknij polecenie Remediation Info (Informacje naprawcze) w celu otwarcia okna dialogowego Remediation (Naprawa).
  2. Wykonaj czynności naprawcze, a następnie kliknij opcję Retry (Spróbuj ponownie), aby przywrócić połączenie.
