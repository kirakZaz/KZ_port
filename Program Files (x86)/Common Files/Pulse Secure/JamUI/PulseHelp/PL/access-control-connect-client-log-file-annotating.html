Umieszczanie adnotacji w plikach dziennika

Adnotacja w pliku dziennika to wstawienie tekstu wyróżniającego plik dziennika w określonej lokalizacji. Aby na przykład rozwiązać problem z połączeniem, użytkownik może zostać poproszony o umieszczenie określonego tekstu adnotacji w pliku dziennika, podjęcie próby nawiązania niedziałającego wcześniej połączenia i ponowne umieszczenie adnotacji w pliku dziennika. Taka sekwencja zdarzeń umożliwi pracownikowi działu pomocy technicznej łatwe wyszukanie wpisów dotyczących problemu z połączeniem.

Aby umieszczać adnotacje w plikach dziennika aplikacji Pulse Secure:

  1. Kliknij kolejno opcje File > Logs > Annotate (Plik > Dzienniki > Umieść adnotację), aby otworzyć okno dialogowe Annotate Logs (Umieść adnotacje w dziennikach).
  2. Wpisz tekst adnotacji i kliknij przycisk OK.
