Zapis plików dziennika

Na jednym z etapów rozwiązywania problemu może pojawić się monit o zapisanie plików dziennika aplikacji Pulse Secure. Aplikacja Pulse Secure zawiera pewną liczbę składników, z których każdy generuje przynajmniej jeden plik dziennika. Operacja Zapisz jako łączy wszystkie pliki dzienników aplikacji Pulse Secure oraz pozostałe pliki diagnostyczne w jeden plik o nazwie LogsAndDiagnostics.zip.

Aby zapisać pliki dziennika aplikacji Pulse Secure:

  1. Kliknij kolejno opcje File > Logs > Save As (Plik > Dzienniki > Zapisz jako). Zostanie otwarte okno dialogowe Save As (Zapisz jako).
  2. Zaakceptuj domyślne wartości lokalizacji i nazwy pliku lub wpisz własne, a następnie kliknij przycisk Save (Zapisz).
