Zmiana rozmiarów okna aplikacji Pulse Secure

Istnieje możliwość zmiany rozmiaru okna aplikacji Pulse Secure i przeniesienia go w inny obszar ekranu. Jeśli okno zostanie następnie zminimalizowane, zachowa nowy rozmiar i pozycję. Jednak kiedy użytkownik zamknie aplikację Pulse Secure, po kolejnym jej uruchomieniu okno powróci do domyślnych rozmiarów i położenia.

Aby zmienić rozmiary okna aplikacji Pulse Secure:

  1. Kliknij uchwyt zmiany rozmiarów w prawym dolnym rogu okna aplikacji Pulse Secure:
  2. Trzymając naciśnięty przycisk myszy, przeciągnij krawędź okna do osiągnięcia żądanych rozmiarów, po czym zwolnij przycisk.
