Usuwanie połączenia

Aby usunąć połączenie:

  1. Zaznacz połączenie, klikając je.
  2. Kliknij przycisk Delete (Usuń):
  3. Przed usunięciem połączenia zostanie wyświetlony monit o potwierdzenie tej decyzji.

Lub

  1. Kliknij prawym przyciskiem myszy połączenie, aby wyświetlić menu podręczne, a następnie kliknij polecenie Delete (Usuń).
  2. Przed usunięciem połączenia zostanie wyświetlony monit o potwierdzenie tej decyzji.
