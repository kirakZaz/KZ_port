Pomijanie zapisanych ustawień

Przy połączeniu z siecią można zaznaczyć pole wyboru Save Settings (Zapisz ustawienia), aby poświadczenia logowania zostały zapisane w aplikacji Pulse Secure. (Uwaga: administrator sieci może wyłączyć tę funkcję). Każdy ekran z monitem o wprowadzenie informacji ma osobne pole wyboru Save Settings (Zapisz ustawienia). Jeśli ustawienia zostaną zapisane, podczas kolejnych prób zalogowania w aplikacji Pulse Secure nie będą wyświetlane monity o wprowadzenie określonych informacji. Jednak w przypadku zmiany poświadczeń logowania najpierw należy wyczyścić zapisane ustawienia w aplikacji Pulse Secure przed zapisaniem nowych poświadczeń.

Jeśli połączenie jest aktywne, po wybraniu opcji Forget Saved Settings (Pomiń zapisane ustawienia) nie będzie dostępna opcja Suspend connection (Wstrzymaj połączenie).

Jeśli wybierzesz opcję Forget Saved Settings (Pomiń zapisane ustawienia), połączenie w stanie wstrzymania zostanie rozłączone.

Aby wyczyścić zapisane poświadczenia logowania:

  1. Kliknij prawym przyciskiem myszy dowolne połączenie, aby wyświetlić menu podręczne.
  2. Kliknij polecenie Forget Saved Settings (Pomiń zapisane ustawienia). Aplikacja Pulse Secure wyczyści zapamiętane ustawienia wszystkich skonfigurowanych połączeń.

Lub

  1. Kliknij kolejno opcje File > Connections > Forget Saved Settings (Plik > Połączenia > Pomiń zapisane ustawienia).
