Uzyskiwanie dostępu do zasobów sieci lokalnej (zawieszanie połączenia)

Jeśli jest aktywne połączenie VPN, uzyskanie dostępu do zasobów lokalnych, takich jak drukarki czy dyski udostępnione, jest często niemożliwe. Program umożliwia wstrzymanie połączenia VPN w celu nawiązania połączenia z zasobami lokalnymi, a następnie ponowne uaktywnienie połączenia VPN ze zdalną siecią bez konieczności podawania poświadczeń logowania.

Jeśli administrator aplikacji Pulse Secure włączył opcję dostępu do podsieci lokalnej, można uzyskać dostęp do zasobu znajdującego się w tej samej podsieci, co komputer, takich jak drukarka, bez konieczności wstrzymywania połączenia aplikacji Pulse Secure.

Aby wstrzymać połączenie VPN i uzyskać dostęp do zasobów lokalnych:

  1. Kliknij aktywne połączenie, aby je zaznaczyć, a następnie kliknij kolejno opcje File > Connections > Suspend (Plik > Połączenia > Wstrzymaj) na pasku menu programu Pulse Secure.

    Jeśli połączenie jest aktywne, po wybraniu opcji Forget Saved Settings (Pomiń zapisane ustawienia) ta opcja będzie niedostępna.

    Lub

    Kliknij prawym przyciskiem myszy aktywne połączenie, a następnie kliknij polecenie Suspend (Wstrzymaj).

    Lub

    Kliknij ikonę aplikacji Pulse Secure w zasobniku systemu Windows, aby wyświetlić menu wyskakujące, po czym kliknij opcję aktywowania połączenia, a następnie kliknij Suspend (Zawieś).

    Etykieta wstrzymanego połączenia w panelu Connections (Połączenia) aplikacji Pulse Secure zmieni się na Resume (Wznów), a obok połączenia zostanie wyświetlona ikona Zawieszono:

    Gdy połączenie jest wstrzymane, ikona w zasobniku wskazuje stan wstrzymania:

Aby ponownie aktywować wstrzymane połączenie VPN:

  1. Wybierz wstrzymane połączenie i kliknij kolejno opcje File > Connections > Resume (Plik > Połączenia > Wznów).

    Lub

    Kliknij opcję Resume (Wznów).

    Lub

    Kliknij ikonę aplikacji Pulse Secure w zasobniku systemu Windows, aby wyświetlić menu wyskakujące, po czym kliknij zawieszone połączenie, a następnie opcję Resume (Wznów).
