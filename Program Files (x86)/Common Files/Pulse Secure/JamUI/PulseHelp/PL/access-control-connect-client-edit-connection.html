Edycja właściwości połączenia

Po utworzeniu połączenia można edytować adres URL i nazwę, która pojawi się w okienku połączeń. Edytować można wyłącznie połączenia nieaktywne, które zostały utworzone ręcznie. Jeśli połączenie zostało utworzone i zainstalowane przez administratora aplikacji Pulse Secure, nie można edytować pola adresu URL.

Jeśli jako adres URL serwera jest wyświetlana informacja Selected at random (Wybrany losowo), oznacza to, że połączenie to zostało określone przez administratora aplikacji Pulse Secure. Zamiast jednego serwera docelowego połączenie to skonfigurowano przy użyciu listy serwerów aplikacji Pulse Secure, a serwer docelowy został z niej wybrany losowo przez aplikację Pulse Secure. W szczegółach wszystkich połączeń zawierających listę serwerów pojawi się przycisk Change (Zmień). Administrator aplikacji Pulse Secure może tak skonfigurować listę, aby aplikacja Pulse Secure zawsze nawiązywała połączenie z pierwszym serwerem na liście, z serwerem, z którym najczęściej nawiązuje się połączenia, lub losowo wybranym na liście. Nie można edytować docelowego adresu URL połączenia, który zawiera listę serwerów.

Aby dokonać edycji połączenia:

  1. Kliknij prawym przyciskiem myszy połączenie, aby wyświetlić menu podręczne, a następnie kliknij polecenie Edit (Edytuj).

    Zostanie otwarte okno dialogowe Edit Connection (Edytuj połączenie).

  2. Dokonaj edycji połączenia, a następnie kliknij opcję Save (Zapisz) w celu zapisania zmian i zamknięcia okna dialogowego lub kliknij przycisk Connect (Połącz), aby zainicjować połączenie i zamknąć okno dialogowe.

Lub

  1. Zaznacz połączenie, klikając je.
  2. Kliknij przycisk Edit Connection (Edytuj połączenie):

    Zostanie otwarte okno dialogowe Edit Connection (Edytuj połączenie).

  3. Dokonaj edycji połączenia, a następnie kliknij opcję Save (Zapisz) w celu zapisania zmian i zamknięcia okna dialogowego lub kliknij przycisk Connect (Połącz), aby zainicjować połączenie i zamknąć okno dialogowe.
