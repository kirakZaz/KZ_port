Wyświetlanie zaawansowanych szczegółów połączenia

Okno dialogowe Advanced Connection Details (Zaawansowane szczegóły połączenia) zawiera szczegółowe informacje na temat wybranego połączenia.

Aby wyświetlić zaawansowane szczegóły połączenia:

  1. Zaznacz połączenie, klikając je.
  2. Kliknij kolejno opcje File > Connections > Advanced Connection Details (Plik > Połączenia > Zaawansowane szczegóły połączenia).
  3. Szczegółowe informacje o połączeniu nie są automatycznie aktualizowane. W polu pozostałego czasu sesji znajduje się na przykład ilość czasu pozostałego w momencie otwarcia okna dialogowego. Aby zaktualizować informacje o zaawansowanych szczegółach, kliknij przycisk Refresh (Odśwież) lub zaznacz pole wyboru Automatically refresh (Odśwież automatycznie).

Okno Advanced Connection Details (Zaawansowane szczegóły połączenia) zawiera następujące informacje:

Nazwa pola

Opis

Session time remaining (Pozostały czas sesji)

Czas przez jaki bieżąca sesja VPN pozostanie aktywna zanim konieczne będzie ponowne wprowadzenie poświadczeń lub ręczne przedłużenie sesji.

Session duration (Czas trwania sesji)

Czas trwania bieżącej aktywnej sesji VPN.

Tunnel type (Typ tunelu)

Informuje, czy połączenie jest tunelem VPN lub mapowaniem portu/aplikacji za pośrednictwem SAM (Secure Access Manager).

VPN type (Typ VPN)

Protokół wykorzystany do utworzenia tunelu (SSL or ESP).

Assigned IPv4 (Przydzielony adres IPv4)

Adres IPv4 przydzielony do wirtualnego adaptera Pulse.

Bytes in (Bajtów przychodzących)

Liczba bajtów odebranych za pośrednictwem tunelu.

Bytes out (Bajtów wychodzących)

Liczba bajtów wysłanych za pośrednictwem tunelu.

Connection Source (Źródło połączenia)

Opisuje, w jaki sposób klient Pulse uzyskał wpis połączenia:

  • Jeżeli wartość to Preconfigured (Wstępnie skonfigurowano), wpis połączenia pochodzi z zestawu połączeń pobranego z bramy głównej.
  • Jeżeli wartość to User-added (Dodane przez użytkownika), wpis połączenia został dodany ręcznie przez użytkownika końcowego za pośrednictwem interfejsu użytkownika programu Pulse.
  • Jeżeli wartość to Dynamic (Dynamiczne), oznacza to, że wpis połączenia powstał w efekcie uruchomienia aplikacji Pulse Secure przez połączenie przeglądarki internetowej z bramą Pulse Secure i naciśnięcie przycisku „Start” na stronie.

Connection Set Name (Nazwa zestawu połączeń)

To nazwa zestawu połączeń, który zawiera ten wpis połączenia. Administrator systemu może korzystać z tej nazwy w celu odszukania zestawu połączeń w konsoli administratora odpowiedniej bramy (zobacz następne pole), aby sprawdzić szczegóły zestawu połączeń.

Connection Set Owner (Właściciel zestawu połączeń)

Jest to nazwa bramy Pulse Secure, która jako ostatnia edytowała zestaw połączeń zawierający ten wpis połączenia. Brama ta wraz i nazwa zestawu połączeń wymieniona powyżej stanowią dwie informacje niezbędne do lokalizacji konfiguracji zestawu połączeń danego klienta wśród wielu bram Pulse Secure.

Connection Set Last Modified (Data ostatniej modyfikacji zestawu połączeń)

Data ostatniej modyfikacji zestawu połączeń klienta Pulse w bramie.

Connection Set Server ID (Identyfikator serwera zestawu połączeń)

Podczas pierwszego wdrożenia klienta Pulse uzyskuje on zestaw połączeń powiązany z określonym identyfikatorem serwera. Podczas wszystkich kolejnych połączeń z bramą klient pobierać będzie zaktualizowany zestaw połączeń tylko wtedy, gdy zestaw połączeń bramy posiada identyfikator serwera odpowiadający identyfikatorowi serwera zestawu połączeń klienta.

Connection Set Download Host (Host pobierania zestawu połączeń)

Serwer, z którego pomocą klient Pulse ostatnio aktualizował swój zestaw połączeń.

Connection Set Download Time (Data pobrania zestawu połączeń)

Czas, kiedy klient Pulse ostatnio aktualizował swój zestaw połączeń.

Powyższe pola są również wyświetlane w konsoli administratora w usłudze Pulse Connect Secure.
