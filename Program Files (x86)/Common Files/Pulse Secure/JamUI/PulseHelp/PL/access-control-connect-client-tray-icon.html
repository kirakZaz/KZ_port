Użyj ikony Pulse Secure na pasku oraz opcji Wyświetl stan połączenia

Ikona aplikacji Pulse Secure w zasobniku informuje o aktualnym stanie połączenia sieciowego. W zasobniku jest zawsze widoczna jedna ikona połączenia — nawet wtedy, gdy nawiązano więcej połączeń. Ikona w zasobniku zmienia stan, gdy wystąpi problem z którymkolwiek z aktywnych połączeń. Stan połączenia wskazują poniższe ikony.

Brak połączenia.

Nawiązywanie połączenia. Połączenie pozostaje w tym stanie do momentu zakończenia lub niepowodzenia operacji.

Zawieszono.

Połączono, ale wystąpiły problemy.

Połączenie się nie powiodło.

Połączono.

Połączono z siecią lokalną bez dostępu do Internetu. W miejscach z publiczną siecią Wi-Fi działa często portal uwierzytelniania, który przed przyznaniem użytkownikowi dostępu do sieci wymaga od niego wprowadzenia informacji uwierzytelniających lub akceptacji warunków usługi. Aplikacja Pulse Secure wykrywa obecność portali uwierzytelniania i nie zainicjuje połączenia z serwerem Pulse Secure aż do uzyskania dostępu do Internetu.

Ikona aplikacji Pulse Secure w zasobniku służy również do realizowania następujących czynności:

  • Open Pulse Secure client (Otwórz aplikację Pulse Secure) — wyświetlenie interfejsu klienta programu Pulse Secure.
  • Pulse Secure Client Collaboration — dostęp do funkcji spotkań pakietu Pulse Collaboration Suite. Element ten jest dostępny tylko wtedy, gdy administrator aplikacji Pulse Secure włączył moduł Pulse Collaboration.
  • Connection List (Lista połączeń) — lista wszystkich połączeń aplikacji Pulse Secure. Kliknij nazwę połączenia, aby wyświetlić dostępne dla niego czynności:
    • Connect (Połącz) — inicjowanie połączenia.
    • Disconnect (Rozłącz) — rozłączanie zaznaczonego połączenia sieciowego.
    • Cancel (Anuluj) — anulowanie czynności zainicjowanej przez serwer aplikacji Pulse Secure. Opcja Cancel (Anuluj) jest dostępna, gdy serwer zainicjował działanie narzędzia Host Checker. Anulowanie spowoduje, że aplikacja Pulse Secure nie nawiąże połączenia z siecią.
    • Suspend (Wstrzymaj) — wstrzymanie połączenia VPN w celu zapewnienia dostępu do lokalnych zasobów sieciowych. Jeśli połączenie jest aktywne, po wybraniu opcji Forget Saved Settings (Pomiń zapisane ustawienia) ta opcja będzie niedostępna.
    • Resume (Wznów) — wznowienie wstrzymanego połączenia VPN.
    • Extend Session (Wydłuż sesję) — wydłużenie aktywnej sesji połączenia bez oczekiwania na wyświetlenie powiadomienia aplikacji Pulse Secure o wygasającej sesji. Wydłużenie sesji jest możliwe w dowolnym momencie i powoduje zresetowanie liczników czasu sesji i czasu bezczynności do ich maksymalnych wartości.
  • Exit (Zakończ) — zamknięcie aplikacji Pulse Secure i rozłączenie wszystkich aktywnych połączeń.
