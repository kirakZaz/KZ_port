Określanie poziomu rejestrowania

Aby ustawić poziom rejestrowania:

  1. Kliknij kolejno opcje File > Logs > Log Level > Detailed (Plik > Dzienniki > Poziom rejestrowania > Szczegółowy) lub File > Logs > Log Level > Normal (Plik > Dzienniki > Poziom rejestrowania > Normalny). Symbol zaznaczenia wskazuje aktualnie włączoną opcję.

Rejestrowanie danych na poziomie Normal (Normalny) jest ustawieniem domyślnym. O ile nie występują żadne problemy z połączeniem, poziom rejestrowania należy ustawić na Normal (Normalny).

Rejestrowanie na poziomie Detailed (Szczegółowy) powoduje tworzenie większej ilości wpisów i zwiększenie rozmiarów plików dziennika. Rejestrowanie szczegółowe jest zwykle włączane jedynie na czas rozwiązywania problemów.
