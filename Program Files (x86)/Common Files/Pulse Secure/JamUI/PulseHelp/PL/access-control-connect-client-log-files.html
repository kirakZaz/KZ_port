Pliki dziennika aplikacji Pulse Secure

W pliku dziennika aplikacji Pulse Secure są zapisywane informacje ułatwiające rozwiązywanie problemów z połączeniem.

Pliki dziennika obejmują połączenia aplikacji Pulse Secure oraz ich właściwości. Aby zapisać konfigurację, wystarczy zapisać pliki dziennika. Aktualna konfiguracja aplikacji Pulse Secure jest zapisywana w pakiecie dzienników. W trakcie odinstalowywania aplikacji Pulse Secure jest wyświetlany monit o zapisanie konfiguracji aplikacji Pulse Secure, umożliwiającej dalsze korzystanie z połączeń aplikacji Pulse Secure i pozostałych ustawień konfiguracji po jej ponownym zainstalowaniu.

Rejestrowanie danych w dzienniku odbywa się w tle. Nie ma konieczności dokonywania żadnych zmian w środowisku dziennika, chyba że pojawią się odpowiednie instrukcje stanowiące część procesu rozwiązywania problemów.

Poza własnymi plikami dziennika aplikacja Pulse Secure zapisuje również zdarzenia w dzienniku zdarzeń aplikacji systemu Windows (dotyczy tylko systemów Windows Vista i Windows 7). Jeśli masz uprawnienia administracyjne na danym komputerze, możesz użyć Podglądu zdarzeń systemu Windows, aby wyświetlić zdarzenia aplikacji Pulse Secure, które zostały zarejestrowane w dzienniku zdarzeń. Informacje zawarte w dzienniku zdarzeń są również zapisywane w plikach dziennika aplikacji Pulse Secure.

Użytkownik może zostać poproszony o wykonanie następujących zadań w ramach procesu rozwiązywania problemów:

  • Annotate the logs (Umieść adnotacje w dziennikach) — umieszczanie adnotacji w pliku dziennika oznacza wstawianie informacji tekstowych oznaczających plik dziennika w określonej lokalizacji. Aby na przykład rozwiązać problem z połączeniem, użytkownik może zostać poproszony o umieszczenie określonego tekstu adnotacji w pliku dziennika, podjęcie próby nawiązania niedziałającego wcześniej połączenia i ponowne umieszczenie adnotacji w pliku dziennika. Taka sekwencja zdarzeń umożliwi pracownikowi działu pomocy technicznej wyszukanie wstawionego tekstu w pliku dziennika. Wstawione adnotacje obejmują pozycje pliku dziennika zawierające informacje o problemie z połączeniem.
  • Set the log level (Określ poziom rejestrowania) — domyślnym poziomem rejestrowania jest Normal (Normalny). Przy rozwiązywaniu problemu może pojawić się monit o zmianę poziomu rejestrowania na Detailed (Szczegółowy).
  • Save the logs (Zapisz dzienniki) — operacja Zapisz jako gromadzi wszystkie pliki dziennika w ramach pojedynczego pliku ZIP i umożliwia określenie jego lokalizacji.
