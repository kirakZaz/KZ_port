Wyświetlanie informacji o wersjach poszczególnych składników

W trakcie rozwiązywania problemów administrator sieci może poprosić o sprawdzenie numerów wersji zainstalowanych składników aplikacji Pulse Secure. Aby wyświetlić informacje o wersjach składników aplikacji Pulse Secure:

  1. Kliknij kolejno opcje Help > About (Pomoc > Informacje) w celu otwarcia okna dialogowego About (Informacje).
  2. Kliknij opcję version details (szczegóły wersji), aby wyświetlić okno dialogowe szczegółów wersji.
