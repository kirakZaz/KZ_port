Usuwanie ręcznego zastępowania przy połączeniu automatycznym

Administrator aplikacji Pulse Secure może skonfigurować połączenia aplikacji Pulse Secure nawiązywane automatycznie. Na przykład może istnieć jedno połączenie aplikacji Pulse Secure, które jest aktywne w biurze po podłączeniu do sieci LAN, i drugie połączenie aplikacji Pulse Secure aktywne poza biurem. Połączenia automatyczne upraszczają nawiązywanie łączności dzięki zmniejszeniu liczby czynności, które należy wykonywać ręcznie. Administrator aplikacji Pulse Secure może zezwolić użytkownikowi na zastąpienie połączeń automatycznych. W aplikacji Pulse Secure pojawi się monit o ręczne zastąpienie połączeń automatycznych, które nie działają już w sposób automatyczny.

Aby przywrócić automatyczny sposób działania połączenia, dla którego ustawiono ręczne zastąpienie:

  1. Zaznacz połączenie, klikając je.
  2. Kliknij kolejno File (Plik) > Connections (Połączenia) > Remove Override (return to automatic control) [Usuń zastąpienie (wróć do sterowania automatycznego)].

Lub

  1. W panelu Connections (Połączenia) aplikacji Pulse Secure kliknij prawym przyciskiem myszy połączenie, aby wyświetlić menu podręczne.
  2. Kliknij Remove Override (return to automatic control) [Usuń zastąpienie (wróć do sterowania automatycznego)].
