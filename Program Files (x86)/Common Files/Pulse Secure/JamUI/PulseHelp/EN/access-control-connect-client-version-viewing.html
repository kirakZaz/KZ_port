View Component Version Information

In a troubleshooting operation, your network administrator might ask you to verify the version numbers of the Pulse Secure client programs. To view Pulse Secure client version information:

  1. Click Help > About to open the About dialog box.
  2. Click version details to open the Pulse Version Details dialog box.
