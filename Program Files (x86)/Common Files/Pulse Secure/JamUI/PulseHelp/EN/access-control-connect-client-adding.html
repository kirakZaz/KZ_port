Add a Connection

Each connection in Pulse Secure client represents a protected network. Typically, your network administrator defines the connections for you and might disable the Add a Connection feature. If you are required to create new connections, your network administrator will tell you the settings that you must use.

To add a new connection:

  1. Click the Add a Connection button:

    The Add Connection dialog box opens.

  2. For Type, typically your network administrator will tell you which option to select. Choose one of the following:
    • Policy Secure (UAC) or Connect Secure (VPN)—Use this network type if you are connecting to Pulse Policy Secure or Pulse Connect Secure.
    • Firewall (SRX)—Use this network type if you are connecting to a Juniper Networks SRX Series device.
  3. For Name, specify a descriptive name for this connection. The name you specify appears in the Pulse Secure client interface.
  4. For Server URL, specify the network that you want to connect to. You can enter the Server URL value in any of the following formats:
    • An IP address, for example, 10.204.71.86. Pulse Secure client also supports IPv6 addresses. Although it is not typical to enter IPv6 addresses manually, if you need to do so, use the following format: fd29:64fc:bd56:5dbd:3:23:202:1a31. An IPv6 address can include numbers 0-9 and letters a-f. You can omit leading zeros.
    • A DNS name, for example, server.mycompany.net.
  5. Click Add to save your new connection and close the dialog box. Click Connect to save your new connection and initiate a connection to the network.
