Pulse Secure Client Connection Status

Pulse Secure client displays the status of a connection in the connections pane and in the system tray. The connections pane can show the following states:

Connecting. A connection stays in this state until it fails or succeeds.

Suspended.

Connected with issues.

Connection failed.

Connected.

Connected to the local network but no Internet access available. Public WiFi locations often deploy a captive portal that requires the user to enter authentication information or to accept terms of service before network access is granted. Pulse Secure client detects the presence of captive portals and does not initiate a connection to a Pulse Secure client server until Internet access is granted.

A single system tray icon indicates the status of all active Pulse Secure client connections. You can right-click the system tray icon to control Pulse Secure client connections, to access Pulse Collaboration Suite meeting functions, to open the Pulse Secure client interface, or to exit from Pulse Secure client.
