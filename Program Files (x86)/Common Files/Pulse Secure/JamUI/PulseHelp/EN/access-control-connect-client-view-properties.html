View Connection Properties

To view the properties for a connection:

  1. Click the arrow next to the connection name. Connection details appear beneath the connection.

If the Server URL appears as Selected at random, it means that this connection was defined by the Pulse Secure client administrator. Instead of one target server, this connection is configured with a list of Pulse Secure client servers and Pulse Secure client randomly selects the target server. A Change button allows you to select a particular server from the list. The Change button appears for every connection that includes a list of servers. Your Pulse Secure client administrator can configure the list so that Pulse Secure client always connects to the first server in the list, to the server where you were most recently connected, or to a server selected randomly from the list.
