Pulse Secure Client Connection Types

The connection type you choose when you define a new connection relates to the type of device that provides access to protected network resources. Your network administrator can tell you which type to select when you add a new connection manually.

  • Policy Secure (UAC) or Connect Secure (VPN)—Use this network type if you are connecting to Pulse Connect Secure or Pulse Policy Secure.
  • Firewall (SRX)—Use this network type if you are connecting to a Juniper Networks SRX Series Services Gateway.
