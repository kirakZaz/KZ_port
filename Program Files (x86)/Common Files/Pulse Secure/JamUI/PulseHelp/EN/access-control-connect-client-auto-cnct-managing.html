Remove Manual Override on an Automatic Connection

Your Pulse Secure client administrator can provide Pulse Secure client connections that connect automatically. For example, you might have one Pulse Secure client connection that activates when you are in the office and connected to the LAN, and a different Pulse Secure client connection that activates when you are away from the office. Automatic connections simplify connectivity by limiting manual connection decisions. Your Pulse Secure client administrator can allow you to override automatic connections. Pulse Secure client displays manual override for automatic connections that no longer respond automatically.

To restore automatic behavior to a connection that has been set to manual override:

  1. Click the connection to select it.
  2. Click File > Connections > Remove Override (return to automatic control).

Or

  1. On the Pulse Secure client Connections pane, right-click click the connection to display the pop-up menu.
  2. Click Remove Override (return to automatic control).
