Edit Connection Properties

After you create a connection, you can edit the URL and the name that appears in the Connections pane. You can edit a connection only if that connection is not currently active and it is one that you have created manually. If a connection was created and installed by your Pulse Secure client administrator, you cannot edit the URL field.

If Server URL appears as Selected at random, it means that this connection was defined by the Pulse Secure client administrator. Instead of one target server, this connection is configured with a list of Pulse Secure client servers and Pulse Secure client randomly selects the target server. A Change button appears in the connection details for every connection that includes a list of servers. Your Pulse Secure client administrator can configure the list so that Pulse Secure client always connects to the first server in the list, to the server where you were most recently connected, or to a server selected randomly from the list. You cannot edit the target URL for a connection that includes a list of servers.

To edit a connection:

  1. Right-click the connection to display the pop-up menu, and then click Edit.

    The Edit Connection dialog box opens.

  2. Edit the connection, and then click Save to save your changes and close the dialog box, or click Connect to initiate a connection and close the dialog box.

Or

  1. Click the connection to select it.
  2. Click the Edit Connection button:

    The Edit Connection dialog box opens.

  3. Edit the connection, and then click Save to save your changes and close the dialog box, or click Connect to initiate a connection and close the dialog box.
