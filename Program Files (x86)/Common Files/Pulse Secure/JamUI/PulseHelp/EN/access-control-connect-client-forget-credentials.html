Forget Saved Settings

When you connect to a network, you can check the Save Settings check box to have Pulse Secure client remember your login credentials. (Note that your network administrator can disable this feature.) Each different screen where you are prompted for a response has its own Save Settings check box. After you save settings, you are not prompted to provide that information on subsequent login attempts. However, if your login credentials change, you must first clear the saved settings in Pulse Secure client before you can save the new credentials.

The Suspend connection option is not available if you choose Forget Saved Settings while you have an active connection.

If you select Forget Saved Settings while a connection is in the Suspend state, the connection is disconnected.

To clear saved login credentials:

  1. Right-click any connection to display the pop-up menu.
  2. Click Forget Saved Settings. Pulse Secure client clears the saved settings for all configured connections.

Or

  1. Click File > Connections > Forget Saved Settings.
