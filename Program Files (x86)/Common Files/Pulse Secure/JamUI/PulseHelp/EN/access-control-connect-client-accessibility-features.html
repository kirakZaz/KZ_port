Accessibility / Section 508 Compliance

The Pulse Secure desktop client contains many features that make it easier for users with physical, sensory or cognitive disabilities to navigate the desktop client user interface.

Some of these features require the setting of a registry key to enable. These keys should be set in:

HKEY_CURRENT_USER\Software\Pulse Secure\Pulse\Accessibility

The accessibility features include:

  • Support for tabbing - The desktop client UI handles moving keyboard focus from one UI widget to another using the Tab key. This makes it easier to navigate the user interface without using a mouse.
  • Other Navigation keys -
    • Most data entry fields and dialog controls can be accessed by holding down the Alt key and pressing the shortcut key. The specific shortcut to be used can be displayed by pressing and releasing the Alt key.
    • The Pulse menus can also be accessed via pressing the Alt key, and the menu shortcut. Once the menu system is entered, either the Alt key shortcuts, or the arrow and enter keys can be used to navigate and select the menu options.
    • Inside of dialogs, the normal combination of Tab and Arrow keys can be used to navigate.
  • Support for screen readers - The JAWS screen reader can be used to read the status of connections. To enable the screen reader functionality, you or your system administrator must set the registry key ScreenreaderMode DWORD to 1.
  • Use System Color settings - The Pulse desktop client UI can adopt the Windows OS’s system colors in high-contrast mode. To enable this feature, you or your system administrator must set the registry key UseSystemColors REG_BINARY to 00000001.
