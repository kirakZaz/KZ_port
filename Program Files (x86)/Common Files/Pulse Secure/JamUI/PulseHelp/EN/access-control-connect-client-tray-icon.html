Use the Pulse Secure Client Tray Icon and View Connection Status

The Pulse Secure client tray icon indicates the current state of your network connection. You see one connection icon in the tray even when you have more than one current connection. When any active connection has an issue, the system tray icon changes its state. The following icons indicate connection status.

No connection.

Connecting. A connection stays in this state until it fails or succeeds.

Suspended.

Connected with issues.

Connection failed.

Connected.

Connected to the local network but no Internet access available. Public WiFi locations often deploy a captive portal that requires the user to enter authentication information or to accept terms of service before network access is granted. Pulse Secure client detects the presence of captive portals and does not initiate a connection to a Pulse Secure client server until Internet access is granted.

You can also use the Pulse Secure client tray icon to perform the following actions:

  • Open Pulse Secure client - Display the Pulse Secure client interface.
  • Pulse Secure client Collaboration - Access Pulse Secure client Collaboration Suite meeting functions. This item appears only if the Pulse Secure client administrator has enabled Pulse Secure client Collaboration.
  • Connection List - List each Pulse Secure client connection. Click the connection name to display the actions that you can perform on that connection:
    • Connect - Initiate a connection.
    • Disconnect - Disconnect the selected network connection.
    • Cancel - Cancel an action that was initiated by the Pulse Secure client server. The Cancel option is available when the server has initiated a Host Checker operation. If you cancel, Pulse Secure client does not make a connection to the network.
    • Suspend - Suspend a VPN connection to allow access to local network resources. Note that this option is not available if you choose to Forget Saved Settings while you have an active connection.
    • Resume - Resume a suspended VPN connection.
    • Extend Session - Extend an active connection session without waiting for Pulse Secure client to notify you of an expiring session. You can select this option at any time to reset the session and idle timers to their maximum values.
  • Exit - Close Pulse Secure client and disconnect any active connections.
