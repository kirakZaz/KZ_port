Resize the Pulse Secure Client Window

You can resize the Pulse Secure client window and move it to another location on your screen. If you then minimize the window, it will retain the new size and position. However, if you close the program, the window will return to its default size and position the next time you start Pulse Secure client.

To resize the Pulse Secure client window:

  1. Click the sizing handle on the lower right side of the Pulse Secure client window:
  2. Hold down the mouse button and drag the edge of the window until you have the window size you want, and then release.
