Access Local Network Resources (Suspend Connection)

When you have an active VPN connection, you might not be able to access local network resource such as printers and shared drives. You can suspend your VPN connection to connect to local resources and then reactivate the VPN connection to the remote network without having to provide login credentials.

If the Pulse Secure client administrator has enabled local subnet access, you can access a resource, such as a printer, that is connected to the same subnet as your computer without suspending the Pulse Secure client connection.

To suspend the VPN connection and access local network resources:

  1. Click the active connection to select it, and then click File > Connections > Suspend on the Pulse Secure client menu bar.

    This option is not available if you choose to Forget Saved Settings while you have an active connection.

    Or

    Right click the active connection, and then click Suspend.

    Or

    Click the Pulse Secure client icon in the Windows system tray to display the pop-up menu, click the active connection, and then click Suspend.

    For a suspended connection, the button label in the Pulse Secure client Connections pane changes to Resume, and the Suspended icon appears next to the connection:

    While a connection is suspended, the tray icon shows the suspended state:

To reactivate a suspended VPN connection:

  1. Select the suspended connection, and then click File > Connections > Resume.

    Or

    Click Resume.

    Or

    Click the Pulse Secure client icon in the Windows system tray to display the pop-up menu, click the suspended connection, and then click Resume.
