Accessibilité / Conformité à la Section 508

Le client Pulse Secure pour ordinateur de bureau inclut de nombreuses fonctionnalités facilitant la navigation dans son interface pour les utilisateurs présentant des déficiences physiques, sensorielles ou cognitives.

Certaines de ces fonctionnalités nécessitent le paramétrage d'une clé de registre à activer. Ces clés doivent être paramétrées dans :

HKEY_CURRENT_USER\Software\Pulse Secure\Pulse\Accessibility

Voici ces fonctionnalités d'accessibilité :

  • Prise en charge de la navigation par tabulations : il est possible de passer d'un widget à un autre dans l'interface du client pour ordinateur de bureau en utilisant la touche de tabulation. L'utilisateur peut ainsi naviguer plus facilement dans l'interface sans avoir besoin d'utiliser une souris.
  • Autres clés de navigation :
    • La plupart des champs de saisie de données et des commandes de dialogue sont accessibles en maintenant la touche Alt enfoncée et en appuyant sur la touche de raccourci. Le raccourci spécifique à utiliser s'affiche en appuyant sur la touche Alt et en la relâchant.
    • Les menus Pulse sont également accessibles en appuyant sur la touche Alt et le raccourci de menu. Une fois le système de menu saisi, il est possible, pour naviguer dans les options de menu et les sélectionner, d'utiliser les raccourcis de touche Alt ou d'utiliser la flèche et de saisir les touches.
    • Dans les boîtes de dialogue, la combinaison normale des touches Flèche et Tab peut être utilisée pour naviguer.
  • Prise en charge de lecteurs d'écran : le lecteur d'écran JAWS peut être utilisé pour lire l'état des connexions. Pour activer la fonction de lecteur d'écran, votre administrateur système ou vous-même devez définir la clé de registre ScreenreaderMode DWORD sur 1.
  • Utilisation des paramètres de couleurs système : l'interface utilisateur du client Pulse pour ordinateur de bureau peut adopter les couleurs système du système d'exploitation Windows en mode contraste élevé. Pour activer cette fonction, votre administrateur système ou vous-même devez définir la clé de registre UseSystemColors REG_BINARY sur 00000001.
