Désactiver la fonction de remplacement manuel pour une connexion automatique

Votre administrateur Pulse Secure Client peut vous fournir des connexions Pulse Secure Client capables de se connecter automatiquement. Par exemple, une première connexion Pulse Secure Client peut s'activer lorsque vous êtes au bureau et connecté au réseau local, et une deuxième connexion Pulse Secure Client peut s'activer lorsque vous êtes en dehors du bureau. Les connexions automatiques permettent de simplifier la connectivité en limitant les décisions de connexions manuelles. Votre administrateur Pulse Secure Client peut vous permettre de remplacer des connexions automatiques. Pulse Secure Client affiche l'option de remplacement manuel pour les connexions automatiques qui ne fonctionnent plus automatiquement.

Pour restaurer le fonctionnement automatique d'une connexion pour laquelle la fonction de remplacement manuel a été activée :

  1. Cliquez sur la connexion pour la sélectionner.
  2. Cliquez sur Fichier > Connexions > Supprimer le remplacement (revenir au contrôle automatique).

Ou

  1. Dans le volet Connexions de Pulse Secure Client, cliquez avec le bouton droit de la souris sur la connexion pour afficher le menu contextuel.
  2. Cliquez sur Supprimer le remplacement (revenir au contrôle automatique).
