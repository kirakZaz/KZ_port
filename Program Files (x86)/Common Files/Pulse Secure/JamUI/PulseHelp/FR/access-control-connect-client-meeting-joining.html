Accéder aux fonctions de réunion collaborative de Pulse Secure Client

Si vous disposez d'une connexion Pulse Secure Client à un serveur activé en tant que serveur Pulse Secure Client Collaboration Suite, vous pouvez accéder aux fonctions de Pulse Secure Client Collaboration Suite à partir de l'icône d'état de Pulse Secure Client et rejoindre, planifier ou quitter une réunion. Vous pouvez également vous connecter au serveur Pulse Secure Client via le portail Web, puis cliquer sur le bouton Réunions pour accéder aux fonctions de Pulse Secure Client Collaboration.

Pour accéder aux options de Pulse Secure Client Collaboration :

  1. Cliquez avec le bouton droit de la souris sur l'icône Pulse Secure Client dans la barre d'état système de Windows pour afficher le menu contextuel.
  2. Cliquez sur Pulse Secure Client Collaboration, le serveur de réunion, puis sur Commencer la réunion.
  3. Si vous y êtes invité, saisissez vos identifiants de connexion. Si une réunion est prévue à cette heure, Pulse Secure Client ouvre une fenêtre de navigation proposant un bouton Commencer la réunion. Sinon, affichez le planning des réunions. Sélectionnez la réunion dans le planning de réunion, puis cliquez sur Commencer la réunion pour ouvrir le client Pulse Secure Client Collaboration Suite et rejoindre la réunion. Lorsque vous rejoignez une réunion Pulse Secure Client Collaboration, cela télécharge et ouvre le client Pulse Secure Client Collaboration. Pour plus de détails sur l'utilisation de Pulse Secure Client Collaboration Suite, cliquez sur le lien Aide dans l'interface Pulse Secure Client Collaboration.

    Si votre administrateur a activé l'intégration Microsoft Outlook, cliquez sur le lien pour télécharger et installer le plug-in Microsoft Outlook. Ce plug-in vous permet de planifier des réunions à partir d'Outlook et d'envoyer des invitations aux participants via la messagerie Outlook.

    Cliquez sur Programmer réunion pour planifier une réunion et inviter des participants.

    Cliquez sur Réunion instantanée ou Réunion d'assistance (si votre administrateur a activé ces types de réunion) en fonction de vos besoins. Une réunion instantanée s'apparente à une réunion normale. Elle vous permet de créer rapidement une réunion en appliquant automatiquement vos paramètres de niveau rôle, tels que l'obligation de s'identifier, le contrôle à distance et la messagerie sécurisée. Une réunion d'assistance est une réunion entre deux personnes destinée principalement à permettre le dépannage en ligne. Elle permet de partager le bureau et d'effectuer des opérations de contrôle à distance.

    Le bouton Réunions situé en haut de la fenêtre du navigateur vous renvoie au planning de réunion, quelle que soit la page du portail Web où vous étiez.

    Pour plus d'informations sur les fonctionnalités de Pulse Secure Client Collaboration, cliquez sur le bouton d'aide dans la fenêtre du navigateur.

Pour se déconnecter d'une réunion :

  1. Cliquez sur Pulse Secure Client Collaboration et le serveur de réunion, puis cliquez sur Déconnexion.

    L'icône de l'élément de menu de déconnexion proposée sur la barre d'état système vous permet de vous déconnecter de la même façon qu'avec l'interface client de Pulse Secure Client Collaboration.
