Accéder aux ressources de réseau local (Suspendre la connexion)

Lorsque vous disposez d'une connexion VPN active, vous risquez de ne pas pouvoir accéder aux ressources de réseau local comme les imprimantes et les lecteurs partagés. Vous pouvez interrompre votre connexion VPN afin de vous connecter aux ressources locales puis réactiver la connexion VPN au réseau distant sans avoir à fournir des données d'identification.

Si l'administrateur Pulse Secure Client a activé l'accès au sous-réseau local, vous pouvez accéder à une ressource, telle qu'une imprimante, si elle est connectée au même sous-réseau que votre ordinateur, et ce sans suspendre la connexion Pulse Secure Client.

Pour interrompre la connexion VPN et accéder aux ressources de réseau local :

  1. Cliquez sur la connexion active pour la sélectionner, puis cliquez sur Fichier > Connexions > Interrompre dans la barre de menus Pulse Secure Client.

    Cette option n'est pas disponible si vous choisissez d'effacer les paramètres enregistrés alors qu'une connexion est active.

    Ou

    Cliquez avec le bouton droit de la souris sur la connexion active, puis cliquez sur Interrompre.

    Ou

    Cliquez sur l'icône Pulse Secure Client dans la barre d'état système de Windows pour afficher le menu contextuel, cliquez sur la connexion active, puis sur Interrompre.

    En cas de connexion interrompue, le libellé du bouton dans le volet Connexions de Pulse Secure Client indique Reprendre, et l'icône Interrompue apparaît à côté de la connexion :

    Lorsqu'une connexion est interrompue, l'icône de la barre d'état indique l'état d'interruption :

Pour réactiver une connexion VPN interrompue :

  1. Sélectionnez la connexion interrompue et cliquez sur Fichier > Connexions > Reprendre.

    Ou

    Cliquez sur Reprendre.

    Ou

    Cliquez sur l'icône Pulse Secure Client dans la barre d'état système de Windows pour afficher le menu contextuel, cliquez sur la connexion suspendue, puis sur Reprendre.
