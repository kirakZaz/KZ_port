Afficher les propriétés de connexion

Pour afficher les propriétés d'une connexion :

  1. Cliquez sur la flèche en regard du nom de la connexion. Les détails relatifs à la connexion apparaissent sous la connexion.

Si l'adresse URL du serveur apparaît comme Sélectionnée de manière aléatoire, cela signifie que cette connexion a été définie par l'administrateur Pulse Secure Client. Cette connexion est configurée non pas avec un seul serveur cible, mais avec une liste de serveurs Pulse Secure Client, puis Pulse Secure Client sélectionne le serveur cible de manière aléatoire. Un bouton Changer vous permet de sélectionner un serveur particulier dans la liste. Le bouton Changer s'affiche pour toutes les connexions ayant une liste de serveurs. Votre administrateur Pulse Secure Client peut configurer la liste de sorte que Pulse Secure Client se connecte toujours au premier serveur de la liste, au dernier serveur auquel vous vous êtes connecté ou à un serveur sélectionné de manière aléatoire dans la liste.
