Prolonger une session de connexion

Votre administrateur réseau peut définir les délais d'expiration de session pour vos connexions réseau. Il peut définir une limite maximale du nombre de sessions et une limite d'inactivité de session. Pulse Secure Client affiche un message lorsque vous atteignez la limite. Le message affiche la durée restante pour la session réseau avant que Pulse Secure Client interrompe la connexion. En l'absence d'action de votre part, Pulse Secure Client interrompt la connexion lorsque le compteur de session atteint zéro. Si vous cliquez sur Prolonger, Pulse Secure Client réinitialise le minuteur sur la valeur maximale définie par l'administrateur réseau et poursuit la session sans interruption de service.

Vous pouvez également prolonger une session sans attendre que Pulse Secure Client affiche le message d'expiration. Vous pouvez prolonger une session à tout moment pour rétablir les valeurs maximales des minuteurs de session et d'inactivité.

La fonction Prolonger la session ne s'applique pas aux connexions de pare-feu.

Pour prolonger une session :

  1. Cliquez sur la connexion que vous souhaitez prolonger.
  2. Cliquez sur Fichier > Connexions > Prolonger la session.

Ou

  1. Dans la barre d'état système, cliquez avec le bouton droit de la souris sur l'icône Pulse Secure Client pour afficher le menu contextuel, puis cliquez sur Prolonger la session.

Si vos sessions sont fréquemment interrompues par des délais d'expiration de session, demandez à votre administrateur système d'augmenter leur durée.
