Afficher les informations sur la version du composant

Il est possible que lors d'une opération de dépannage, votre administrateur réseau vous demande de vérifier les numéros de version des programmes de Pulse Secure Client. Pour afficher les informations de version Pulse Secure Client :

  1. Cliquez sur Aide > À propos de pour ouvrir la boîte de dialogue À propos de.
  2. Cliquez sur détails de la version pour ouvrir la boîte de dialogue Détails de la version de Pulse.
