Effacer les paramètres enregistrés

Lorsque vous vous connectez à un réseau, vous pouvez cocher la case Enregistrer les paramètres pour que Pulse Secure Client mémorise vos données d'identification. (Notez que votre administrateur réseau peut désactiver cette fonction.) Chacun des écrans sur lesquels vous êtes invité à saisir une réponse comporte sa propre case à cocher Enregistrer les paramètres. Lorsque vous enregistrez les paramètres, vous n'êtes plus invité à saisir ces informations lors des prochaines tentatives de connexion. Cependant, si vos données d'identification de connexion changent, vous devrez d'abord effacer les paramètres enregistrés dans Pulse Secure Client pour pouvoir enregistrer les nouvelles données d'identification.

L'option Suspendre la connexion n'est pas disponible si vous choisissez d'effacer les paramètres enregistrés alors qu'une connexion est active.

Si vous sélectionnez Effacer les paramètres enregistrés alors qu'une connexion affiche l'état Suspendue, la connexion est interrompue.

Pour effacer les données d'identification enregistrées :

  1. Cliquez avec le bouton droit de la souris sur une connexion pour afficher le menu contextuel.
  2. Cliquez sur Effacer les paramètres enregistrés. Pulse Secure Client efface les paramètres enregistrés pour toutes les connexions configurées.

Ou

  1. Cliquez sur Fichier > Connexions > Effacer les paramètres enregistrés.
