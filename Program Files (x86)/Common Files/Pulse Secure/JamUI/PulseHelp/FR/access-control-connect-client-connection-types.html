Types de connexion Pulse Secure Client

Le type de connexion que vous choisissez lorsque vous définissez une nouvelle connexion dépend du type de dispositif donnant accès aux ressources réseau protégées. Votre administrateur réseau peut vous indiquer celui que vous devez sélectionner lorsque vous ajoutez une nouvelle connexion manuellement.

  • Policy Secure (UAC) ou Connect Secure (VPN) : utilisez ce type de réseau si vous vous connectez à Pulse Policy Secure ou Pulse Connect Secure.
  • Pare-feu (SRX) : utilisez ce type de réseau si vous vous connectez à une passerelle de services SRX Juniper Networks.
