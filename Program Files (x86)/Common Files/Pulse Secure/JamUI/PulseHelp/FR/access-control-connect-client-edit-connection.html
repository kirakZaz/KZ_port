Modifier les propriétés de connexion

Après avoir créé une connexion, vous pouvez modifier l'URL et le nom affiché dans le volet Connexions. Vous pouvez modifier une connexion uniquement si celle-ci n'est pas actuellement active et que vous l'avez créée manuellement. Si une connexion a été créée et installée par votre administrateur Pulse Secure Client, vous ne pouvez pas modifier le champ d'adresse URL.

Si l'adresse URL du serveur apparaît comme Sélectionnée de manière aléatoire, cela signifie que cette connexion a été définie par l'administrateur Pulse Secure Client. Cette connexion est configurée non pas avec un seul serveur cible, mais avec une liste de serveurs Pulse Secure Client, puis Pulse Secure Client sélectionne le serveur cible de manière aléatoire. Un bouton Changer apparaît dans les détails de connexion pour toutes les connexions comportant une liste de serveurs. Votre administrateur Pulse Secure Client peut configurer la liste de sorte que Pulse Secure Client se connecte toujours au premier serveur de la liste, au dernier serveur auquel vous vous êtes connecté ou à un serveur sélectionné de manière aléatoire dans la liste. Vous ne pouvez pas modifier l'URL cible d'une connexion comportant une liste de serveurs.

Pour modifier une connexion :

  1. Cliquez avec le bouton droit de la souris sur la connexion pour afficher le menu contextuel, puis cliquez sur Modifier.

    La boîte de dialogue Modifier la connexion s'ouvre.

  2. Modifiez la connexion puis cliquez sur Enregistrer pour enregistrer les modifications et fermer la boîte de dialogue, ou cliquez sur Connexion pour établir une connexion et fermer la boîte de dialogue.

Ou

  1. Cliquez sur la connexion pour la sélectionner.
  2. Cliquez sur le bouton Modifier la connexion :

    La boîte de dialogue Modifier la connexion s'ouvre.

  3. Modifiez la connexion puis cliquez sur Enregistrer pour enregistrer les modifications et fermer la boîte de dialogue, ou cliquez sur Connexion pour établir une connexion et fermer la boîte de dialogue.
