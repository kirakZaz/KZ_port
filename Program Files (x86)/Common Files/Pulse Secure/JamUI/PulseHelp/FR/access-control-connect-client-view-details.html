Afficher les détails de connexion avancés

La boîte de dialogue Détails de connexion avancés affichent des informations détaillées sur une connexion sélectionnée.

Pour afficher les détails de connexion avancés :

  1. Cliquez sur la connexion pour la sélectionner.
  2. Cliquez sur Fichier > Connexions > Détails de connexion avancés.
  3. Les informations détaillées sur les connexions ne sont pas mises à jour automatiquement. Par exemple, le temps restant pour la session correspond au temps restant lorsque vous ouvrez la fenêtre. Pour mettre à jour les informations avancées, cliquez sur Actualiser ou cochez la case intitulée Actualiser automatiquement.

La fenêtre Détails de connexion avancés fournit les informations suivantes :

Nom de champ

Description

Temps restant pour la session

Durée de validité de la session VPN actuelle avant que les données d'identification ne doivent être ressaisies ou que la session soit prolongée manuellement.

Durée de la session

Durée pendant laquelle la session VPN actuelle est active.

Type de tunnel

Décrit si la connexion est un tunnel VPN ou un port/une application mis en correspondance via SAM (Secure Access Manager).

Type de VPN

Protocole utilisé pour créer le tunnel (SSL ou ESP).

Adresse IPv4

Adresse IPv4 assignée à l'adaptateur virtuel Pulse.

Octets entrants

Nombre d'octets reçus via le tunnel.

Octets sortants

Nombre d'octets envoyés via le tunnel.

Source de connexion

Décrit la façon dont le client Pulse reçoit l'entrée de connexion :

  • Si la valeur est préconfigurée, l'entrée de connexion provient d'un jeu de connexion téléchargé depuis une passerelle.
  • Si la valeur est ajoutée par l'utilisateur, l'entrée de connexion a été ajoutée manuellement par l'utilisateur final à l'aide de l'interface utilisateur du client Pulse.
  • Enfin, si la valeur est dynamique, cela signifie que l'entrée de connexion a été obtenue en lançant le client Pulse en connectant un navigateur Web à une passerelle Pulse Secure et en appuyant sur le bouton « Démarrer » de la page Web.

Nom du jeu de connexion

Il s'agit du nom du jeu de connexion contenant cette entrée de connexion. Votre administrateur système peut utiliser ce nom pour trouver le jeu de connexion correspondant sur la console d'administration de la passerelle appropriée (voir le prochain champ) afin de voir les détails du jeu de connexion.

Propriétaire du jeu de connexion

Il s'agit du nom de la passerelle Pulse Secure ayant le plus récemment modifié le jeu de connexion doté de cette entrée de connexion. Cette passerelle, ainsi que le nom de jeu de connexion stipulé ci-dessus, sont les deux éléments d'information nécessaires pour localiser la configuration du jeu de connexion du client parmi plusieurs passerelles Pulse Secure.

Jeu de connexion dernièrement modifié

Il s'agit de l'heure à laquelle le jeu de connexion du client Pulse a été mis à jour sur la passerelle.

ID serveur du jeu de connexion

Lorsque le client Pulse est déployé pour la première fois, il reçoit un jeu de connexion lié à un ID serveur particulier. Lors des connexions suivantes à une passerelle, le client télécharge un jeu de connexion mis à jour uniquement si l'ID serveur du jeu de connexion de la passerelle correspond à l'ID serveur du jeu de connexion du client.

Hôte de téléchargement du jeu de connexion

Il s'agit du serveur à partir duquel le client Pulse a dernièrement actualisé son jeu de connexion.

Heure de téléchargement du jeu de connexion

Il s'agit de l'heure à laquelle le client Pulse a dernièrement actualisé son jeu de connexion.

Les champs ci-dessus sont également affichés dans la console d'administration Pulse Connect Secure.
