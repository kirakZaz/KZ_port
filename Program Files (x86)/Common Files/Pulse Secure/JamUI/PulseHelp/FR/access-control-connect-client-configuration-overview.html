Présentation de la configuration de Pulse Secure Client

Pulse Secure Client télécharge les paramètres sur votre point terminal lorsque vous vous connectez à un réseau. La plupart des tâches de configuration de Pulse Secure Client sont effectuées par votre administrateur réseau. Ce dernier peut également activer ou désactiver les fonctions de Pulse Secure Client. Par exemple, votre administrateur peut configurer Pulse Secure Client de manière à ce qu'il se connecte automatiquement à un réseau spécifique ou vous permette de choisir la connexion souhaitée.

Généralement, votre administrateur réseau définit les réseaux auxquels vous pouvez accéder. Il peut également vous permettre de définir des connexions supplémentaires. Si vous disposez d'un ordinateur portable que vous utilisez au bureau, à domicile et lors de vos déplacements, vous avez probablement besoin d'une connexion pour vous connecter au bureau et d'une connexion différente pour les emplacements distants. Cependant, une fois que vous avez établi un type de connexion à un réseau, Pulse Secure Client peut s'adapter automatiquement pour que vous restiez connecté. Par exemple, si vous vous connectez au réseau de l'entreprise à partir d'un emplacement distant, puis fermez votre ordinateur portable sans vous déconnecter avant de vous rendre à votre bureau, vous pouvez ensuite reprendre votre session sans qu'aucune reconnexion ne soit requise. La fonction de portabilité, appelée migration de session, dépend de la manière dont votre administrateur a configuré l'accès au réseau.
