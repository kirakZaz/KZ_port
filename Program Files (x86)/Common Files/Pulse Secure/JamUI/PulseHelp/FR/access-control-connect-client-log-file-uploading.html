Télécharger des fichiers journaux

Le client Pulse Secure pour ordinateur de bureau facilite la transmission des lots de fichiers journaux et de diagnostic aux passerelles PCS afin qu'ils soient analysés par les administrateurs système. Pour envoyer un lot de fichiers journaux à la passerelle PCS, alors qu'une connexion VPN est active, choisissez les options suivantes dans l'interface utilisateur du client de bureau : Fichier > Journaux > Télécharger.

Vous devez sélectionner le serveur auquel vous souhaitez envoyer les fichiers journaux. Une boîte de dialogue s'affiche, indiquant la progression du téléchargement.

L'administrateur système doit activer cette fonction côté serveur pour qu'un utilisateur puisse télécharger des fichiers journaux vers la passerelle Pulse Secure.

La taille maximale de l'ensemble des journaux du client téléchargés sur le serveur ne doit pas dépasser 200 Mo.

La passerelle Pulse Connect Secure inclut une partition de disque à capacité limitée, dédiée au stockage des fichiers journaux téléchargés. Si cet espace sur la passerelle est occupé par d'autres fichiers journaux, le téléchargement peut échouer. Le cas échéant, demandez à l'administrateur système de libérer de l'espace pour que vous puissiez télécharger vos fichiers journaux.
