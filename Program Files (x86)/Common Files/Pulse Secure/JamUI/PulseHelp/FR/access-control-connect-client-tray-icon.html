Utiliser l'icône de la barre d'état Pulse Secure Client et afficher l'état de la connexion

L'icône de la barre d'état Pulse Secure Client indique l'état actuel de votre connexion réseau. Même si plusieurs connexions sont actives, une seule icône de connexion est affichée dans la barre d'état. Lorsqu'une connexion active rencontre un problème, l'état de l'icône de la barre d'état système change. Les icônes suivantes indiquent l'état de connexion.

Pas de connexion.

Connexion en cours. La connexion conserve cet état jusqu'à sa réussite ou son échec.

Connexion suspendue.

Connecté avec des erreurs.

Échec de la connexion.

Connecté.

Connecté au réseau local mais sans accès Internet. Les lieux publics avec accès WiFi disposent généralement d'un portail captif qui demande aux utilisateurs de saisir des informations d'authentification ou d'accepter les conditions de service pour accéder au réseau. Pulse Secure Client détecte la présence de portails captifs et n'assure de connexion à un serveur Pulse Secure Client que si l'accès à Internet a été accordé.

Vous pouvez également utiliser l'icône de la barre d'état Pulse Secure Client pour réaliser les opérations suivantes :

  • Ouvrir Pulse Secure Client : affiche l'interface de Pulse Secure Client.
  • Pulse Secure Client Collaboration : fonctions de réunion d'Access Pulse Secure Client Collaboration Suite. Cet élément n'apparaît que si l'administrateur Pulse Secure Client a activé Pulse Secure Client Collaboration.
  • Liste des connexions : répertorie les connexions Pulse Secure Client. Cliquez sur le nom de la connexion pour afficher les actions réalisables sur cette connexion :
    • Connexion : établit une connexion.
    • Déconnexion : met fin à la connexion réseau sélectionnée.
    • Annuler : annule une action lancée par le serveur Pulse Secure Client. L'option Annuler est disponible lorsque le serveur a lancé une opération Host Checker. Si vous choisissez cette option, Pulse Secure Client ne se connecte pas au réseau.
    • Interrompre : bloque une connexion VPN afin de permettre l'accès aux ressources de réseau local. Notez que cette option n'est pas disponible si vous choisissez d'effacer les paramètres enregistrés alors qu'une connexion est active.
    • Reprendre : reprend une connexion VPN interrompue.
    • Prolonger la session : prolonge une session de connexion active sans attendre que Pulse Secure Client affiche le message d'expiration de session. Vous pouvez sélectionner cette option à tout moment pour rétablir les valeurs maximales des minuteurs de session et d'inactivité.
  • Quitter : ferme Pulse Secure Client et met fin aux connexions actives.
