Enregistrer des fichiers journaux

Dans le cadre du dépannage, il peut vous être demandé d'enregistrer les fichiers journaux Pulse Secure Client. Pulse Secure Client inclut un certain nombre de composants possibles, chacun d'entre eux générant un ou plusieurs fichiers journaux. L'opération Enregistrer sous rassemble tous les fichiers journaux Pulse Secure Client et autres fichiers de diagnostic dans un fichier unique appelé LogsAndDiagnostics.zip.

Pour enregistrer les fichiers journaux Pulse Secure Client :

  1. Cliquez sur Fichier > Journaux > Enregistrer sous. La boîte de dialogue Enregistrer sous apparaît.
  2. Acceptez les valeurs par défaut pour l'emplacement et le nom du fichier ou spécifiez de nouvelles valeurs, puis cliquez sur Enregistrer.
