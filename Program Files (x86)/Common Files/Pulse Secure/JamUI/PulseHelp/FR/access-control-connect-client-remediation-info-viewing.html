Afficher les informations de correction relatives à une connexion

Lorsqu'une tentative de connexion échoue, ou lorsqu'une connexion active rencontre un problème, Pulse Secure Client présente des informations de correction. Les informations de correction incluent des instructions sur la procédure à suivre pour résoudre le problème ou affichent l'état des procédures de correction qui ont été initiées automatiquement.

Pour afficher les informations de correction :

  1. Cliquez sur la connexion pour la sélectionner.
  2. Cliquez sur Fichier > Connexions > Informations relatives à la correction pour ouvrir la boîte de dialogue Correction.

    Le menu Correction est grisé s'il n'y a aucune action de correction active ou en attente.

  3. La partie supérieure de la boîte de dialogue décrit le problème et la partie inférieure, les étapes à suivre pour le résoudre. Suivez les étapes de la correction, puis cliquez sur Réessayer pour restaurer la connectivité.

Ou

  1. Cliquez sur la flèche en regard d'une connexion pour la développer et afficher son état.
  2. Dans l'état de la connexion, cliquez sur le lien Détails pour afficher des informations plus détaillées concernant le problème.

Ou

  1. Cliquez avec le bouton droit de la souris sur la connexion pour afficher le menu contextuel, puis sur Informations relatives à la correction pour ouvrir la boîte de dialogue Correction.
  2. Suivez les étapes de la correction, puis cliquez sur Réessayer pour restaurer la connectivité.
