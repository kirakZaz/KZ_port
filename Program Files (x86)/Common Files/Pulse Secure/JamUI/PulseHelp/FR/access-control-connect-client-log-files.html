Fichiers journaux Pulse Secure Client

Un fichier journal Pulse Secure Client comporte des informations permettant de résoudre les problèmes de connexion.

Les fichiers journaux comprennent vos connexions Pulse Secure Client et leurs propriétés. Si vous souhaitez enregistrer la configuration, il vous suffit d'enregistrer les fichiers journaux. Votre configuration Pulse Secure Client actuelle est enregistrée dans le module des journaux. Si vous désinstallez Pulse Secure Client, le programme vous demande si vous souhaitez enregistrer votre configuration Pulse Secure Client ; ainsi, lorsque vous réinstallerez Pulse Secure Client ultérieurement, vous récupérerez vos connexions Pulse Secure Client et les autres paramètres de configuration.

La journalisation est une opération d'arrière-plan. Il n'est pas nécessaire d'apporter des modifications à votre environnement de journalisation, sauf si vous y êtes invité dans le cadre du dépannage.

Pulse Secure Client consigne les événements à la fois dans ses propres fichiers journaux et dans le journal d'événements de l'application Windows (systèmes Windows Vista et Windows 7 uniquement). Si un accès administrateur est déployé sur votre ordinateur, vous pouvez utiliser l'Observateur d'événements Windows pour afficher les événements Pulse Secure Client ayant été consignés dans le journal d'événements. Les informations du journal des événements sont incluses lorsque vous enregistrez les fichiers journaux Pulse Secure Client.

Pour faciliter le dépannage, il peut vous être demandé d'effectuer les tâches suivantes :

  • Annoter les journaux : lorsque vous annotez un fichier journal, vous insérez du texte marquant le fichier journal à un emplacement spécifique. Par exemple, pour résoudre un problème de connexion, il peut vous être demandé d'annoter le fichier journal avec du texte spécifique, de tenter d'établir la connexion ayant échoué, puis d'annoter de nouveau le fichier journal. Cette séquence d'événements permet à un agent d'assistance de rechercher le texte que vous avez inséré dans le fichier journal. Le texte met entre crochets les entrées du fichier journal établissant le suivi de votre problème de connexion.
  • Définir le niveau de journalisation : le niveau de journalisation par défaut est Normal. Pour une opération de dépannage, il peut vous être demandé de définir le niveau de journalisation sur Détaillé.
  • Enregistrer les journaux : l'opération Enregistrer sous rassemble tous les fichiers journaux dans un fichier ZIP unique et vous permet de spécifier sa destination.
