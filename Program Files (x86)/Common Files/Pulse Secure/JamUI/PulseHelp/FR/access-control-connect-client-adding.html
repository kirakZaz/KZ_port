Ajouter une connexion

Dans Pulse Secure Client, chaque connexion représente un réseau protégé. L'administrateur réseau définit généralement vos connexions et peut désactiver la fonction Ajouter une connexion. Si vous devez créer des connexions, l'administrateur réseau vous dira quels paramètres utiliser.

Pour ajouter une nouvelle connexion :

  1. Cliquez sur le bouton Ajouter une connexion :

    La boîte de dialogue Ajouter une connexion s'ouvre.

  2. L'administrateur réseau vous indiquera l'option à sélectionner pour le champ Type. Choisissez l'une des options suivantes :
    • Policy Secure (UAC) ou Connect Secure (VPN) : utilisez ce type de réseau si vous vous connectez à Pulse Policy Secure ou Pulse Connect Secure.
    • Pare-feu (SRX) : utilisez ce type de réseau si vous vous connectez à un dispositif SRX Juniper Networks.
  3. Dans le champ Nom, entrez un nom descriptif pour cette connexion. Le nom que vous spécifiez apparaît dans l'interface Pulse Secure Client.
  4. Pour l'adresse URL du serveur, spécifiez le réseau auquel vous voulez vous connecter. Vous pouvez entrer la valeur de l'adresse URL du serveur dans l'un des formats suivants :
    • Une adresse IP, par exemple 10.204.71.86. Pulse Secure Client prend également en charge les adresses IPv6. Les utilisateurs entrent rarement les adresses IPv6 manuellement. Toutefois, si vous avez à le faire, utilisez le format suivant : fd29:64fc:bd56:5dbd:3:23:202:1a31. Une adresse IPv6 peut contenir des chiffres de 0 à 9 et des lettres de a à f. Vous pouvez omettre les zéros non significatifs.
    • Un nom DNS, par exemple serveur.masociete.net.
  5. Cliquez sur Ajouter pour enregistrer votre nouvelle connexion et fermer la boîte de dialogue. Cliquez sur Connexion pour enregistrer votre nouvelle connexion et établir une connexion au réseau.
