; JUNS catalogUtil requires catalog message files to be saved with UTF-8 encoding, which put 3 bytes {0xEF, 0xBB, 0xBF} at the file beginning called Byte Order Mark (BOM) for UTF-8 files . For modifications in this files, please use Text Editors which persist these BOM bytes (like Notepad.exe, Visual Studio IDE, etc).

; kMsg8021xAMErrNoWindowsProfile
[1500]
Short-desc = Brak profilu WLAN systemu Windows
Long-desc = Sprawdź konfigurację sieci bezprzewodowej i spróbuj ponownie wykonać operację.

; kMsg8021xAMErrNoSSID
[1501]
Short-desc = Brak SSID w zasięgu do żądania połączenia.
Long-desc = Sprawdź konfigurację sieci bezprzewodowej i moc sygnału SSID, a następnie spróbuj ponownie wykonać operację.

; kMsg8021xAMErrIVEFailed
[1502]
Short-desc = Połączenie z IVE nie powiodło się
Long-desc = Ręcznie ponownie uruchom połączenie.

; kMsg8021xAMErrReconnectFailed
[1503]
Short-desc = System Windows nie odpowiada na żądanie ponownego połączenia.
Long-desc = Być może kabel sieciowy został odłączony albo podłączono go do portu innego niż 802.1x (w przełączniku). Usuń możliwą przyczynę i ręcznie ponownie uruchom połączenie.

; kMsg8021xAMErrWirelessFailed
[1504]
Short-desc = Żądanie dotyczące sieci bezprzewodowej 802.1x nie powiodło się
Long-desc = Ręcznie ponownie uruchom połączenie.

; kMsg8021xAMErrTerminated
[1505]
Short-desc = Ponownego uruchamiania systemu Microsoft Windows jest uwierzytelnianie.
Long-desc = Microsoft Windows 802.1x petent jest ponowne uruchomienie uwierzytelniania. Połączenie powinno być wznowione wkrótce.

; kMsg8021xAMErrWirelessSuppression
[1506]
Short-desc = Sieci bezprzewodowe 802.1x zostaną pominięte.
Long-desc = Pomijanie sieci bezprzewodowych zostało włączone przez administratora. Sieci bezprzewodowe 802.1x zostaną pominięte po wykryciu prawidłowej sieci przewodowej.
